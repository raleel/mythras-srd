3.  תוכלו לבצע את כל התהליך הזה דרך הדפדפן מבלי להכיר את Git. עם זאת, לימוד Github Desktop יאפשר לכם לבדוק שינויים באופן מקומי לפני פריסתם בשידור חי שהעולם יראה.

## הוספת התוכן שלכם

### קבצי ‎*.md

קבצי ‎.md הם קבצי מרקדאון (Markdown). מרקדאון היא שפת סימון קלת משקל שבה ניתן להשתמש כדי להוסיף אלמנטי עיצוב למסמכי טקסט רגילים. בהתאם לגודל התוכן שלכם, תוכלו למקם את כל המידע של המשחק שלכם בקובץ ‎*.md יחיד, או כפי שעשיתי ב-[7thextinctionrpg.com](https://srd.7thextinctionrpg.com), ליצור קבצים נפרדים לכל פרק. הקובץ "sample.md" מספק כמה דוגמאות עיצוב.

שפת המרקדאון קלה מאוד ללמידה, השתמשו בקישור המסופק בסעיף 2 תחת "דרישות קדם" לעיל לרשימה מלאה של תחביר.

### _Sidebar.md

זהו התפריט שלכם, חלונית הניווט בצד שמאל של האתר. לאחר שהוספתם קובץ מרקדאון למאגר שלכם, עליכם להוסיף את הקובץ לכאן כדי שתכניו ייכללו. עקבו אחר העיצוב הקיים.

## שימוש בתוסף Fontawesome ו/או Material Icons להצגת אייקונים חינמיים ב-SRD שלכם

### Fontawesome

השתמשו ב-Fontawesome כדי לקבל אייקונים חינמיים ואייקונים מונפשים. משתמשים מנויים יכולים להשתמש באייקוני Pro.

1.  מצאו את האייקון ב-[Fontawesome](https://fontawesome.com/icons)
2.  בחרו את האייקון
3.  בחלון הקופץ, תחת לשונית ה-HTML העתיקו כל מה שנמצא בין " "

```EXAMPLE
<i class="fa-solid fa-dice-d20"></i>
```

4.  מקמו את הטקסט שהועתק בין `:` ללא רווחים

```EXAMPLE
fa-solid fa-dice-d20
```

### Material Icons

השתמשו ב-Material Icons לעוד אייקונים חינמיים.

1.  מצאו את האייקונים ב-[Material Icons](https://fonts.google.com/icons)
2.  העתיקו והכניסו את קישור ה-span ישירות לקבצי המרקדאון שלכם

```markup
<span class="material-icons">face</span>
```
