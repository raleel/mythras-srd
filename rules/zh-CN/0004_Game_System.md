# 游戏系统

在《_Mythras Imperative_》游戏中，可能会发生许多不同的事情。有些是通过技能使用或主要游戏机制（如战斗或魔法）来解决的；另一些则需要其自身的规则来裁定。本章提供了一系列不同情境下的规则和指导，玩家在游戏过程中会遇到这些情境。

这份列表并非详尽无遗，但应涵盖最常见的情况。如果游戏主持人发现还有其他情况需要某种形式的游戏系统裁定，并且这些情况似乎经常发生，那么他们应被鼓励开发自己的规则来应对这些特殊情况，并以本章提供的规则为基础进行复制或开发。

## 窒息

角色可以屏住呼吸的秒数等于其耐力（Endurance）技能值。然而，角色必须做好准备（尽可能充满肺部空气）；如果未准备，那么如果角色处于被动状态，屏息时间减半；如果角色正在进行剧烈活动，则减少到五分之一。

一旦屏息时间结束，角色必须进行一次耐力（Endurance）检定：

- 如果检定结果是**大成功**，则不会进一步恶化。
- 如果检定**成功**，角色会额外获得一级**疲劳**。
- 如果检定**失败**，角色在该回合额外承受1d2级**疲劳**。
- 如果检定**笨拙**，角色在该回合额外承受1d3级**疲劳**。

如果没有帮助，窒息导致的死亡通常很快。如果在角色死亡之前窒息结束，他们会相对较快地恢复因窒息而失去的疲劳等级，每分钟恢复一级。

根据窒息的方式，游戏主持人可能希望阻止完全恢复，施加一个持续的疲劳等级（参阅[疲劳](0004_Game_System.md?id=fatigue)部分），以表示因吸入烟雾或水而对肺部造成的损害。

## 角色提升

每个《_Mythras Imperative_》角色都会随着时间和经验而提升。提升可以在几个方面进行：

- 提升现有技能
- 提升热情
- 学习新技能
- 学习新能力

#### 经验检定

大多数角色提升的机制是**经验检定**。游戏主持人会在战役的适当节点发放经验检定：在每个成功的剧本或故事情节结束时；或者如果故事很长，需要时间完成，则可能在两到三场游戏会话之后。频率由游戏主持人决定。高频率的经验检定将导致角色以更快的速度发展。

发放经验检定没有对错之分，但故事中的自然断点可能暗示了合适的时间。然而，游戏主持人始终是这里的决策者。虽然玩家不能要求经验检定，但他们有权在某些时候期待它们——因为没有它们，他们的角色将无法发展。

发放的经验检定数量也由游戏主持人决定。然而，一个好的经验法则是自上次发放以来，每场游戏会话1-3个经验检定。

#### 提升现有技能

角色卡上的任何技能，无论是标准技能还是职业技能，都可以通过花费一个**经验检定**来提升。玩家可以选择不将经验检定用于提升技能，而是积累起来用于提升其他事物——例如职业能力（见下文）。

玩家掷1d100并与正在提升的技能进行比较。角色的INT值会加到检定结果中。

- 如果掷出的数字等于或大于正在提升的技能，该技能提升1d4+1%。
- 如果掷出的数字小于所选技能，该技能仍然提升，但只提升1%。
- 如果角色在之前的一次或多次会话中——即在上一组经验检定和本次之间——**笨拙**了任何技能，该笨拙的技能会免费提升1%。我们从错误中学到的比从成功中学到的更多，这代表了角色在灾难性失败后进行的思考。同一技能多次笨拙不会叠加。如果笨拙的技能也用经验检定来提升，笨拙获得的1%会在提升检定之前应用。

同一技能每场游戏会话不能受益于一个以上的经验检定。

#### 提升热情

如[热情](0002_Characters.md?id=step-8-passions)部分的热情所述，热情的值可以通过经验检定以与技能相同的方式提升。如果得到游戏支持，游戏主持人甚至可能允许使用经验检定来降低热情。

_例如，一段没有回报的爱情的持续拒绝最终可能会消磨掉即使是最热情的爱人的热情，特别是那些希望追求一段没有过去幽灵困扰的恋情的人。_

#### 学习新技能

有些角色可能希望学习他们从未有机会从其文化中习得的新职业技能。在他们开始投入经验检定之前，他们必须首先找到一个知识来源来学习。这可能像专业导师一样平淡无奇，例如由行会或公会提供的导师。或者，它可能是一个更奇特的教育来源，例如一份古老而破旧的训练卷轴。

一旦找到教育来源，角色必须花费整整一个月的时间学习和练习，以掌握该技能的基本知识（学习到基础等级）。这需要3个经验检定，以及支付教师（如果存在）和购买（或租用）所需设备和工具所需的任何游戏内费用。

## 疾病和毒药

角色可能遇到的最可怕的事情之一是世界上那些黑暗而严酷之地滋生的无数疾病和毒药。没有什么比一场袭击角色家园城市的瘟疫更令人不安的了，尤其是当他们还在城门内时。同样，在床上发现一条毒蛇或被巨型蝎子攻击都可能是极富戏剧性的时刻。

疾病和毒药的种类几乎无限，但出于游戏目的，它们以相同的方式处理。

#### 疾病和毒药特性

所有疾病和毒药都表现出对其效果很重要的几个特性。

**施用方式：** 疾病或毒药进入受害者体内的方法。

- **摄入：** 通过饮食发生作用。
- **吸入：** 通过呼吸或嗅闻发生作用。
- **接触：** 通过皮肤吸收发生作用。
- **注射：** 通过穿刺将物质注入体内后发生作用。

**效力：** 疾病或毒药的毒性。此值将与适当的抵抗技能（通常是耐力或意志力）进行对抗检定。如果角色赢得检定，他们会摆脱其影响。如果他们输掉检定，那么他们会遭受所有疾病或毒药的**症状**，每个都在适当的时间发生。

**抵抗：** 疾病或毒药如何被抵抗——通常是耐力或意志力，但也可能有更奇特的抵抗要求。抵抗可以在接触时进行检定，或者推迟到**发作时间**以隐藏角色可能已被感染或中毒的事实。

**发作时间：** 许多疾病和毒药不会立即生效。延迟称为**发作时间**，这可能是几秒钟、几分钟、几小时，甚至更长。具有多种效果的毒药或疾病可能对每种效果有不同的发作时间，如**症状**所述。

**持续时间：** 疾病或毒药的**症状**持续多长时间。

**症状：** 每种毒素都有一种或多种**症状**。这些症状具有特定效果，如下面的**症状表**所述，如果受害者未能抵抗，他将遭受所有描述的症状。

**解药/治疗：** 如果毒素可以治疗，将在此处注明。成功治疗可防止任何进一步症状的发生，但在某些情况下，可能会使受害者遭受已造成的持久影响。

##### 症状表

| 症状 | 效果 |
| :-- | :-- |
| 剧痛 | 受害者受到剧烈疼痛的阻碍。无论是局部还是全身，任何涉及使用受影响区域的技能检定，其结果都必须小于或等于角色的意志力，否则尝试失败，他们会因疼痛而呻吟或尖叫。 |
| 窒息 | 受害者遭受窒息——他倒地不起，无法呼吸。使用[窒息](0004_Game_System.md?id=asphyxiation)部分的规则。窒息可能是哮喘性质的，意味着受害者只遭受气短；也可能是完全的呼吸衰竭，导致死亡。在后一种情况下，受害者可以通过赢得急救（First Aid）技能对抗疾病或毒药效力的对抗检定来维持生命。 |
| 出血 | 受害者遭受内出血或表面出血，导致下文[疲劳](0004_Game_System.md?id=fatigue)和[失血](0004_Game_System.md?id=blood-loss)部分所述的疲劳和恢复时间。 |
| 传染性 | 受害者可以通过接触传播毒药或疾病。 |
| 死亡 | 受害者倒地不起，并在相当于其体质（CON）特性的轮数后死亡。 |
| 发烧 | 受害者的体温剧烈波动——忽冷忽热——肌肉酸痛。所有技能遭受“困难”难度等级惩罚。 |
| 狂躁 | 受害者被某种强迫症驱动，例如避水、偏执、自残等。疾病或毒药引起的狂躁将在其描述中注明。 |
| 恶心 | 受害者无法进食，每次进行有压力的体力活动时都必须对抗其耐力（Endurance）以避免身体不适。呕吐持续1d3轮，在此期间他无法行动。长时间的恶心可能导致饥饿。 |
| 麻痹 | 受害者身体无法移动。受影响的区域，如果是一个部位，在持续期间无法使用。如果影响全身，角色完全无法移动。 |

#### 疾病样本：红斑病

红斑病是一种令人讨厌的疾病，通过皮肤接触已感染者或其接触过的物品传播。从物品感染疾病的几率在接触后1d3天内仍然存在，这使得一旦爆发就很难控制疾病。

受害者最初会出现轻微皮疹，但一天内就会出现严重发烧，通常会使他们卧床不起整个病程。在此期间，皮疹会演变成令人毛骨悚然的瘙痒脓疱，稍一移动就会破裂，流出少量血液——尤其是在腋窝和腹股沟。这个阶段可能持续一周或更长时间，并可能致命。

- **施用方式：** 接触
- **效力：** 50
- **抵抗：** 耐力（Endurance）
- **发作时间：** 1d6+6 小时
- **持续时间：** 1d6+3 天
- **症状：** 一旦疾病发作，受害者会变得具有**传染性**。1天后，他们开始出现**发烧**和**出血**。每天脓疱流血，受害者失去一级**疲劳**，直到疾病持续时间结束才恢复。
- **解药/治疗：** 可通过治疗（Healing）技能治愈。

## 坠落

坠落造成的伤害量取决于坠落距离。护甲点数不会减少坠落伤害，但坠落角色或生物的伤害修正会应用于坠落所受的伤害。

##### 坠落距离表

| 坠落距离 | 承受伤害 |
| :-- | :-- |
| 1米或更少 | 无伤害。 |
| 2-5米 | 对一个随机部位造成1d6点伤害。 |
| 6-10米 | 对两个随机部位造成2d6点伤害。 |
| 11-15米 | 对三个随机部位造成3d6点伤害。 |
| 16-20米 | 对四个随机部位造成4d6点伤害。 |
| 每增加+5米 | +1d6伤害。 |

#### 坠落物体

坠落物体造成的伤害量基于其体型（SIZ）和坠落距离。一个物体每6点体型（或不足6点的部分）造成1d6点伤害，加上相当于坠落距离表中“承受伤害”列的伤害量——包括对较小尺寸物体的任何减少。

## 疲劳

疲劳衡量疲倦及其逐渐累积的影响。它用于追踪许多不同的事物，从剧烈活动到疾病的虚弱影响。

累积疲劳的主要方式是进行某种形式的体力活动。锻炼或工作越费力，角色越快疲倦。在游戏主持人的裁量下，角色必须进行适当的技能检定——体育（Athletics）、蛮力（Brawn）或耐力（Endurance）——以抵抗获得一级疲劳。

每次**失败**的检定都会累积一级疲劳。每级疲劳都会对技能使用、移动、先攻和行动点数造成惩罚。窒息、失血和其他依赖于设定的效果也会导致疲劳累积。

对于大多数角色来说，当达到“丧失行动能力”级别时，任何活动都变得几乎不可能。在这个阶段，角色仍然清醒，但除了最绝望的活动外，无能为力。

超过“丧失行动能力”级别，角色完全无法行动。“半昏迷”、“昏迷”和“死亡”等疲劳级别通常用于衡量窒息、疾病、失血、饥饿、暴露等最极端的影响。

#### 从疲劳中恢复

角色从疲劳中恢复的速度取决于其**治愈率**。从每级累积疲劳中完全恢复所需的完整休息时间等于**恢复周期**除以角色的**治愈率**。

请注意，下表代表体力消耗后的疲劳恢复。从窒息中恢复时，疲劳恢复可能更快；从失血中恢复时，疲劳恢复可能更慢。

##### 疲劳等级

| 疲劳等级 | 技能难度等级 | 移动 | 先攻 | 行动点数 | 恢复周期 |
| :-- | :-: | :-: | :-: | :-: | :-: |
| 精力充沛 | – | – | 无惩罚 | – | – |
| 气喘吁吁 | 困难 | 无惩罚 | 无惩罚 | 无惩罚 | 15分钟 |
| 疲倦 | 困难 | -1米 | 无惩罚 | 无惩罚 | 3小时 |
| 疲惫不堪 | 艰巨 | -2米 | -2 | 无惩罚 | 6小时 |
| 精疲力尽 | 艰巨 | 减半 | -4 | -1 | 12小时 |
| 虚弱无力 | 巨大 | 减半 | -6 | -2 | 18小时 |
| 丧失行动能力 | 巨大 | 无法移动 | -8 | -3 | 24小时 |
| 半昏迷 | 绝望 | 无法进行任何活动 | 无法进行任何活动 | 无法进行任何活动 | 36小时 |
| 昏迷 | 无法进行任何活动 | 无法进行任何活动 | 无法进行任何活动 | 无法进行任何活动 | 48小时 |

## 火灾

火灾作为武器使用或失控时，总是危险的来源。下方的“火灾强度表”给出了五种不同的热损伤强度，并附带了一些例子。“伤害”列中给出了每回合造成的伤害。由于相对较小，强度1和强度2的火源对其接触到的单个**命中部位**造成伤害。强度3和强度4的火源较大，对最近的1d4+1个**命中部位**造成伤害，表示辐射热的程度。强度5的火源同时影响所有**命中部位**。

火灾可以点燃易燃材料。如果未能立即扑灭，这些材料将在火灾强度所示的回合数内燃烧。一旦被点燃，易燃材料将持续燃烧直到物理扑灭。伤害直接作用于材料的**生命值**，忽略**护甲点数**，并作用于其下方的任何肉体。如果任其 uncontrolled 蔓延，每回合蔓延到与其强度相等的**命中部位**数量。

##### 火灾强度表

| 强度 | 例子 | 点燃时间 | 伤害 |
| :-: | :-- | :-: | :-: |
| 1 | 蜡烛 | 1d4 | 1d2 |
| 2 | 火把 | 1d3 | 1d4 |
| 3 | 篝火 | 1d2 | 1d6 |
| 4 | 充满房间的大火 | 1d2 | 2d6 |
| 5 | 火山熔岩 | 瞬间 | 3d6 |

基于火焰的攻击通常有一定百分比的机会点燃易燃材料，因为它们通常在一个回合内结束。除非另有说明，对于基于火焰的攻击，这个机会是伤害掷骰结果的5%；对于电击攻击，是伤害掷骰结果的1%。一旦被点燃，易燃材料每回合造成1d3-1点伤害，直到被物理扑灭。将此伤害直接作用于材料的**生命值**，忽略**护甲点数**，并作用于其下方的任何肉体。生物的天然护甲提供其正常保护。受害者可以尝试通过俯卧并成功进行一次体育（Athletics）检定来扑灭火焰，这需要一个**行动点数**，并且每回合都可以尝试。如果其他人以某种方式协助扑灭火焰，检定难度会降低一个等级。如果伤害掷骰结果为零，则该位置的火焰已自行熄灭。对于每个自行熄灭的位置，扑灭火焰的检定难度也会降低一个等级。

## 游戏时间

在《_Mythras Imperative_》中，时间是一个重要因素，主要用于确定事件发生的顺序，以便游戏机制在适当的时候应用。请记住，游戏内时间通常不等于实际游玩时间。有时，游戏主持人可能需要用一句话概括几天内发生的事件，例如“你们花了一周时间到达安特拉克斯城堡”，而在其他时候，尤其是在战斗中，几秒钟内的行动可能需要几分钟甚至更长时间来解决。

以下是游戏时间的重要区别：

#### 战斗回合

一个**战斗回合**代表真实世界的五秒钟。它们用于衡量短暂、狂乱的活动爆发，这些活动瞬间完成，或者衡量需要逐一解决的非常详细的活动。例如，猎人与猎物之间的追逐，每回合都决定了追逐的曲折。

#### 技能时间

**技能时间**代表几分钟到几小时，用于衡量不需要像战斗回合那样详细关注，但需要特定努力和明确结果的活动。撬锁可能需要五分钟，而观察守卫队在城堡城墙周围巡逻可能需要几个小时，两者都是技能时间的例子。

#### 叙事时间

这是游戏主持人向玩家叙述或玩家讨论时所花费的时间。除非有特殊原因，否则大多数实际的角色扮演都发生在叙事时间中。在角色扮演时，叙事时间类似于真实时间，一次对话需要多长时间就扮演多长时间。如果游戏会话包括长时间的旅行，或者不注重精确时间段的活动，那么时间会被大大压缩到几个游戏时刻。

## 命中部位

大多数成功的（武器）攻击都落在特定的**命中部位**上。这可以随机确定，或者在某些情况下，可以通过“选择部位”等**特殊效果**进行选择。要随机计算哪个部位被击中，请掷1d20并将其与相关的人形生物或生物**命中部位表**进行比较。

动物和怪物的**命中部位表**通常与人形生物略有不同，以反映其独特的生理特征。这些都列在每个生物的描述中，通常只对游戏主持人有用。

为便于参考，人形生物**命中部位表**在此重复。

| 1d20 | 命中部位 |

请注意，某些攻击，特别是魔法、科技或由特定怪物（例如龙的烈焰吐息或爆炸手榴弹）产生的攻击，可能会同时击中多个**命中部位**。单个**命中部位**可能被护甲保护，无论是穿着的防护还是生物外表的自然坚韧。这在考虑下文的**伤害和伤口等级**时很重要。

## 伤害和伤口等级

如果攻击成功击中，它会根据武器的伤害评级造成伤害。例如，一把短剑会造成1d6点伤害。

武器或攻击的掷骰伤害按以下特定顺序进行修正：

1. 应用进行攻击的角色的**伤害修正**。
2. 修正任何武器增强（或削弱）魔法造成的伤害。
3. 如果被招架，根据武器尺寸的比较适当减少伤害。
4. 如果**命中部位**有护甲，则按**护甲点数**值减少伤害。

如果攻击造成的伤害高于零，则**命中部位**承受该伤害量，伤害总数立即从该部位的**生命值**中扣除。记录该部位剩余的**生命值**，以及由此产生的伤口类别。

- **轻伤：** 命中部位仍有正生命值
- **重伤：** 命中部位生命值降至零或以下。
- **重创：** 命中部位生命值降至负值，等于或大于其起始生命值。

如果一个部位受到**重伤**或**重创**，请使用以下规则。请注意，需要进行耐力（Endurance）检定的伤口会将其与原始攻击掷骰的结果进行**对抗检定**。这反映了高技能战士进行更致命打击的能力。耐力检定不会重复，除非该部位再次受伤。

### 轻伤

**轻伤**是割伤、擦伤、瘀伤和扭伤。它们会疼痛，可能会流血，但不足以减慢或妨碍受害者。

### 重伤

如果一个部位的**生命值**降至零或以下，受害者会受到**重伤**。该部位会永久留下疤痕，并且受害者在接下来的1d3回合内无法攻击或开始施法（但仍可以招架或闪避），因为被伤口的疼痛震慑或分散了注意力。

角色肢体受到**重伤**，必须立即进行一次他的耐力（Endurance）对抗敌人成功攻击掷骰的**对抗检定**。失败会导致该肢体无法使用，直到该部位恢复到正**生命值**。如果是腿部，受害者会倒地。如果是手臂，所持物品会掉落，除非物品被绑住（此处请运用常识）。

角色腹部、胸部或头部受到**重伤**，必须立即进行一次他的耐力（Endurance）对抗敌人成功攻击掷骰的**对抗检定**。失败会导致昏迷，昏迷时间等于造成**重伤**的攻击所承受的伤害点数。急救（First Aid）或治疗（Healing）技能可用于帮助昏迷的受害者恢复意识。

在游戏主持人的裁量下，即使角色仍然能够行动，所有需要使用该**命中部位**的任务都将遭受持续一个**难度等级**的惩罚，直到伤势降至**轻伤**。

### 重创

如果一个部位的**生命值**降至负值，等于或大于其起始**生命值**，角色会受到**重创**。角色立即**丧失行动能力**，无法继续战斗。肢体可能会因**重创**而被切断、刺穿、粉碎或撕裂。角色倒地，身体**丧失行动能力**，必须立即进行一次耐力（Endurance）对抗敌人成功攻击掷骰的**对抗检定**。失败会导致因剧痛而昏迷。如果一个被切断、刺穿或撕裂的部位在相当于其**治愈率**五倍的分钟数内未得到治疗，角色会因失血和休克而死亡。

腹部、胸部或头部受到**重创**的受害者会昏迷，完全**丧失行动能力**，并且必须立即进行一次耐力（Endurance）对抗敌人成功攻击掷骰的**对抗检定**。失败会导致即刻且血腥的死亡（斩首、劈成两半、心脏被刺穿、撕裂等等）。如果他们幸存下来，并且该部位在相当于角色**治愈率**两倍的**战斗回合**数内未得到治疗，他们仍会因失血和休克而死亡。

由于大多数**重创**都需要某种形式的手术或强大的魔法才能治愈，受害者极不可能及时从**重创**中恢复并重新加入战斗。根据可用的治疗，受伤的部位可能会致残（参见下文的[伤势愈合](0004_Game_System.md?id=healing-from-injury)）。

### 失血

因外伤和内伤（通常在战斗中造成）导致的**失血**会很快耗尽角色的体力。其影响通过累积**疲劳等级**来衡量，很像窒息。不受控制的**失血**通常会导致死亡，除非得到某种形式的医疗救助。与窒息不同，严重**失血**对疲劳的影响是持久的。角色以每天一级疲劳的速度恢复因出血而失去的疲劳，从止血后的第二天开始。

### 英勇的最后行动

在关键角色受到**重创**的特殊情况下，游戏主持人可能会允许一次英勇的最后行动——例如，吐出最后一个法术，试图击倒他们的杀手，爬走躲藏，或者试图包扎自己的伤口。他们必须有适用于该行动的可用肢体并消耗一个**幸运点**。他们只有一次机会尝试他们的行动，然后立即陷入昏迷，并可能死亡。

## 伤势愈合

伤口和损伤的自然愈合基于角色的**治愈率**。**治愈率**决定了每个部位根据损伤性质恢复多少**生命值**：

- **轻伤：** 天
- **重伤：** 周
- **重创：** 月

因此，一个**治愈率**为3的角色，在一个**命中部位**受到伤害导致其生命值降至-3（即**重伤**），将以每周3点生命值的速度自然愈合，直到伤口恢复到零以上，然后以每天3点生命值的速度愈合，直到完全康复。

自然愈合有一些限制：

- 正在愈合的角色不能进行剧烈活动：否则，**治愈率**会减少1d3。因此，即使是从**轻伤**中恢复的角色，如果他们决定从事任何可能加重伤势的体力任务，他们的恢复进程也可能受阻。
- **重创**在受害者未成功进行治疗（Healing）检定或施放某种形式的治疗魔法之前，自然愈合不会开始治愈。未被截肢的**重创**如果未在相当于治疗（Healing）技能值二十分之一天数内得到治疗，则会致残，永久性地降低该部位的**生命值**。

## 无生命物体

所有无生命物体都拥有**护甲点数**和**生命值**，用于确定对伤害和破坏的抵抗力。**护甲点数**在**生命值**受到影响之前减少伤害。一旦物体的**生命值**降至零，它就变得无用。

破坏尝试要求角色进行适当的蛮力（Brawn）、徒手（Unarmed）或战斗风格（Combat Style）检定，以造成伤害。成功会对其造成伤害。

#### 武器对抗无生命物体

使用武器攻击**护甲点数**等于或大于该武器的无生命物体时，会对物体和武器都造成伤害。例如，用斧头砍铁门可能会损坏门，但斧头本身会在铁门被毁坏之前很久就先毁坏。游戏主持人应酌情判断哪些武器或工具可以影响哪些物体。例如，凿子是专门设计用来雕刻木头或石头的，而剑虽然锋利，却不是。因此，凿子在雕刻物体时不会受到伤害，而剑很可能会受到伤害。

## 幸运点

**幸运点**有助于区分英雄与普通人。它们代表角色将失败转化为成功甚至欺骗死亡的能力。

**幸运点**可以在游戏中途使用，并在下次游戏会话开始时恢复到其通常值。

#### 使用幸运点

**幸运点**可以通过多种方式使用。一次行动只能使用一个**幸运点**。**幸运点**不能用于经验或训练检定。以下每个选项都花费一个**幸运点**。

#### 欺骗命运

角色可以使用一个**幸运点**来重掷或交换（例如，75会变成57）他们进行的任何骰子检定。这可以是技能检定、伤害掷骰，或任何其他有影响的检定。角色甚至可以强制对手重掷针对他们进行的攻击或伤害掷骰。

#### 绝望的努力

如果角色在战斗中用尽了**行动点数**，需要寻找最后爆发的绝望能量来避免悲惨的结局，他们可以花费一个**幸运点**来获得一个额外的**行动点数**。

#### 减轻伤害

遭受**重创**的角色可以花费一个**幸运点**将伤势降级为**重伤**。这会将所受伤害减少到比造成**重创**所需的伤害少一点。
