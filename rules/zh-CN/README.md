# Mythras Imperative 系统资源文档

## 介绍

_Mythras Imperative_ 是 _Mythras_ 规则集的一个开放的 ORC 授权版本。本文档提供了一套全面但不详尽的规则，包括角色创建、技能、某些典型情况的规则、战斗和魔法——足以让玩家和游戏主持人入门，并为进一步发展奠定坚实基础。_Mythras Imperative_ 与 Mythras 核心规则和其他 ORC 授权的百分比游戏系统兼容。其配套卷册 _Classic Fantasy Imperative_ 也完全兼容并根据 ORC 发布，这意味着两个 Imperative 可以轻松结合使用。

## ORC 授权

_Mythras Imperative_ 使用 ORC 许可发布。ORC 许可为游戏创作者提供了一种公开分享其游戏机制的方式，并允许下游用户自由使用、修改和重新分享这些游戏机制的改编版本。

ORC 公开分享的内容被称为 _**授权材料**_，包括游戏的广泛功能元素，例如属性块、游戏规则、角色属性以及游戏固有的方法和系统，以及许可人明确希望分享的任何其他内容。此许可范围广泛，允许您在全球范围内将授权材料与印刷书籍、视频游戏、播客、AI 或任何可能存在或将来创建的技术结合使用。_Mythras Imperative_ 中绝大部分内容均构成授权材料。

ORC _不_ 分享的内容被称为 _**保留材料**_，包括商标、世界背景、故事线、独特角色和视觉艺术。我们已在下方的 ORC 通知中列出了保留材料。

根据 ORC 许可，您可以随意使用、分享、改编和基于授权材料进行创作，只要您在发布的著作中以通知形式给予适当的署名。此通知既认可了上游许可人及其对著作的贡献，也为下游用户提供了方便的方式来认可该通知所在著作的作者。通过使用 ORC 授权的材料，您将自动同意在您发布的著作中以相同条款许可您的游戏机制。如果您开发自己的保留材料，则其为您的专属财产。

## d100 新手？

_Mythras Imperative_ 是一个 d100 或百分比系统。百分比骰子（1d100）用于解决关键动作——技能使用、战斗、魔法施放等。它是一个“向下掷骰”系统，这意味着目标数字以百分比表示（例如，运动 65%），游戏主持人和玩家都需要使用 1d100 掷出等于或小于此目标数字，才能取得成功。如果愿意，可以很容易地用 d20 替换 1d100，并且所有百分比都可以除以 5（向上取整）以获得 d20 掷骰的目标数字。在我们的例子中，运动 65% 将变为运动 13。

然而，1d100 提供了大量的灵活性和细微差别，这是整个游戏的基础，因此我们建议您首先尝试 1d100 方法。虽然掷出 _小于_ 某个数字可能看起来有悖常理（在许多游戏中，通常越大越好！），但将目标数字表示为一个需要掷入的百分比范围，可以轻松简单地看到成功的机会，并允许进行相当多的“骰子技巧”，例如对抗掷骰、暴击和失误，以及这些规则后面探讨的其他一些内容。百分比角色扮演自角色扮演的早期便已存在，是一个拥有久经考验的核心机制的成熟游戏系统，该机制灵活、直观，并且在游戏过程中往往能融入背景。

## 简化

_**Mythras**_ 的长期拥有者会注意到，在某些地方 _Mythras Imperative_ 规则已被简化。这样做是为了让游戏主持人能够更快地向新玩家介绍游戏，并最大限度地减少麻烦。

---
## 数字和结果的四舍五入

在某些情况下，您需要对数字进行除法——通常是技能的评级（例如确定暴击，其值为技能值的 1/10）。每当除法结果产生小数时，总是向上舍入到整数。因此，例如，63% 的 1/10 是 6.3；这被向上舍入为 7。

## 使用的骰子

_Mythras Imperative_ 使用标准的多面体骰子组：d4、d6、d8、d10、d12、d20 和 d100。此外，还要求使用以下骰子：

d2：掷任何一个骰子。奇数等于 1，偶数等于 2。或者抛硬币。

d3：掷一个六面骰；1-2 = 1，3-4 = 2，5-6 = 3
