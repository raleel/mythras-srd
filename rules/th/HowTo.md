# ข้อกำหนดเบื้องต้น

1. ความเข้าใจพื้นฐานเกี่ยวกับ [ภาษา Markdown](https://www.markdownguide.org/basic-syntax/)

2. คุณจะต้องสร้างบัญชี [github](https://github.com/) ฟรี หากคุณยังไม่มี ขอแนะนำให้ใช้ชื่อของ RPG ของคุณ เนื่องจากชื่อนี้จะเป็นส่วนหนึ่งของ URL ที่ผู้คนจะใช้ในการเข้าถึง SRD ของคุณ

3. คุณสามารถทำกระบวนการทั้งหมดนี้ผ่านเบราว์เซอร์ได้โดยไม่ต้องรู้เรื่อง Git แม้ว่าการเรียนรู้ Github Desktop จะช่วยให้คุณสามารถทดสอบการเปลี่ยนแปลงใดๆ ในเครื่องก่อนที่จะนำไปใช้งานจริงให้โลกเห็น

> หากคุณมีชื่อโดเมนที่ลงทะเบียนไว้ คุณสามารถใช้กระบวนการนี้เพื่อสร้าง SRD ได้เหมือนกับที่ผมมี https://srd.7thextinctionrpg.com

## การเพิ่มเนื้อหาของคุณ

### ไฟล์ *.md

ไฟล์ .md คือไฟล์ Markdown, Markdown เป็นภาษามาร์กอัปน้ำหนักเบาที่คุณสามารถใช้เพื่อเพิ่มองค์ประกอบการจัดรูปแบบลงในเอกสารข้อความธรรมดาได้ ขึ้นอยู่กับขนาดของเนื้อหาของคุณ คุณสามารถใส่ข้อมูลเกมทั้งหมดของคุณในไฟล์ *.md ไฟล์เดียว หรือเหมือนที่ผมมีที่ [7thextinctionrpg.com](https://srd.7thextinctionrpg.com) สร้างไฟล์แยกต่างหากสำหรับแต่ละบท ไฟล์ "sample.md" มีตัวอย่างการจัดรูปแบบบางอย่าง

ภาษา Markdown เรียนรู้ได้ง่ายมาก ใช้ลิงก์ที่ให้ไว้ในข้อ 2 ภายใต้ข้อกำหนดเบื้องต้นด้านบนสำหรับรายการไวยากรณ์ทั้งหมด

### _Sidebar.md

นี่คือเมนูของคุณ แผงนำทางทางด้านซ้ายของเว็บไซต์ เมื่อคุณเพิ่มไฟล์ Markdown ลงใน Repository ของคุณแล้ว คุณต้องเพิ่มไฟล์นั้นที่นี่เพื่อให้เนื้อหาของไฟล์นั้นถูกรวมเข้าด้วย ทำตามการจัดรูปแบบที่มีอยู่

## เปิดใช้งาน Github Pages เพื่อเปลี่ยน Repository ให้เป็นเว็บไซต์

1. ใน Repository ของคุณ คลิกที่ 'Settings'

![Website-Step1](/_media/Website-Step1.png)

2. เลือก 'Pages' ในเมนูด้านซ้ายมือ

![Website-Step2](/_media/Website-Step2.png)

3. หากคุณยังไม่ได้ตั้งค่า Repository เป็น 'public' คุณมีสองทางเลือก

5. รอ 10-20 วินาทีแล้วรีเฟรชหน้า

6. ตอนนี้คุณจะเห็น URL ของคุณ เช่น https://coweater.github.io/mysrd/

## การใช้ปลั๊กอิน Fontawesome และ/หรือ Material Icons เพื่อแสดงไอคอนฟรีใน SRD ของคุณ

### Fontawesome

ใช้ Fontawesome เพื่อรับไอคอนฟรีและไอคอนเคลื่อนไหว ผู้ใช้ที่สมัครสมาชิกสามารถใช้ Pro Icons ได้

1. ค้นหาไอคอนบน [Fontawesome](https://fontawesome.com/icons)
2. เลือกไอคอน
3. ในหน้าต่างป๊อปอัป ภายใต้แท็บ HTML คัดลอกทุกอย่างระหว่าง " "

```EXAMPLE
<i class="fa-solid fa-dice-d20"></i>
```

4. วางข้อความที่คัดลอกมาระหว่าง `:` โดยไม่มีช่องว่าง

```EXAMPLE
fa-solid fa-dice-d20
```

### Material Icons

ใช้ Material Icons สำหรับไอคอนฟรีเพิ่มเติม

1. ค้นหาไอคอนบน [Material Icons](https://fonts.google.com/icons)
2. คัดลอกและแทรกลิงก์ web span โดยตรงลงในไฟล์ Markdown ของคุณ

```markup
<span class="material-icons">face</span>
```
