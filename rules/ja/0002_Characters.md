# キャラクター

『ミストラス・インペラティブ』では、すべてのプレイヤーがキャラクターを操作します。キャラクターはプレイヤーの分身であり、通常、肉体的にも精神的にもプレイヤーとは大きく異なります。キャラクターはゲームの設定に応じてさまざまな種族から選べます。歴史的な設定では人間のみが登場しますが、ファンタジー設定ではドワーフ、エルフ、さらにはリザードマンも登場し得ます。基本的なキャラクターは、その能力を記述する一連の異なる要素によって定義されます。これらの要素は以下の通りです。

- **特性値（Characteristics）：** 筋力やカリスマなど
- **属性（Attributes）：** 身長や移動力など
- **技能（Skills）：** 基礎的な能力における専門知識

これらの要素のほとんどは数値で記述され、一部は他の要素に依存したり、それらによって算出されたりします。各要素が何を意味し、どのように決定されるかについては、以下のセクションで説明します。

---
## ステップ1：キャラクターコンセプト

まず、どのようなキャラクターをプレイしたいか、ある程度のアイデアを持つことから始めると良いでしょう。例えば、狡猾な戦士や、ずる賢い密輸業者、あるいは勤勉な学者、賢明な魔術師が良いかもしれません。この段階では、キャラクターコンセプトは凝ったものである必要はありません。文化や経歴などの選択を導くための単なるアイデアで構いません。

---
## ステップ2：特性値

誰もが7つの特性値によって定義されており、キャラクターについて、どれほど強いか、速いか、賢いか、健康であるかといった情報を示します。特性値はすべての『ミストラス・インペラティブ』キャラクターの核心であり、属性や技能などの他のほとんどの要素の基礎を形成します。7つの特性値は以下の通りです。

- 筋力 (STR)
- 体力 (CON)
- 体格 (SIZ)
- 器用度 (DEX)
- 知力 (INT)
- 精神力 (POW)
- 魅力 (CHA)

特性値を計算する前に、以下のセクションでそれぞれが何を表すかを説明します。

#### 筋力 (STR)

STRは肉体的な強さを表します。どれだけ持ち上げられるか、どれだけ強く打てるかなどです。STRはダメージ修正の構成要素です。キャラクターのSTRがゼロに減少すると、物体を動かしたり持ち上げたりする能力を失います。

#### 体力 (CON)

CONは健康と頑健さの指標です。CONはヒットポイントと回復力の決定要素です。何らかの理由でCONがゼロに落ちた場合、キャラクターは死亡します。

#### 体格 (SIZ)

SIZは質量を測定し、身長と体重を示すのに役立ちます。SIZはヒットポイントを算出するのに使用され、より大きく重いクリーチャーはダメージに対する耐久性が高くなる傾向があるためです。SIZはキャラクターのダメージ修正を決定するのにも使用でき、質量は打撃の力を増加させるのに役立ちます。

#### 器用度 (DEX)

敏捷性、平衡感覚、反射神経はDEXによって測定されます。これはアクションポイントとイニシアチブの重要な側面です。キャラクターのDEXがゼロに減少すると、機能的な麻痺に苦しみます。

#### 知力 (INT)

INTは認知能力の測定値です。INTスコアが低い者が必ずしも愚かであるとは限りませんが、機知をどれほど創造的に活用できるかについては制約を受ける可能性が高いでしょう。INTはアクションポイントとイニシアチブの計算要素です。INTがゼロに減少すると、キャラクターは完全に意識を失います。

#### 精神力 (POW)

POWはキャラクターの魂、精神、内なる衝動、または魔術の能力の測定値です。POWはキャラクターの魔力点と幸運点を支配します。キャラクターのPOWがゼロに落ちた場合、すべての独立した意思を失います。

#### 魅力 (CHA)

カリスマは存在感と個性の測定値であり、身体的外見とは無関係です。CHAはキャラクターの経験点修正に影響します。CHAがゼロに落ちた場合、キャラクターは極度に内気になったり反社会的になったりして、他者と社会的に交流できなくなり、無視されたり、追い払われたりすることもあります。

---
### 特性値の計算

各特性値には、その強さを決定する数値があります。各ダイスロールの後の括弧内の数値は種族の平均値であり、あなたのキャラクターの能力をその種族の他のメンバーと比較する際の目安として使用されます。特性値を決定するための提案は以下の通りです。

- **ダイスロール：** STR、CON、SIZ、DEX、INT、POW、CHAにダイスを振り、出目をそのまま適用します。これらのロールの結果はややランダムになる可能性があるため、ダイスを振った後にキャラクターコンセプトを決定する方が良い場合が多いです。そうしないと、最終的な特性値が想定していた役割を妨げる可能性があります。STR、CON、DEX、POW、CHAには3d6を、SIZとINTには2d6+6を振ります。
- **ダイスロール、割り振り：** 上記と同様ですが、プレイヤーは同じダイス範囲の特性値間で値を交換できます。例えば、人間を作成する場合、プレイヤーはSTR、CON、DEX、POW、CHAの間で値を交換できます。これらは3d6の範囲を共有しているためです。またはSIZとINTの間で交換できます。これらは2d6+6の範囲を共有しています。ただし、常に同じダイス範囲グループ内である必要があり、あるグループのロールを別のグループに切り替えることはできません。
- **ダイスロール、高値選択：** STR、CON、SIZ、DEX、INT、POW、CHAについて、示されたダイスに加えて1つ余分に振り、それぞれから最低の出目を削除します。
- **ポイントビルド：** プレイヤーは80点のあらかじめ決められたプールから特性値スコアを割り当てます。特性値は最低ロール値よりも低く、最高ロール値よりも高くすることはできません。

## ステップ3：属性

各キャラクターは一連の属性も持っています。これらは特性値から派生するか、特性値によって決定され、ゲームプレイの特定の要素を管理するために使用される能力です。

#### アクションポイント

キャラクターが戦闘ラウンドでどれくらいの頻度で行動できるかは、アクションポイントによって決定されます。『ミストラス・インペラティブ』のすべてのキャラクターは2アクションポイントを持っています。

#### ダメージ修正

より大きく、より強いキャラクターやクリーチャーは、特定の状況で追加のダメージを得る場合があります。逆に、より弱く、より小さなキャラクターは、与えることができるダメージを減少させる場合があります。ダメージ修正は通常、戦闘状況で使用されますが、物体を破壊しようとする際にも使用できます。ダメージ修正は追加のダイスロールであり、武器や道具によって与えられるダメージに加算または減算されます。負のダメージ修正が武器のダメージをゼロ以下にした場合、ダメージはまったく与えられません。

#### 経験点修正

プレイの過程でキャラクターは技能と能力を向上させます。これは経験点ロールを使用して達成され、詳細については[ゲームシステム](0004_Game_System.md?id=experience-rolls)セクションで説明されています。

キャラクターのCHAスコアは、キャラクターが受ける経験点ロールの数を調整する場合があります。これは仲間との関係やコミュニティでの評判を反映しています。CHAが高い場合、人々はキャラクターの訓練や支援のために協力することをいとわないでしょう。逆に、CHAが特に低い場合、他者の助けなしに自身の能力を向上させるのが難しいかもしれません。例えば、スパーリング相手を見つけるのが難しいといった場合です。

|:-:| :-:|
| 6以下 | -1 |
| 7-12 | +0 |
| 13-18 | +1 |
| 以後6点毎 | +1 |

#### 回復力

負傷した後、キャラクターは回復する必要があります。回復力は、傷から自然に回復する速さを決定します。負傷の重症度に応じて、回復力は1日、1週間、または1ヶ月あたりに回復するヒットポイントの数を表します。

##### 回復力表

| CON | 回復力 |
| :-:| :-:|
| 6以下 | 1 |
| 7-12 | 2 |
| 13-18 | 3 |
| 以後6点毎 | +1 |

#### ヒットポイント

ヒットポイントは、身体の部位がどれだけの負傷に耐えられるかを表し、それ以上になると機能しなくなり、場合によっては行動不能になり、最終的にはキャラクターの死に至ります。すべてのクリーチャーの体は別々の部位に分かれています。例えば、人間には頭、胸、腹部、腕、脚の7つの部位があり、それぞれが独自のヒットポイントを持っています。

モンスターや異なる種類の動物など、一部のクリーチャーは、その性質上、根本的に異なる身体構造を持つことが多いですが、彼らが持つ各部位には依然としてヒットポイントがあります。部位のヒットポイントが何らかの種類のダメージによって特定のレベルまで減少すると、キャラクターはその負傷の重症度を反映した特定の結果を被ります。負傷に関する詳細については、[ダメージと負傷レベル](0004_Game_System.md?id=damage-and-wound-levels)セクションを参照してください。

#### イニシアチブ

戦闘での反応のタイミングはイニシアチブによって管理されます。イニシアチブはイニシアチブロールへの修正として機能します。イニシアチブが高いほど、戦闘状況での反応が速くなり、いつ行動できるかが決定されます。例えば、鎧などのさらなる要素がこれを修正します。[戦闘](0005_Combat.md?id=combat-style)の章では、イニシアチブがどのように使用されるかについて詳しく説明しています。

イニシアチブはDEXとINT特性値の平均値です。

#### 幸運点

幸運点は、冒険する英雄と一般の人々を区別する奇妙な力を表します。これを運命、カルマ、あるいは単なる幸運と呼んでも良いでしょう。幸運点は以下に使用できます。

- 不利なダイスロールを再ロールする。
- 物理的ダメージやその他の不運な状況を軽減する。
- 戦闘の重要な瞬間に優位を得る。

それらがどのように、いつ使用されるかの詳細は、[ゲームシステム](0004_Game_System.md?id=game-system)セクションで説明されています。幸運点が消費されると、プールは減少します。幸運点がなくなると、ゲームマスターが臨時で与えない限り、次のゲームセッションで通常の値に補充されるまで利用できません。

##### 幸運点表

| POW | 幸運点 |
| :-- | :-: |
| 6以下 | 1 |
| 7-12 | 2 |
| 13-18 | 3 |
| 以後6点毎 | +1 |

#### 魔力点

『ミストラス・インペラティブ』で使用される神秘的な能力、パワー、呪文は、通常、魔力点に依存します。魔法の利用可能性はキャンペーン設定や特定の魔法の職業によって制限される場合があるため、一部のキャラクターはこの属性を不要と感じるでしょう。

そのような力を使用できるキャラクターの場合、魔力点はPOWに等しくなります。これらの点は呪文を唱えるために使用され、そのコストは使用される魔法の種類に依存します。術者が魔力点を使い果たすと、魔力点が補充されるまで呪文を唱える能力を使い果たします。

#### 移動力

すべてのクリーチャーは移動力を持っています。これは特定の時間内に移動できるメートル数です。移動力は特性値から計算されるのではなく、種族ごとに異なるデフォルト値です。人間の基本移動力は6メートルです。

歩行よりも速い移動方法（ゲイト）は、移動力に倍率を適用します。
- **走行（Run）：** 移動力 x3
- **疾走（Sprint）：** 移動力 x5

## ステップ4：技能

技能は2つの種類に分類されます。誰もが持っている共通技能（Standard skills）は、隠密行動や他者に影響を与えることなどです。一方、専門技能（Professional skills）は、試行する前に特定の訓練が必要なもので、野生動物の追跡、宇宙船の操縦、古代言語の理解などがこれに当たります。キャラクターには、技能を獲得するために3つのポイントプールが与えられます。最初のプールは育った文化のため、2番目のプールは幼少期後に選択する経歴のため、最後のプールは技能をさらに補強するためのものです。デフォルトでは、新しいキャラクターは文化技能プールに100ポイント、経歴技能プールに100ポイント、さらに150ポイントのボーナスポイントを割り当てて補強します。各文化と経歴には、標準技能と専門技能の定義された範囲があり、これらの技能ポイントを割り当てることができます。

##### 戦闘スタイル

戦闘スタイルは以下の文化記述に記載されています。戦闘スタイルについては、[戦闘](0005_Combat.md?id=combat-style)セクションで詳しく説明されており、基本成功率（Basic Percentage）はSTR+DEXに等しいです。

## ステップ5：文化

文化は、キャラクターが育った社会の種類を決定し、それによって彼または彼女が持つ可能性のある特定の見方や哲学を規定します。より実用的には、文化はキャラクターの全体的な能力を形作るさまざまな技能の値を増強するのに役立ちます。人間の基本的な文化には、野蛮人（Barbarian）、文明人（Civilized）、遊牧民（Nomadic）、原始人（Primitive）の4つがあります。各文化的背景には、それに適切な共通技能の範囲がリストされており、プレイヤーは提示された専門技能の中から最大3つを選択できます。このようにして、同じ文化的背景を持つプレイヤーであっても、彼らの冒険者が開発した分野で異なるようにすることができます。文化技能ポイント100点を、経歴にリストされた共通技能と選択した専門技能に割り振り、1ポイントを費やすごとに各技能を1%増加させます。ゲームマスターは、特定の技能に適用できる技能ポイントの数に上限を設けることを検討しても良いでしょう。

### 野蛮人

部族社会の野蛮人は文明を嫌い、大きな町や都市に住む人々を弱くて堕落していると見なす傾向があります。彼ら自身の集落も定住型ですが、はるかに小さく、自然に近い場所にあります。野蛮な部族は、狩猟、放牧、農業のために自分たちのものと見なす特定の領域、しばしば半分手つかずの荒野を占有しています。彼らのほとんどは、野生のクリーチャーに直面したり、ライバル部族から土地を守るために集まったりするなど、人生を通して多くの危険を克服しなければならないため、武器の使用に熟練しています。小規模なコミュニティで生活しているため、親族や氏族の絆は彼らの社会の強い要素です。

#### 技能

- **共通技能：** 運動、腕力、耐久、応急手当、地誌、知覚、そして操船か乗馬のいずれか。さらに、プレイヤーは文化的な戦闘スタイルを選択することもできます。
- **専門技能：** 工芸（任意）、治療、知識（任意）、音楽、航海、操船術、サバイバル、追跡。

### 文明人

文明文化は、複雑な社会規範に支えられ、官僚制によって裏打ちされた法と秩序の体裁によって象徴されます。文明人は、彼らの功績が恒久性を持つ傾向があるため、他のすべての文化よりも優れていると信じています。広大な都市、そびえ立つ寺院、記念碑、書物、墓石、巻物への歴史と出来事の記録などです。もちろん、これは薄っぺらい見せかけです。最も文明化された都市でも、野蛮な辺境の集落よりも無法であることがあります。社会規範は遊牧民の歓待の儀式よりも粗野な場合もあります。しかし、文明文化は、その功績、インフラ、専門職の職人、そして望む場所ならどこへでも容易にその影響力を及ぼせる能力を誇りにしています。

#### 技能

- **共通技能：** 隠蔽、欺瞞、運転、影響、洞察、地誌、意志。さらに、プレイヤーは文化的な戦闘スタイルを選択することもできます。

### 遊牧民

遊牧民は常に移動しており、自分の家と呼べる場所がありません。彼らは目的もなくさまよったり、一年を通していくつかのキャンプ地を移動したりするかもしれません。彼らはほとんど作物を育てず、代わりに動物の移動や魚の群れを追ったり、自分たちの家畜を飼育したりします。遊牧民は、周囲から素早く簡単に調達できるもので生活することに長けており、何も無駄にしません。遊牧民文化の技能は、彼らの環境に合わせて調整されるべきです。ある者は徒歩で広大な距離を旅し、ある者はキャラバンで生活したり奇妙な生物に乗ったりし、またある者は巨大な浮遊いかだに乗って大洋を漂流します。

### 原始人

しかし、迷信に基づいた特定の文化的信念は非常に明確である場合があります。文字言語に近いものを開発した原始文化はほとんどありませんが、絵画やその他の絵記号がコミュニケーション手段として機能します。

## ステップ7：ボーナス技能ポイント

最後のステップとして、キャラクターを補強するために、150ポイントの最後の技能ポイントプールを割り当てます。これらは、共通技能、または文化や経歴の一部として獲得した専門技能に適用できます。ここでも、ゲームマスターは特定の技能に適用できる技能ポイントの数に上限を設けることがあります。

### 経験豊富なキャラクターの作成

初心者キャラクターは、経験がほとんどない状態でゲームを開始することが推奨されます。もちろん、キャラクターのレベルは設定や、ゲームが短い単発の冒険であるか、長期キャンペーンであるかによって大きく異なります。また、プレイ中に死亡したキャラクターを置き換えたり、既存のキャンペーンに参加したりする場合に、プレイヤーがより経験豊富なキャラクターを作成することもよくあります。

年齢カテゴリごとに経験を分類した表が提供されています。キャラクターの経験は年齢と技能の能力に直接関連しており、経験豊富なキャラクターは若い仲間よりも当然多くの人生経験を持っています。ただし、記載されている年齢ボーナスは概算として扱うべきです。キャンペーンは異なる速度で進行するため、冒険の間に多くのゲーム内時間が経過するキャンペーンもあれば、一連の途切れない冒険であるキャンペーンもあります。既存のキャラクターを変換する場合は、表に記載されている実際の年齢を単に置き換えてください。

#### 経験表

| カテゴリ | 年齢 | ボーナス技能ポイント | 最大技能増加 |
| :-- | :-: | :-: | :-: |
| 若年 | 10+1d6 | 100 | +10 |
| 成人 | 15+2d6 | 150 | +15 |
| 中年 | 25+3d6 | 200 | +20 |
| 高齢 | 40+4d6 | 250 | +25 |
| 老年 | 60+5d6 | 300 | +30 |

#### 経験表の注記

- ボーナス技能ポイントは、キャラクターが技能開発に費やすことができるポイントの数を決定します。
- 最大技能増加は、ボーナスポイントのうち、任意の1つの技能に適用できるポイントの数を制限します。

### 技能ピラミッド

『ミストラス・インペラティブ』の技能ポイント割り振りプロセスは、文化（100）、経歴（100）、ボーナス技能ポイント（150）の3つのソースから合計350ポイントを導き出します。これらのポイントを割り振ることは時間がかかる場合があり、キャラクターの文化を決定することがそれほど重要でない場合や、プレイヤーがコア・ルールで提供されているテンプレートでは対応できない特定の職業を考えている場合もあるでしょう。時間が差し迫っている場合やプレイヤーが熱心な場合のために、技能ポイントを割り振る代替システムをここに示します。

文化、経歴、ボーナス技能ポイントのプールからポイントを割り振る代わりに、技能ピラミッドを使用します。これは350ポイントを5つの固定ポイントのブロックに分割します。

1つの技能に50%、2つの技能に40%、3つの技能に30%、4つの技能に20%、5つの技能に10%。

これにより便利なピラミッドが形成されます。割り振るたびに各値を消してください。

```
    50

   40 40

  30 30 30

 20 20 20 20

10 10 10 10 10
```

#### いくつかの簡単なルール

- ピラミッドからのすべてのポイントを割り当てる必要があります。
- どの技能もピラミッドからの割り当てを複数回受けることはできません。
- 「慣習」と「母国語」は、コア・ルールに従い、ピラミッドからの割り当てに加えて、両方とも+40を得ます。
- [情熱](0002_Characters.md?id=step-8-passions)セクションに記載されている値を得る「情熱」も、ピラミッドからの割り当てに加えて、その値を得ます。
- いかなる技能も、割り当ての結果、80%を超えることはできません。したがって、キャラクターのSTRが16、DEXが17で基本が33%の場合、プレイヤーは戦闘スタイルに50ポイントの値を割り当てることはできません。
- 6つの値は専門技能に割り当てる必要があります。
- キャラクターは引き続き文化と職業を選択でき、適切な記述にリストされている技能を割り当てのガイドとして使用できます。または、割り当ては完全に自由でも構いません。
- 少なくとも1つの値を戦闘スタイルに割り当てることをお勧めしますが、これは必須ではありません。
- ゲームマスターが常に最終決定権を持ちます。

### 規格外の英雄的行為

『ミストラス・インペラティブ』は主に通常の人間を扱いますが、時折、特別な英雄的行為を要求するキャンペーンや、ヒーローやヴィランが現実離れしているパルプアクション設定で行われるキャンペーンでは、別の何かが必要になることがあります。

そのような英雄は、必ずしも超能力を持っているわけではありませんが（ただし、これらが望まれる場合は、魔法とパワーの章を使用できます）、彼らは少し強く、少し速く、少し回復力があり、そして通常はるかに強い意志を持っています。キンメリアの特定の野蛮人、殺人許可証を持つ特殊エージェント、青銅の男、億万長者のプレイボーイでもあるマントをまとった自警団員など、多くの人気のある英雄的キャラクターを想像してください。彼らは通常の人間なら崩壊するような状況に頻繁に直面しますが、英雄として、プロットが彼らに投げかけるものすべてに耐え、次の冒険に（比較的）無傷で参加することができます。

これらのキャラクタータイプをゲームに導入するつもりなら、すべてのプレイヤーが同じレベルのキャラクターを持つようにしてください。これらのキャラクタータイプは、風景を噛み砕き、アクションを推進するように設計されています。平凡なキャラクターは（魔法や他の利点がない限り）影に隠れてしまうかもしれません。

### パルプヒーローキャラクター

これらは伝統的なパルプヒーローです。彼らはスーパーヒーローや半神ではなく、狙いの良いショットガンの一撃で命を落とす可能性はありますが、彼らはより頑丈で、より賢く、より有能で、一般的に普通の人々よりも1、2段階上の存在です。また、彼らの人生は冒険に満ちている傾向があります。英雄的キャラクターは落ち着いて平穏な生活を送ろうと選択できますが、しばしば問題や助けを求める人がキャラクターのドアを訪れるでしょう。

#### 特性値

パルプヒーローキャラクターを生成する場合、[キャラクター](0002_Characters.md?id=calculating-characteristics)セクションの「ダイスロール、割り振り」方法を使用します。4d6を振り、最低のダイスを捨てて、これを5回繰り返し、その結果をSTR、CON、DEX、POW、CHAに割り当てます。次に、3d6+6を振り、最低のダイスを2回捨て、これらの数値をSIZとINTに割り当てます。ポイントビルド方法を使用する場合、プレイヤーは90点のあらかじめ設定されたプールからキャラクターをビルドします。

#### 属性

これらは通常通り計算されますが、パルプヒーローキャラクターは以下のうちから2つの「利点」を得ます（利点は重ねて取得できません）。

- +1 アクションポイント
- +1 幸運点
- 各ヒット部位に+1ヒットポイント
- 耐久ロールが1段階容易になる
- 隠密ロールが1段階容易になる
- 意志ロールが1段階容易になる

すべてのパルプヒーローキャラクターは、軽傷または重傷からの回復力が2倍になりますが、致命傷はキャラクターの標準回復力で処理されます。

#### 技能

パルプヒーローキャラクターは、必要に応じて割り振るための追加の50ボーナス技能ポイントを得ます。

### パラゴンキャラクター

パラゴンキャラクターは卓越性の典型です。鞭を振るう考古学者や殺人許可証を持つスパイがパルプヒーローキャラクターであるのに対し、キンメリアの野蛮人、青銅の男、ダークナイト探偵、シャーロック・ホームズはパラゴンキャラクターです。パラゴンキャラクターは、単に1つの専門分野で非常に有能なだけでなく、通常、いくつかの分野で優れています。優秀な発明家で物理学者であり、同時に二拳銃の腕利きで戦争の英雄でもある人物は、パラゴンキャラクターの素晴らしい例です。世界最高の探偵たちに学んだ億万長者のプレイボーイ犯罪ファイターもそうです。パラゴンキャラクターは、複数の大学の学位、独立した富、謎の支援者、そして数多くの他の経験を持っているかもしれません。

結果として、パラゴンキャラクターはすべての人向けではありませんが、設定が太陽系全体の運命を変えるようなキャラクター、またはどんな挑戦にも立ち向かえるキャラクターを必要とするなら、パラゴンレベルのキャンペーンが適しています。通常のキャンペーンや英雄的なキャンペーンでも、国家や惑星の運命といった非常に高い賭けが時折関わることがありますが、パラゴンキャンペーンではそれが日常的に行われるべきです。

#### 特性値

[キャラクター](0002_Characters.md?id=calculating-characteristics)セクションの「ダイスロール、割り振り」方法を使用してキャラクターを生成する場合。4d6を振り、最低のダイスを6回捨て、選択した5つの結果（通常は最高の5つ）をSTR、CON、DEX、POW、CHAに割り当てます。次に、3d6+6を振り、最低のダイスを3回捨て、選択した2つの結果（通常は最高の2つ）をSIZとINTに割り当てます。ポイントビルド方法を使用する場合、プレイヤーは100点のあらかじめ設定されたプールからキャラクターをビルドします。

#### 属性

これらは通常通り計算されますが、パラゴンキャラクターは以下のうちから3つの「利点」を得ます（利点は重ねて取得できません）。

- +1 アクションポイント
- +2 幸運点
- 各ヒット部位に+2ヒットポイント
- 耐久ロールが1段階容易になる
- 隠密ロールが1段階容易になる
- 意志ロールが1段階容易になる

すべてのパラゴンキャラクターは、軽傷または重傷からの回復力が2倍になりますが、致命傷はキャラクターの標準回復力で処理されます。

#### 技能

パラゴンヒーローキャラクターは、必要に応じて割り振るための追加の100ボーナス技能ポイントを得ます。

## ステップ8：情熱

情熱は、プレイヤーやゲームマスターがキャラクターの性質を深め、キャラクターの願望や偏見に基づいたプロットフックを導入し、プレイヤーが自分のキャラクターが誰であるか、何が彼を突き動かすのかなどを理解するのを助けるために使用されます。

情熱は、人物、組織、理想、さらには物体など、何に対しても向けられます。情熱は通常、次のような動詞で表現されます。望む、嫌う、破壊する、擁護する、恐れる、逃れる、誓ってやめる、憎む、愛する、忠誠を誓う、守る、拒否する、尊敬する、求める、転覆させる、苦しめる、支持する。

情熱は技能と同じ方法で測定されます。1から100（またはそれ以上）の範囲で評価されます。情熱は技能と同じように向上させることができ、また状況に応じて審判によって増減することもあります。

初期のキャラクターは最大3つの情熱から始めることができます。それぞれ2つの特性値に基づいており、最初の情熱に+40、2番目に+30、3番目に+20のボーナスが加算されます。プレイヤーは重要度の順序を決定します。これらは、彼の文化、職業、または個人的な気まぐれに結びつけることができます。例えば、ソード＆ソーサリーキャンペーンでは、「スティギア人を憎む」、「魔術師を不信」、「酒を望む」を選択するかもしれません。一方、ディストピアSFゲームでは、「大統領に忠誠を誓う」、「異星人を破壊する」、「法を守る」を選択するかもしれません。

#### 情熱表

| 情熱の対象 | 初期成功率 |
| :-- | :-- |
| 恋愛または家族関係の人物 | POW+CHA |
| 組織または集団の人々 | POW+INT |
| 種族または生物種 | キャラクターのPOW x2 |
| 場所 | キャラクターのPOW+INT |
| 物体または物質 | キャラクターのPOW x2 |
| 性格特性 | キャラクターのPOW+CHA |

### 情熱の使用

情熱とは、ゲームプレイ中に出来事に影響を与える能力を持つ、深く抱かれたコミットメントのことです。それらは独立した能力として使用できますが（技能と呼ぶのは不当ですが）、情熱がどのように機能し、『ミストラス・インペラティブ』のキャラクターやキャンペーンにどのように影響を与えるかをより詳細に説明するために、以下でさらに詳しく解説します。

要するに、情熱は次のように使用できます。

- 他の技能を増強するために、その感情の深さと行動を駆り立てる方法を反映します。この場合、その増強がテーマ的かつ劇的に重要であれば、情熱はその値の20%を、使用される技能に加算します。
- 選択、欲望、感情的な行動や反応を駆り立てる能力として。この方法で使用する場合、キャラクターが何かについてどれだけ強く考え、感じるかを判断するために、情熱に対して標準ロールが行われます。ロールが成功した場合、キャラクターはその情熱が指示する行動を取ります。ロールが失敗した場合、キャラクターはその情熱の駆り立てられる感覚に縛られずに自由に行動できます。
- 他の情熱（同じキャラクターが抱いている情熱でさえ）に対抗するために。これは通常、2つの情熱が対立する場合に使用されます。例えば、個人的な愛情が誓いや忠誠に反する行動を指示する場合があります。ここで、2つの情熱の間で対立ロールを行い、より成功した方がキャラクターの行動を決定します。
- コミットメント、信念、大義への忠誠心の深さの一般的な尺度として。情熱の値が高いほど、キャラクターのコミットメントは深まります。似たような情熱を持つキャラクターは、その評価を比較して、どちらがより深いコミットメントを示しているかを判断できます。
- ある種の心理操作や魔法による支配に抵抗するために。キャラクターがその情熱に反する行為を強制されている場合、彼らは対立ロールで「意志」の代わりに情熱を使用することができます。

新しい情熱は、状況が許せばゲームセッションのどの時点でも開発できます。誰かが強い憎悪を呼び起こすようなことをした場合、「Xを憎む」という情熱を即座に開発するかもしれません。領主や酋長が誓いを求める場合、その誓いが立てられたり宣誓されたりした瞬間に、「Xへの忠誠心」という情熱が即座に確立されます。

このようにプレイ中に開発された新しい情熱は、経験点ロールを必要とせず、その場で確立されます。もちろん、キャラクターは向上中に、経験点ロールを消費して、その基本値で情熱を確立することを選択することもできます。

### 深化と衰退

情熱は、増加の引き金となった出来事の強さに応じて、経験点ロールとは独立してゲーム中に増加する可能性があります。また、それらは衰退することも、実際に衰退します。強く抱いていた信念は、多くのことによって揺るがされ、その情熱が減少したり、より極端な状況では完全に逆転したりすることもあります。

例えば、「酋長を信頼する」という情熱を持つキャラクターが、もし酋長が裏切り行為をした場合、その情熱を減少させるか、同じパーセンテージで「酋長を不信」に転じるかもしれません。ゲームマスターは、情熱が減少するのか、逆転するのかを判断する必要があります。減少する場合、それは変化の原因の強さに応じて減少します。

深化と衰退の表は、情熱がどれだけ変化するかを示しています。

#### 深化と衰退表

| 変化の深さ | 情熱の修正量 |
| --- | --- |
| 弱い | 1d10 |
| 中程度 | 1d10+5 |
| 強い | 1d10+10 |

## ステップ9：装備

キャラクターがほぼ完成したら、いくつかの装備が必要です。これには武器、防具、または特定の職業で使用されるその他の重要な備品が含まれます。冒険中には、さらに多くの装備を手に入れる機会がありますが、各初期キャラクターは、以下のものをすぐに所持しています。

- キャラクターの職業と設定に合った一式の衣服。
- 小額の小遣い。苦労なくしばらく生活できるだけの額。
- 個人の家宝、記念品、または相対的に価値の低い装身具。
- キャラクターの職業に適した、持ち運び可能な取引道具や装備（該当する場合）。
- キャラクターの戦闘スタイルに合わせた個人用武器。
- 設定に適切であり、GMの承認を得たその他のアイテム。
