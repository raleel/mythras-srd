# 戦闘

戦闘はロールプレイングゲームの重要な特徴です。復讐に燃える宿敵同士の戦いから、戦時の軍隊の衝突に至るまで、暴力的な対立は常に問題を解決する手段でした。しかし、戦闘は死に至る残忍な戦いである必要はありません。一部の戦闘遭遇は、負傷することなく終わる剣術の決闘や、あるいは気楽な酒場の乱闘であることもあります。『ミトラス・インペラティブ』では、戦闘の核となる要素を説明するためにいくつかの用語を使用しています。これらは以下のセクションで説明されていますが、概要としては以下の通りです。
- **戦闘スタイル:** 文化や職業の一環として習得した武器のグループを操る技能。
- **戦闘ラウンド:** 5秒ごとの記録用時間区分。
- **戦闘アクション:** アクションポイントを消費して、キャラクターがラウンド中に行える行動。
- **武器サイズ:** 特定の武器を受け流すのがどれほど難しいか。
- **交戦:** 状況に応じて、戦闘員が直接の相手を攻撃したり、攻撃されたりできる状態。
- **特殊効果:** 戦闘員の一方が優位に立った場合に実行できる戦闘機動。

## 戦闘スタイル

戦闘スタイルは、その性質によって関連する複数の武器を「パッケージ」として表現するものであり、個々の武器を別々に学ぶ必要性を回避します。

戦闘技能の最も重要な側面は、キャラクターがその技能に含まれる各武器を、単独でも組み合わせても使用する方法を学ぶことです。これにより、必要に応じて武器を交換できるようになります。戦闘訓練は、特定の武器や最高の条件下での武器の使用に焦点を当てるだけでなく、極めて不利な状況に置かれた場合の対処法も網羅しています。このような異種混合訓練は、武器の破損や武装解除が常に起こりうる戦場の予測不可能な出来事に戦闘員を準備させる主要な部分です。

したがって、ハイパーボレアの野蛮人は槍、手斧、投げ槍、盾の使い道を知っていると期待される一方、シュメールの戦士は代わりにショートソード、メイス、盾、弓の訓練を受けているかもしれません。単一のスタイルに含めるべき武器の数を決定することは、最終的にはゲームマスターとゲームの舞台設定によって決まる選択です。

自身のスタイルでカバーされていない武器を使用しようとするキャラクターは、技能に1つ以上の難易度ペナルティを受けます。

戦闘スタイルは、単に武器技能のグループを学ぶこと以外のもう一つの機能を持っています。各スタイルは特定の文化や職業によって教えられるため、そのグループが通常戦う環境や軍事戦術に本質的に最適化されています。例えば、馬の遊牧民は馬上での戦い方を学び、歩兵は密集陣形での戦い方を訓練します。したがって、追加のボーナスとして、一部の戦闘スタイルには、特定の状況下で利点を与える1つ以上の特性が割り当てられる場合があります。特性は、その特定のスタイルの武器と状況でのみ使用されることを意図しています。明らかに、無数の特性があり、その多くは特定の時代やキャンペーン世界に特有のものです。

##### 戦闘スタイルの特性表

| 特性 | 説明 |
| :-- | :-- |
| 獣背槍騎兵 | この戦闘スタイルで騎乗突撃を実行しても、命中への難易度ペナルティは発生しない。 |
| 盲目戦闘 | 劣悪な照明や一時的な目くらましによるペナルティを無視できる。 |
| 命知らず | 近接戦闘で、倒れ伏すことなく回避技能を使って攻撃をかわすことができる。 |
| 防御志向 | そのラウンドで攻撃行動を取らない限り、受け流す際の武器サイズが1段階大きくなる。 |
| 陣形戦闘 | 側面を突かれていない3人以上の戦士のグループが密集陣形を組むことを許可し、開けた場所にいるか乱れた敵を不利にし、交戦する敵のアクションポイントを1点減少させる。 |
| ノックアウトブロー | 奇襲で攻撃する場合、スタン位置はターンではなく分数として扱う。 |
| 騎乗戦闘 | 騎乗技能によって戦闘ロールに課せられる技能制限を無視できる。 |
| 遠距離射手 | 遠隔武器を使用する際、ランダムな命中部位ロールを隣接する身体部位にずらす。 |
| 散兵戦術 | 歩きながらまたは走りながら遠隔攻撃を繰り出すことができる。 |
| 武器投擲 | スタイル内のあらゆる近接武器を技能にペナルティなしで投擲できるが、この方法で使用した場合、武器のダメージロールは半分になる。 |
| 素手熟練 | 素手での防御や受け流しを「中」サイズとして扱い、武装した相手から身を守りやすくする。 |

## 戦闘の進行

その内臓的な性質を強調するため、戦闘は一撃ごとに追跡されます。記録を容易にするため、戦闘はそれぞれ5秒の戦闘ラウンドに分割されます。この期間中、戦闘員は近接攻撃を行ったり、それらから防御したり、戦闘の出し入れをしたり、遠隔武器を発射または投擲したりすることができます。現実生活と同様に、ほとんどの戦闘エンゲージメントは、一旦開始されると、数分ではなく数秒で終了します。ゲームの用語では、これはいくつかの戦闘ラウンドを意味します。ただし、参加者がイニシアチブをリセットしたり、戦術的状況を再評価したり、あるいは再交戦する前に一時的な休息を取ったりするために撤退しない限りです。互角の敵同士の戦い、または連続する敵との戦いだけが長く続く傾向があります。

## 銃器

『ミトラス・インペラティブ』のゲームで銃器を使用する場合、銃撃戦の致命性を考慮すべき点がいくつかあります。第一に、人間の反射神経よりも速く移動するレーザービームや銃弾を避けることはほぼ不可能です。提示されている戦闘ルールを使用すると、回避を使ってその場から飛び退こうとしない限り、いかなる命中も即座に犠牲者を1つ以上の特殊効果にさらすことになります。また、アクションポイントの経済性も変化します。受動的に自分自身を守るためにポイントを使用できない場合、ほとんどのキャラクターは敵の無能さに賭けて、全てのアクションを撃ちまくることに費やすことになるでしょう。このパラダイムシフトに対処するため、ゲームマスターとプレイヤーの両方が銃撃戦に対する見方やアプローチを変える必要があるかもしれません。留意すべき点は以下の通りです。

- 銃撃戦は危険である！
- 全体として、人々は他者を撃つことよりも、撃たれないことを優先する。
- 最も一般的な防御方法は、カバーを取ることである。
- 抑制射撃は、敵の反撃を防ぐ最良の方法である。
- 銃器は、構えたり狙いを定めたりする時間を取らない限り、驚くほど不正確である。
- 撃たれたら、悪いことが起こる。

銃撃から身を守ることは、主にボディアーマー、カバーの使用、そして相手に反撃させないようにすることになります。結果として、ほとんどの銃撃戦は非常に戦術的になりがちです。多くの抑制射撃、側面攻撃、深いカバーからの敵の掃討などが行われ、キャラクターは相手が射撃を止めるまで、ラウンド全体を身動きが取れずに過ごすこともあるかもしれません。これを一部のプレイヤーは極めて退屈だと感じるかもしれません。

銃撃戦全体で負傷者が一人も出ないことも十分に考えられます。片方が弾切れで降伏したり、戦場から完全に逃げ出したりするだけです。結局のところ、フルオートの武器からの一斉射撃で蜂の巣にされる危険を冒したいと思う人はほとんどいないでしょう。

これらの戦闘ルール全体を通じて、特定の行動タイプ、特殊効果、および状況は銃器を使った戦闘にのみ適用され、それらは適切にマークされています。

## 戦闘ラウンドとイニシアチブ

戦闘ラウンドではいくつかの重要な用語を使用します。

- **イニシアチブ:** 戦闘ラウンドのターン中に、誰がいつ行動するかを示す順序。ゲームマスターはイニシアチブの値を最も高いものから低いものへとカウントダウンし、各参加者は自分の番が来たときにアクションを行う機会を得ます。
- **ターン:** 各5秒の戦闘ラウンド中、各参加者は、所持するアクションポイントの数とそれらの使用方法に応じて、1回以上のアクションを行う機会があります。自分のターンでは、参加者は1つの戦闘アクションを実行します。これには宣言、可能な移動、必要なダイスロール（リアクションを含む）、およびアクションの解決が含まれます。参加者にアクションポイントが残っている場合、プレイヤーはイニシアチブ順にさらにターンを取ることができます。アクションポイントが残っていないか、残りのポイントがリアクションのために保留されている場合、次の戦闘ラウンドに進みます。

各用語は、以下でさらに詳しく説明されています。

#### イニシアチブ

イニシアチブは、戦闘ラウンドにおける参加者の行動順序を決定します。戦闘開始時にロールされ、各キャラクターがいつ行動するかを決定します。特定の戦闘アクションや特殊効果など、状況を変える出来事が起こらない限り、イニシアチブは再ロールされるまで有効です。

イニシアチブは、各参加者が1d10をロールし、イニシアチブ修正値を加算することで計算されます。最も高い結果を得た者が最初に、次に高い者が2番目にというように行動します。2人以上の参加者が同点になった場合、DEXが高い者が先に、それでも同点の場合は、それぞれダイスをロールし、出目の高い方が先に行動します。

#### 戦闘中のアクション

イニシアチブが決定されると、参加者は各戦闘ラウンド中にいくつかの戦闘アクションを実行する可能性があります。全てのキャラクターはラウンドごとに2アクションポイントを獲得しますが、行動できるタイミングはアクションが先行型（Proactive）か反応型（Reactive）かによって制限されます。

先行型アクションは、キャラクター自身のターン、つまり自身のイニシアチブ中のみ試みることができます。先行型アクションとは、武器で攻撃するなど、キャラクターが主導者となる行動です。

反応型アクションは、自分に対して行われた行為に対抗または抵抗するためにキャラクターが取る行動です。各脅威に対して許可される反応の試みは1回のみであり、攻撃を受け流そうとするのが一例です。

未使用のアクションポイントは、次のラウンドに持ち越されません。

## 戦闘アクション

戦闘アクションは、戦闘中に行うことができる行動です。武器を抜く、呪文を唱える、攻撃から身をかわす、これらはすべて戦闘アクションの例です。ほとんどは戦闘そのものの要素に関係していますが、一部は呪文や武器の領域外の活動に関連しています。

前述の通り、キャラクターがラウンド中に行動できる回数は、利用可能なアクションポイントによって制限されます。これらは各戦闘ラウンドを通じて、様々な戦闘アクションを実行するために消費されます。戦闘員のアクションポイントが尽きると、そのラウンドの残りの時間は行動できず、次のラウンドの開始時にポイントがリセットされるのを待たなければなりません。

特に指定がない限り、いかなる戦闘アクション（「フリー」アクションを除く）も1アクションポイントを消費します。したがって、戦闘員はアクションポイントをどのように、いつ使用するかを慎重に検討する必要があります。彼らは、戦闘の進化する状況に動的に反応し、望むだけ積極的にまたは防御的に行動することができます。

### 先行型アクション

以下は、キャラクターが自分のターンにアクションポイントを消費して試みることができる行動です。呪文の詠唱やリロードなど、完了までに数ターンかかるアクションもあることに注意してください。各ターンはそれぞれ独自のアクションポイントを消費します。

#### 攻撃

キャラクターは近接武器で攻撃を試みるか、遠隔武器を使用できます。

#### 体勢を固める（Brace）

キャラクターは、来るべき攻撃の方向に向かってしっかりとした姿勢を取り、身を固めます。ノックバックや跳躍攻撃に抵抗する目的で、キャラクターのSIZは150%大きいものとして扱われます。バッシュ特殊効果に対しては、SIZは2倍になります。

#### 魔法を唱える/能力を使う

キャラクターは呪文を唱えたり、特定の能力を発動させたり、その他を試みることができます。複雑な魔法は詠唱完了までにいくつかのアクションを要する場合があります。完了後、魔法は術者の次のターンまでいつでも解放できます。その時点から後で効果を発揮させるために保持することも可能ですが、後での解放に備えてそれを維持するには、「魔法/能力を保持する」アクション（下記参照）が必要です。

#### 距離変更

キャラクターは、相手と近接戦闘を行っていない、または遠隔戦闘状況で敵の射撃によって行動を制限されていない限り、相手に接近したり、相手から後退したりすることができます。

#### 突撃（Charge）

突撃アクションは、キャラクターが走るか全力疾走の速度で交戦範囲に入り、突撃の勢いを利用してより強力な近接攻撃を行うことを可能にします。これらのルールは、以下の[突撃](0005_Combat.md?id=charging)セクションで詳しく説明されています。

#### 遅延

キャラクターは、後で中断や受け流しなどの反応型アクションを実行するために、1つ以上のアクションを温存します。遅延に要するアクションポイントのコストは、最終的に実行される行為によってカバーされます。遅延されたアクションがキャラクターの次のターンまでに実行されない場合、キャラクターは躊躇した（下記参照）と見なされ、アクションポイントは失われます。

#### 躊躇（Dither）

他のアクションが選択されない場合のデフォルトオプションで、キャラクターは単に自分のターンを無駄にし、何の役にも立たないことをします。

#### 魔法/能力を保持する

詠唱が完了した後、キャラクターは呪文や能力を一時的に保留し、最適な解放の瞬間を待ちます。このアクションをその後のターンで取り続ける限り魔法は保持できますが、呪文に関連する場合、敵によるカウンター・スペル反応を自由に許します。保持された呪文を実際に唱えるための技能ロールは、解放されるまで行われません。

#### 騎乗

キャラクターは騎乗獣または乗り物に乗降することができます。特に大きな乗り物は完了までに数ターンを要する場合があります。

#### 移動（Move）

相手と交戦していない限り、キャラクターはゲームマスターが状況に適していると判断する任意の距離を移動できます。

#### 巧みに立ち回る（Outmaneuver）

キャラクターは複数の相手に対して回避技能のグループ対抗ロールを行うことができます。ロールに失敗した者は、その戦闘ラウンド中、彼を攻撃できません。以下の[巧みな立ち回り](0005_Combat.md?id=outmaneuvering)のセクションを参照してください。

#### 武器を構える

キャラクターは武器やその他の物体を取り出す、抜く、しまう、引き抜く、またはリロードすることができます。近くに落とした物体を拾い上げるには2つのアクションが必要です。1つは移動して物体に手を伸ばすため、もう1つは構えた姿勢に戻るためです。一部の投擲武器はリロードに数アクションを要します。

#### リロード (銃器)

武器を構えるに準じます。リロード時間は武器の種類によって大きく異なります。単発の黒色火薬銃は、弾倉の交換や動力源の変更よりもはるかにリロードが遅くなります。

#### 足場を回復する

相手と交戦していない場合、キャラクターは転倒したり倒されたりした状態から自動的に足場を回復できます。交戦中の場合、キャラクターは立ち上がる前に相手との筋力または運動技能の対抗ロールに勝たなければなりません。この対抗戦は解決にアクションポイントを消費しません。

#### 隙をうかがう (銃器)

隙をうかがうことで、現在カバーに隠れている人物は素早く様子をうかがい、戦術的状況を判断することができます。この行動は、実質的に安全であり、観察者がカバーに戻る前に1回の知覚ロールを行うことを許可します。しかし、知覚テストがファンブルした場合、彼らは愚かにも頭を出しすぎたり、敵が射撃するのに十分な時間、視界に留まってしまいます。

#### 武器を固定する (銃器)

キャラクターが確固たる射撃姿勢を確立し、自動火器を安定させることを可能にします。これには、キャラクターが少なくとも部分的に露出しており、カバーによって完全に保護されていないことが必要です。主な利点は、バーストまたはフルオート射撃の反動ペナルティを1段階減少させることです。この効果は「体勢を固める」と同様に、キャラクターが移動するまでその恩恵が続きます。

#### もがく

特定の種類の攻撃や特殊効果の犠牲者となった場合、キャラクターはその状況から抜け出すことを試みることができます。例えば、敵の拘束から解放されるなど。

#### 射撃モード切り替え (銃器)

射撃モード切り替えは、自動火器または二目的火器のユーザーが、他の射撃モードを利用するために武器のセレクターを変更することを可能にします。これにより、アサルトライフルをセミオートからバーストに切り替えたり、異なる種類の弾薬を選択したり、統合されたグレネードランチャーのような補助システムを起動したりできます。現代の武器では、モード切り替えにキャラクターのターンにアクションポイントが必要ですが、十分に高度なハイテク銃器では、電子インプラントやユーザーの心をサイキックで読み取ることで、フリーアクションとして許可される場合があります。

#### カバーを取る

カバーを取るは先行型アクションであり、射手が身近にある利用可能なカバーの陰に身を隠し、飛来する射撃に対するある程度の保護を得ることを可能にします。回避とは異なり、使用者を倒れ伏す状態にはしませんが、何らかの形のカバーが利用可能であることに依存します。例えば、廊下の角に身を隠したり、バーのテーブルの陰にしゃがみ込んだりするような場合です。状況によっては、利用可能なカバーがキャラクターを完全に保護するのに十分でない場合があります。カバーの種類もその保護特性を決定します。例えば、厚い鋼鉄のドアは銃弾に対して貫通不可能かもしれませんが、薄い木製の壁は4アーマーポイントしか提供しないかもしれません。特定の材料の保護特性に関する一般的なガイドラインについては、[無生物](0004_Game_System.md?id=inanimate-objects)のセクションの表を参照してください。

### 反応型アクション

このリストは、差し迫った脅威に対する反応として、戦闘ラウンド中のいつでも使用できる反応を指定しています。前のリストと同様に、反応を実行するには1アクションポイントを消費します。

#### カウンター・スペル

キャラクターは飛来する呪文を解除または打ち消すことができます。これは、打ち消し魔法の詠唱時間が1ターンであると仮定しており、そうでない場合は事前に準備され、魔法保持アクションを使用して一時的に保留されなければなりません。この方法で魔法を首尾よく阻止した場合、複数の標的や効果範囲を持つ呪文であっても、呪文全体が無効化されるとみなされます。

#### 回避

キャラクターは回避技能を使い、飛来するミサイルや突撃攻撃などの脅威から身をかわすことを試みます。能力や特別な状況によって軽減されない限り、これによりキャラクターはうつ伏せになります。したがって、キャラクターの次のターンは、通常、足場を回復するアクションで再び立ち上がるために費やされます。

#### 中断

遅延中のキャラクターのみ。この反応型アクションは、相手のターンを任意の時点で停止させ、遅延されたターンアクションを取ることを可能にします。戦術的状況に変化がないと仮定した場合、相手はキャラクターのターンが完了した後もターンを継続します。元の宣言を達成できない場合、相手のアクションポイントは無駄になります。中断は、遅延中のキャラクターの近くを通り、武器の届く範囲内にいる者に対して、機会攻撃を行うためにも使用できます。

#### 受け流し

キャラクターは、受け流し、身をかわす、しゃがむ、横にステップするなどのフットワークを組み合わせて、飛来する攻撃から身を守ることを試みます。

### フリーアクション

フリーアクション（以下にリスト）は、戦闘ラウンド中のいつでも実行でき、アクションポイントを消費しません。

#### 状況判断

交戦中でない場合、キャラクターはアクションポイントなしで知覚ロールを行うことができます。成功すると、戦術的状況における関連する変化（突撃を開始する敵の発見など）が明らかになります。走りながら判断すると、困難な知覚ロールとなり、全力疾走中は手ごわい知覚ロールとなります。

#### アイテムを落とす

アイテムを落とすのはフリーアクションです。

#### 合図

交戦中でない場合、1人以上の参加者に身振りや合図を送る（その合図を知覚できる場合）のはフリーアクションです。

#### 話す

キャラクターは戦闘中いつでも話すことができますが、話す内容は5秒以内に発せられる短いフレーズに限定すべきです。例えば、「後ろに気をつけろ！」とか「くそっ！」などです。

#### 幸運ポイントを使う

幸運ポイントを使用する（例えば、特定のロールをやり直すなど）のはフリーアクションです。

#### 部位防御

キャラクターは、特定の命中部位が攻撃されるのを防ぐために、自身の武器の一つをその領域を受動的にブロックするために捧げます。その部位に着弾した攻撃は、そのサイズの受け流し武器の通常の処理に従って、自動的にダメージが格下げされます。このカバーは、その捧げられた武器が攻撃または積極的に受け流しに使用されるまで続きます。カバーする命中部位の設定または変更は、相手がキャラクターを攻撃するロールを行う前に行われなければなりません。その設計上、盾は複数の領域をカバーすることができます。

詳細については、以下の[受動的防御](0005_Combat.md?id=passive-blocking)を参照してください。

## 移動

以下のルールが戦闘における交戦を規定します。

- 移動は「移動」または「突撃」アクションによって行われます。
- 「移動」アクションは、キャラクターが基本移動速度に等しい距離を移動することを許可します。
- 移動は、交戦していないキャラクターのみが行うことができます（巧みな立ち回りなどの特殊な場合を除く）。
- 特定の場合（突撃など）を除き、キャラクターが意図する相手の交戦範囲に入ると、すべての重要な移動は停止します。
- 接触を伴う突撃でない限り、相手の交戦ゾーンに移動すると、お互いが交戦状態になります。
- 一度交戦状態になると、キャラクターは、まず近接戦闘から離脱しない限り、相手から離れて移動することはできません。
- 「遅延」戦闘アクションを使用している交戦していない敵を通り過ぎようとすると、その敵は（特定の例外として）通り過ぎるキャラクターを攻撃したり、キャラクターの進行を妨げたりすることができ、次のターンまたはラウンドで近接戦闘に突入します。

### 状況的移動速度

これらの移動速度は状況的であり、常に選択できるわけではありません。多くは他の場所で詳しく説明されていますが、便宜上ここに含められています。これらの状況的速度のほとんどは、設定された制限の下で通常の移動速度に追加して使用できます。そのような情報は以下の特定の記述で詳しく説明されています。

#### 登攀

登攀可能な表面や物体の隣に移動したり、その隣でターンを開始したりするキャラクターは、基本移動速度と同じ速度で登ることができます。キャラクターは歩行よりも速く移動してはならず、まだ移動距離が残っている必要があり、登った距離はすでに使用された移動距離によって減少します。着用している防具は、着用しているアーマーポイントの合計に基づいて登攀者を妨げます。

##### 防具による移動ペナルティ表

| 総AP着用数 | 荷重カテゴリ | 移動 |
| :-: | :-: | :-: |
| 0-7 | なし | 0 |
| 8-15 | 軽 | -1 |
| 16-23 | 中 | -2 |
| 24-31 | 重 | -3 |
| 32-39 | 手ごわい | -4 |
| 40+ | 超人的 | -5 |

#### 這う

移動ルールは『ミトラス』ルールブックで完全にカバーされています。

#### 跳躍

跳躍可能な障害物の隣に移動したり、その隣でターンを開始したりするキャラクターは、その障害物を飛び越えることを試みることができます。キャラクターは跳躍する時点までどのような速度で移動していても構いません。

運動技能ロールに成功すると、自分の身長の2倍まで水平に跳躍するか、自分の身長の半分まで垂直に跳躍できます（キャラクターが少なくとも5メートルの助走が可能である場合）。立ち位置から跳躍する場合、これらの距離は半分になります。

跳躍距離は、運搬している「アイテム」の合計の5分の1（メートル）だけ減少します。着用している防具は跳躍者を妨げます。軽装甲で覆われた各部位は上記の計算で1「アイテム」として数えられ、重装甲は2「アイテム」として数えられます。

## 戦闘の仕組み

『ミトラス・インペラティブ』での戦闘は、各攻撃または受け流しが一回の突き、振り、呪文詠唱、または武器の一発を表す形で解決されます。いかなる攻撃アクションも、反応によって抵抗される可能性があります。したがって、キャラクターが相手を攻撃できたとしても、敵はそれが着弾する前に攻撃を受け流せるかどうかのロールを行うことができます。この同じ哲学は、飛来するミサイルがカバーに飛び込む目標に当たる場合でも、巨大なタコの鉄の掴みから逃れようとする場合でも適用されます。

### 攻撃と受け流し

近接戦闘は、以下のステップバイステップ方式で処理されます。

1. 攻撃者は自分のターンにアクションポイントを消費し、戦闘技能に対してロールを行い、結果を記録します。
2. 望む場合、防御者はアクションポイントを消費して受け流しを行い、自身の戦闘技能に対してロールを行い、結果を記録します。
3. 結果の成功レベルが差分ロールに従って比較されます。
4. いかなる差分も、優れたロールをした成功した戦闘員に1つ以上の特殊効果を与えます。
5. 攻撃者が成功またはクリティカルを達成した場合、武器ダメージをロールし、ダメージ修正値（もしあれば）を適用できます。該当する場合、攻撃の命中部位が決定されます。
6. 防御者が成功またはクリティカルを達成した場合、関係する武器の相対的なサイズに応じて、与えられたダメージを軽減します。
7. 残りのダメージを、自然または着用保護のアーマーポイントで軽減します。

交換によって生成される特殊効果は、ダメージが与えられるかどうかとは無関係であることに注意してください。防御者がより高い成功レベルを達成したにもかかわらず、負傷を負う可能性は十分にあります。

### 武器サイズ

『ミトラス・インペラティブ』のすべての武器は、与えるダメージから両手で扱う必要があるかまで、いくつかの異なる属性を持っています。サイズは、武器の質量、てこの原理、安定性の組み合わせであり、武器がダメージを与えたり、受け流したりする能力を決定するために使用されます。サイズカテゴリは、小、中、大、巨大、特大です。クリーチャーの素手攻撃は、その身体的特徴に応じて同様のカテゴリに割り当てられます。

### ダメージ軽減

防御者が受け流しに成功した場合、使用された武器の比較サイズに応じて、攻撃者のダメージを軽減することができます（もしあれば）。

- 同等またはそれ以上のサイズの武器や盾で攻撃を受け流した場合、すべてのダメージが防御されます。
- 1サイズ小さい武器や盾で受け流した場合、ダメージは半分だけ防御されます。
- 2サイズ以上小さい武器や盾で受け流した場合、ダメージは全く防御されません。

_例として、グレートアックス（巨大）をカイトシールド（これも巨大）で受け流すと全ダメージを防ぎます。ロングソード（大）で受け流すとダメージは半分になり、ショートソード（中）で受け流すとダメージは全く防げません。_

### 防具

防御者の受け流しを貫通したダメージは、そのキャラクターが持つ自然な保護（鱗や甲羅など）または着用している防具のアーマーポイントによってさらに軽減されます。防御者が両方を持っている場合、両方が積み重なってダメージを軽減します。

その質量と移動制限のため、キャラクターが着用している防具の最も高いアーマーポイント値は、イニシアチブロールに対するペナルティとして機能します。

### 外れた攻撃の受け流し

攻撃者が最初の攻撃ロールをミスした場合、防御者はアクションポイントを消費して受け流しを行う選択肢があります。いずれにせよ外れる攻撃を受け流すことは不誠実に思えるかもしれませんが、熟練した防御者はこれを利用して1つ以上の特殊効果を獲得し、敵を弱体化または無力化し、次のターンを他の行動のために温存することができます。

### 受け流しができない、または受け流しをしない場合

防御者が残りのアクションポイントがないために受け流しができない状況、または攻撃に耐えられる自信があるため受け流しをしないことを選択した場合、自動的に失敗をロールしたと見なされます。これにより、成功した攻撃者に1つ以上の特殊効果が付与される結果となります。

### 失敗ロールとファンブル

両方の戦闘員がロールに失敗した場合、または防御者が外れた攻撃の利点を利用しないと決定した場合、攻撃と受け流しのシーケンスは終了し、次に高いイニシアチブを持つ参加者に戦闘が続行されます。

## 特殊効果

戦闘は単に相手を傷つけたり殺したりするだけではありません。格闘技は、敵を打ち破る多くの方法を教えてくれます。それは、必ずしも危害を加えることなく、敵を無力にしたり、降伏せざるを得ない状況に追い込んだりすることもあります。特殊効果はこれらの技術を表現し、ゲームプレイでの発生方法を制御します。

相手がそれぞれの戦闘技能で差分ロールを行った場合、結果として生じる成功レベルの差は、特殊効果が発生する機会を示します。これは、一方の戦闘員が相手を不利な状況に追い込み、巧妙な策略や戦術でそれを悪用できることを反映しています。

受け取られる特殊効果の数は、差分成功レベル表に示されているように、成功レベル間の差に依存します。

交換中に特殊効果が獲得された場合、ダメージと命中部位（もしあれば）がロールされる前に選択されなければなりません。特殊効果の適用に加えて対象がひどく負傷した場合、耐久力ロールは効果の適用後に解決されます。

特殊効果は、多様な状況と技術をカバーしています。一部は攻撃専用に設計されており、他は防御を意図しており、いくつかはどちらの状況にも適応可能です。特定の特殊効果は、特定の武器タイプや特定のダイスロールに限定されることもあり、例えば、技能チェックでクリティカルまたはファンブルの結果を要求する場合があります。

2つ以上の特殊効果を獲得した場合、戦闘員は自由にそれらを組み合わせて選択できます。ただし、それぞれの前提条件が満たされている必要があります。一部の効果は累積可能です。例えば、クリティカルサクセスをロールし、2つの特殊効果を獲得した攻撃者は、2つの別々の攻撃マニューバを選択するのではなく、ダメージ最大化を2回選択することができます。

### 特殊効果の詳細

#### 偶発的負傷

防御者は相手の攻撃をそらすか、ねじることで、攻撃者がファンブルし、自身を傷つけます。攻撃者は、使用した武器でランダムな命中部位に自身に対してダメージをロールしなければなりません。素手の場合、攻撃者は内部の何かを裂くか折るため、ダメージロールは防具を無視します。

#### 立ち上がる

防御者が一瞬の隙を利用して立ち上がることができる。

#### バッシュ

攻撃者は意図的に相手のバランスを崩すように打ちます。防御者がどれだけ後ろや横によろめくかは、使用している武器によって異なります。盾はロールされたダメージ2点ごとに相手を1メートル後退させ（防具や受け流しによる減算前）、鈍器は3点ごとに1メートル後退させます。バッシュは攻撃者のSIZの2倍までのクリーチャーにのみ有効です。対象が障害物に後退させられた場合、アクロバット技能ロール、または運動技能をハード難易度でロールし、転倒やよろめきを避ける必要があります。銃器の場合、バッシュは純粋な力で相手を後退させるというよりも、バランスを崩させることに依存するため、投射弾薬が手足に当たった場合に標的がよろめくのは完全に合理的です。エネルギー兵器は、ビームの物理的特性が何らかの表面爆発を引き起こさない限り、そのような効果をもたらす可能性は低いです。

#### 出血

攻撃が主要な血管を切り開きます。攻撃がアーマーポイントを克服し、標的に負傷を与えた場合、防御者は元の攻撃ロールに対する耐久力対抗ロールを行わなければなりません。防御者が失敗した場合、多量に出血し始めます。各戦闘ラウンドの開始時、対象は疲労レベルを1段階失い、最終的に倒れて死亡する可能性があります。出血した傷は応急手当技能ロールに成功することで止めることができますが、対象は傷が再開することなく、激しいまたは暴力的な行動を行うことはできません。ゲームシステムセクションの「失血」を参照してください。銃器の場合、ほとんどの銃器は非常に狭い傷跡しか作らないため、切り裂きや突き刺し武器に比べて主要な血管を傷つけたり切断したりすることは実際に難しいです。このため、出血はクリティカルサクセスのみであるべきです。ただし、銃が手裏剣、かみそりのフレシェット、平らな力場、またはそのようなエキゾチックな弾薬を発射する奇妙な兵器である場合は別です。

#### 敵を盲目にする

クリティカル時に、防御者は砂を投げたり、盾に日光を反射させたりするなどの戦術で、相手の視覚を一時的に妨害し、相手を短時間盲目にします。攻撃者は、防御者の元の受け流しロールに対して、自身の回避技能（盾を使用している場合は武器技能）の対抗ロールを行わなければなりません。攻撃者が失敗した場合、目くらましの方法に応じて、次の1d3ターンにわたり、困難または手ごわいという状況修正を受けます。

#### 防具迂回

クリティカル時に、攻撃者は防御者の自然または着用防具の隙を見つけます。防御者が自然な保護の上に防具を着用している場合、攻撃者はどちらを迂回するかを決定しなければなりません。この効果は両方を迂回するために累積できます。この効果の目的上、魔法によって得られる物理的な保護は着用防具と見なされます。防具迂回はどのような種類の攻撃でも使用できますが、弾薬の種類によってはこの効果を利用できない（散弾銃のペレットはその拡散のため）か、その基本性能に組み込まれており、他の獲得した特殊効果に加えてボーナス特殊効果として付与される場合があります（テレポート弾など）。ゲームマスターが武器をどのようにモデル化したいかに大きく依存します。

#### 部位選択

近接戦闘武器を使用する場合、攻撃者は通常届く範囲内であれば、攻撃が着弾する部位を自由に選択できます。銃器を含む遠隔武器を使用する場合、「部位選択」はクリティカルサクセスのみであり、標的が近距離にいて、静止しているか攻撃者に気づいていない場合に限ります。銃器の場合、望遠照準器はこの範囲を拡張する可能性があります。

#### 受け流し迂回

クリティカル時に、攻撃者は通常成功する受け流しを完全に迂回することができます。

#### 武器損傷

攻撃または受け流しの一環として、相手の武器を損傷することを許可します。攻撃の場合、キャラクターは防御者の受け流し武器を specifically に狙い、その武器に自身のダメージロールを適用します。狙われた武器は、ダメージに抵抗するために独自のアーマーポイントを使用します。ヒットポイントがゼロに減少した場合、武器は破壊されます。銃器の場合、遠距離で手持ちの武器を狙うことは特定の身体部位を狙うよりもさらに困難であるため、これは通常、クリティカルサクセスのみに限定されるべきです。しかし、いくつかの例外があります。まず、盾は、その性質上、はるかに命中しやすいはずです。2つ目は、その影響範囲内の手持ちの物体を狙うことができる円錐形のビームを発射するエキゾチックな武器に関するものです。例えば、金属物体を加熱する誘導銃などです。

#### 武装解除

キャラクターは相手の武器を叩き落とすか、引き抜くか、ねじって相手の手から奪い取ります。相手は、キャラクターの元のロールに対する自身の戦闘スタイルの対抗ロールを行わなければなりません。武装解除された者が敗北した場合、武器は武装解除した者のダメージ修正値（メートル単位）に等しい距離だけ投げ飛ばされます。ダメージ修正値がない場合、武器は武装解除された者の足元に落ちます。武器の相対的なサイズがロールに影響します。武装解除するキャラクターの武器が1サイズ大きくなるごとに、相手のロールの難易度は1段階上がります。逆に、武装解除するキャラクターの武器が1サイズ小さくなるごとに、難易度は1段階下がります。武装解除は、攻撃者のSTRの2倍までのクリーチャーにのみ有効です。

#### 敵を倒す

攻撃、銃器のみ。攻撃が標的に少なくとも軽傷を与えたと仮定し、標的は攻撃者の命中ロールに対する耐久力対抗ロールを行わなければなりません。失敗は、標的がショックと痛みにより行動不能となり、戦闘を継続できなくなることを示します。行動不能からの回復は、応急手当チェックの成功、またはキャンペーンに存在するなら何らかの技術的または麻薬的ブースターの使用によって行うことができます。そうでなければ、一時的な行動不能は標的の治癒率で割った1時間に等しい期間続きます。

#### 身をかがめる

攻撃、銃器のみ。この特殊効果は、射手が「カバーを取る」アクションのために次のターンを待つ必要なく、即座にカバーに身を隠すことを可能にします。この効果を使用するには、キャラクターはすでに何らかのカバーの隣に立っているかしゃがんでいる必要があります。

#### 受け流し強化

クリティカル時に、防御者は武器のサイズに関係なく、攻撃の全威力をそらすことに成功します。

#### 絡みつき

鞭や網のような絡みつき武器を振るうキャラクターが、命中した部位を不動にすることを可能にします。絡みついた腕は持っているものを使えなくなり、絡まった脚は標的の移動を妨げ、頭、胸、腹が絡みついた場合は、全ての技能ロールが1段階難しくなります。続くターンに、振るい手はアクションポイントを消費して、自動的な敵転倒の試みを行うことができます。絡みつかれた犠牲者は、自分のターンに、筋力を使って引き剥がす対抗ロール、または特殊効果を獲得して「武器損傷」「武装解除」「抜け出す」を選択することによって、自由になろうと試みることができます。

#### 失敗強制

相手がファンブルした際に使用します。このキャラクターは、「失敗強制」を、対抗ロールを必要とする他の特殊効果と組み合わせて使用できます。「失敗強制」は相手が抵抗ロールに自動的に失敗することを強制します。これにより、自動的に武装解除されたり、転倒させられたりするなどとなります。

#### 掴み

相手が攻撃者の素手戦闘のリーチ内にある場合、空いた手（または掴むことのできる同様の四肢）を使って相手を掴み、戦闘から離脱するのを妨げることができます。相手は自分のターンに、筋力または素手戦闘のいずれかを掴んでいる者が選んだ技能に対する対抗ロールを試みることで、自由になろうと試みることができます。

#### 貫通

武器ダメージを2回ロールし、攻撃者は2つの結果のうちどちらを攻撃に使用するかを選択します。武器が近接武器であると仮定した場合、防具が貫通されて負傷が発生した場合、攻撃者は次のターンに武器を傷口に残すか、引き抜くかを選択できます。武器を傷口に残すと、犠牲者の今後の技能試みに難易度グレードが課せられます。ペナルティの深刻さは、クリーチャーとそれを貫通している武器のサイズの両方に依存し、以下の「貫通効果表」にリストされています。簡略化のため、同じサイズの武器によるさらなる貫通は追加のペナルティを課しません。近接戦闘中に貫通した武器を引き抜くには、「武器を構える」戦闘アクションを使用する必要があります。振るい手は、抵抗する相手がいた場合は非対抗の筋力ロールに成功するか（または対抗ロールで筋力に勝つか）なければなりません。成功すると武器が引き抜かれ、その武器の通常のダメージロールの半分に等しい（ただしダメージ修正値なし）さらなる負傷が同じ部位に与えられます。失敗は、武器が傷口に詰まったままで、さらなる効果はないことを意味しますが、振るい手は次のターンに再度試みることができます。特に逆棘のある武器（モリなど）は通常のダメージを与えます。防具は引き抜きダメージを軽減しません。貫通したままである間、攻撃者はその貫通武器を受け流しに使用することはできません。

#### 射撃手

射撃手が命中した部位を1段階移動させ、すぐに隣接する身体部位に変更することを許可します。生理学的な影響は、何が再標的化できるかに影響を与えるため、常識が適用されるべきです。したがって、この特殊効果を人型生物に使用した場合、足に命中した攻撃者が、代わりに腹部に移動させることが許可されます。逆に、グリフィンの胸を撃った場合、前脚、翼、または頭を選択することが許可されます。

#### ダメージ最大化

クリティカル時に、キャラクターは武器のダメージダイスの1つをその最大値に置き換えることができます。例えば、通常1d6ダメージを与えるハンドアックスは6と扱われ、2d6ダメージを与えるグレートクラブは1d6+6ダメージを与えることになります。この特殊効果は累積可能です。自然武器にも使用できますが、ダメージ最大化は攻撃者のダメージ修正値には影響せず、通常通りロールする必要があります。

#### 過貫通

クリティカル、銃器のみ。直線上に配置された敵や密集した集団を射撃する場合、この特殊効果は、最初の標的のボディアーマーを貫通した場合、射撃が最初の犠牲者を完全に通過してその背後にいる2番目の標的を攻撃することを可能にします。ただし、2番目の犠牲者は、減衰または射撃の減速により、半分のダメージしか受けません。過貫通は、大量のダメージを与える高出力武器や、何らかの徹甲能力を持つ武器により有用です。最初の標的に与えられた特殊効果は2番目の標的には適用されません。

#### 行動制限

銃器のみ。「優位を保つ」と同様に、この特殊効果は、標的が攻撃者の命中ロールに対して意志力対抗ロールを行うことを強制します。失敗した場合、標的は利用可能なカバーの陰に身を隠し、次のターンに反撃できません。行動制限は、標的に実際のダメージが与えられなくても（回避に成功したり、射撃がカバーに当たったりした場合でも）、非常に近くを通過する銃撃の威嚇効果に依存するため機能します。

行動制限された犠牲者は、必要な時間反撃することはできませんが、カバーから身を晒さない限り、新しいカバーに這い寄る、他者とコミュニケーションを取る、武器をリロードするなど、他の行動を行うことができます。

#### カウンター準備

防御者は敵のパターンを読み取り、特定の特殊効果に対するカウンターを準備します（これは秘密裏に記録されるべきです）。敵が戦闘中に選択した特殊効果を彼に与えようと試みた場合、防御者は攻撃者の効果を自身の効果と即座に置き換え、それは自動的に成功します。

#### 迅速リロード

遠隔武器を使用する際、攻撃者は次の射撃のリロード時間を1減らします。この効果は累積可能です。銃器の場合、黒色火薬銃や単発銃には有用ですが、自動火器（次の弾薬が瞬時に装填またはチャージされる）に適用した場合、その効果はあまり明らかではありません。しかし、迅速リロードは、武器の弾倉を素早く交換するために使用できます。また、コンデンサの再充電や銃身の冷却に長いサイクル時間を要するエネルギー兵器にも適用できるかもしれません。

#### 敵を傷つける

戦闘員は相手に、その後の人生で皆の語り草となるような傷跡を与えます。例えば、ほとんど切断された喉や、胸に巧妙に刻まれた「Z」の文字などです。

#### 標的選択

攻撃者がファンブルした場合、防御者は攻撃を操るかそらすことで、隣接する傍観者を代わりに命中させることができます。これは、新しい標的が攻撃者の近接武器のリーチ内にあるか、遠隔攻撃の場合、射線上に立っている場合に可能です。新しい犠牲者は予期せぬ事故によって完全に不意を突かれ、攻撃を避ける機会はなく、自動的に命中します。しかし、その代償として、特殊効果を受けることはありません。

#### 抜け出す

クリティカル時に、防御者は自動的に絡みつき、掴み、または行動制限の状態から脱出できます。

#### 部位スタン

攻撃者は鈍器を使い、命中した身体部位を一時的にスタンさせることができます。攻撃がアーマーポイントを克服し、標的に負傷を与えた場合、防御者は元の攻撃ロールに対する耐久力対抗ロールを行わなければなりません。防御者が失敗した場合、その命中部位は受けたダメージに等しいターン数だけ行動不能になります。胴体への打撃は防御者をよろめかせ、息切れさせ、防御しかできなくなります。頭部への射撃は敵を一瞬意識不明にし、行動不能にします。

#### 破壊

攻撃者は斧などの適切な武器を使用し、相手の防具や自然な保護を損傷させることができます。受け流しや魔法による減算後の武器ダメージは、保護のアーマーポイント値に適用されます。アーマーポイントを超える余剰ダメージは、その防具（装着部位）のアーマーポイント値を減少させるために使用されます。ストラップを引き裂く、リングを破る、プレートに折り目を付ける、または怪物の皮、鱗、キチンを裂き取るなどです。保護のAPがゼロになった後もダメージが残る場合、それは命中した部位のヒットポイントに持ち越されます。銃器の場合、破壊はアブレイティブアーマーを破壊するために設計されたエネルギーベースの銃器や、重力波銃のようなよりエキゾチックな概念に適しているでしょう。

#### 敵を転倒させる

キャラクターは相手のバランスを崩すか、地面に投げつけることを試みます。相手は、キャラクターの元のロールに対する自身の筋力、回避、またはアクロバット技能の対抗ロールを行わなければなりません。標的が失敗した場合、うつ伏せに倒れます。四足の相手（またはそれ以上の脚を持つクリーチャー）は、回避技能の代わりに運動技能を使用し、ロールを1難易度段階容易にすることができます。銃器の場合、質量のある弾薬を発射する銃器であれば、着用している保護具を貫通しなくても、相手の脚を狙ってこの効果を使用できない理由はありません。

#### 武器故障

攻撃者ファンブル、銃器のみ。攻撃者の銃器が故障し、現場で分解して清掃する時間を費やすまで使用不能になる。

#### 離脱

防御者は自動的にリーチ外に離脱し、その特定の相手との交戦を断ち切ることができます。

## 近接戦闘

近接戦闘とは、戦闘員が武器や自然の武器で互いを攻撃する古典的な意味での白兵戦です。以下のガイドラインは、近接戦闘をより面白くし、この章の冒頭で示唆されたいくつかの概念をさらに説明する方法を提供します。

### 状況修正

状況修正は、特定の条件下での戦闘時、または特殊効果の結果として適用される場合があります。例えば、暗闇での戦闘は、超人的な難易度グレードを課す可能性があります。特に明記されていない限り、修正はゲームマスターによって決定されます。2つ以上の状況がキャラクターに関連する場合、最も厳しいものを適用します。

一部の修正は状況依存ですが、難易度グレードを適用するのではなく、戦闘技能は関連する技能によって制限されます。例えば、馬の背中からの戦闘は、戦闘員の戦闘技能を騎乗技能の値で上限とし、水中での戦闘はキャラクターの水泳技能の値で上限とするかもしれません。

#### 近接戦闘の状況修正

| 状況 | 難易度グレード |
| :-- | :-: |
| 無力な標的を攻撃する | 自動 |
| 閉じ込められた状況での攻撃 | 困難 |
| 低地で防御中、または騎乗した敵に対して防御中 | 困難 |
| 不安定な地面での戦闘 | 困難 |
| しゃがみながらまたは片膝をついての戦闘 | 困難 |
| 視界不良の状況での戦闘（濃霧、吹雪） | 困難 |
| 背後からの攻撃に対する防御 | 手ごわい |
| 伏せた状態での戦闘 | 手ごわい |
| 部分的な暗闇での戦闘（薄暗い照明） | 手ごわい |
| 真っ暗な状況での戦闘（全く照明がない） | 超人的 |
| 目が見えない、または主要な知覚感覚の喪失 | 超人的 |

### 交戦

キャラクターは、相手の近接武器の範囲内にいる場合、交戦中と見なされます。これは、両方の戦闘員が互いに到達できることを必ずしも要求するものではなく、どちらか一方が他方に攻撃される可能性があることだけを意味します。一度キャラクターが相手と交戦すると、その戦闘から自由に離れることはできません（通り過ぎる突撃は特別な例外です）。相手との接触を断ち切るには、通常、「巧みな立ち回り」などの戦闘アクションや「離脱」特殊効果の使用など、何らかの意図的な行動が必要です。

### 突撃

突撃アクションは、攻撃の力と衝撃を増すために素早く移動する行為です。突撃には、攻撃者が突撃アクションを選択し、走行または全力疾走の歩行タイプ（上記の[移動速度](0002_Characters.md?id=movement-rate)セクションを参照）で移動する必要があります。移動ルールは『ミトラス』ルールブックで完全にカバーされています。

もしこれが相手との接触につながる場合、そこで停止し、突撃によって修正された攻撃アクションを実行しなければなりません。突撃が行われる前にカバーしなければならない最低距離はなく、移動距離よりも達成された勢いが要因となります。したがって、走るか全力疾走します。

突撃中、攻撃ロールは難易度「困難」のペナルティを受けます。その代わり、突撃は攻撃武器のサイズを1段階大きくし、二足歩行の攻撃者のダメージ修正値を1段階、四足歩行（またはそれ以上の脚を持つ）の場合は2段階増加させます。騎乗キャラクターは、自分のダメージ修正値の代わりに騎乗獣のダメージ修正値を使用できます。突撃するキャラクターの最終的な位置は、交換の結果と、攻撃者が停止するか、通り過ぎてすぐに交戦を解除したいかによって異なります。

受け流しまたは回避する代わりに、突撃の対象は同時に突撃する攻撃者に反撃することができます。この場合、より長い武器の使い手が最初に攻撃します。武器が突撃を受け止めるように設定できる場合（槍など）、対象は自身のダメージ修正値の代わりに突撃者のダメージ修正値を使用できます。

### カバー

敵は攻撃を妨害するためにカバーを使用します。これは、遮蔽物の頑丈さによって物理的に攻撃を防ぐか、敵が正確にどこにいるかを隠すことによって行われます。カバーの正確な価値は、その保護の厚さとカバー範囲に依存します。

遮蔽された命中部位に着弾した標的への攻撃は、カバー固有の保護の範囲でブロックされます。カバーは、「部位選択」特殊効果を使用して標的の見える部分を狙うか、武器が貫通できる場合はカバーを貫通して攻撃することによって部分的に無効化できます。

カバーが完全であるものの、攻撃者が攻撃が貫通すると信じる場合、隠れた標的に盲目的に攻撃することができます。そのような場合、攻撃ロールは1段階難しくなり、特殊効果を獲得しても「部位選択」は選択できません。これは、攻撃者がカバーの背後にある標的の位置をかなり正確に知っていることを前提としています。

### 回避

戦闘における回避とは、圧倒的な攻撃から身をかわすことです。これには、防御者の回避技能と、攻撃者の関連するロールとの対抗ロールが必要です。これは、突進する戦車の運転技能から魔術師の呪文詠唱技能まで、何でもあり得ます。攻撃者が勝てば、通常通りダメージを与えます。防御者が勝てば、ダメージは完全に回避されます。

対抗ロールの勝者が相手に対して1つ以上の成功レベルを達成した場合、通常の戦闘と同様に適切な特殊効果を選択できます。結果がどうであれ、回避行動は防御者をうつ伏せにします。通常、キャラクターは次のターンに「足場を回復する」アクションを取って再び立ち上がる必要があります。ただし、「立ち上がる」特殊効果を使用できる場合は別です。

### ノックバック

受けるダメージが対象のSIZを超える攻撃は、デフォルトでキャラクターをノックバックさせます。この状況でのダメージは、受け流しや防具による軽減前のものです。そのような攻撃を受けた際、対象は倒れ伏すのを避けるために易しいアクロバットロールまたは標準的な運動技能ロールに成功しなければなりません。また、SIZを超えるダメージ5点ごとに（またはその端数ごとに）1メートル後退させられます。先行型で「体勢を固める」戦闘アクションを使用することで、ノックバックの効果を軽減または無効にすることができます。

### 跳躍攻撃

跳躍攻撃は様々な状況から繰り出されます。そのほとんどは待ち伏せの一部として、または突撃の終わりに引き起こされますが、一部のクリーチャーは助走や高所からの必要なしに相手に飛び乗ることができます。

跳躍攻撃は、跳躍者の運動技能と、防御者の筋力または回避技能の対抗ロールで解決されます。四足の標的は攻撃ロールを1難易度段階難しくします。跳躍者が勝てば、防御者は自動的に倒れ伏し、攻撃者がその上に乗り上げます。失敗は、防御者が衝撃に耐えたか、身をかわしたことを意味します。対抗ロールの勝者が相手に対して1つ以上の成功レベルを達成した場合、通常の戦闘と同様に適切な特殊効果を選択できます。

跳躍自体でダメージは与えられません。ただし、その後に倒れ伏した犠牲者は、攻撃者が去るか、最終的に「立ち上がる」ことを許可する特殊効果を獲得するまで、足場を回復することはできません。さらに、跳躍攻撃は攻撃者のSIZの2倍までの相手にのみ有効です。標的は、差し迫った跳躍攻撃を認識している限り、「体勢を固める」戦闘アクションを積極的に使用することで、自身を転倒させにくくすることができます。

### 巧みに立ち回る

複数の相手と対峙しているキャラクターは、常に位置を shifting し、敵同士が邪魔し合うようにすることで、一度に攻撃できる敵の数を制限するために移動を利用できます。「巧みに立ち回る」には、キャラクターが動き回るスペースがあり、狭い場所に閉じ込められていないことが必要です。

「巧みに立ち回る」には、キャラクターが敵を回避技能のグループ対抗ロールに巻き込む必要があります。巧みに立ち回るキャラクターと彼を追い詰めようとする敵の両方の参加者は、アクションポイントを消費しなければなりません。そして、それぞれが一度ロールし、巧みに立ち回るキャラクターのロールに勝てなかった者は、その戦闘ラウンドの残りの間、味方や地形の特徴によってブロックされ、彼を攻撃できません。

巧みに立ち回るキャラクターが全ての相手に勝った場合、そのラウンドの残りの間、単一の敵と安全に交戦するか、戦闘から完全に離脱するかを選択できます。

### 受動的防御

受動的防御は、武器を持つキャラクターが、選択した部位（盾を使用する場合は複数の部位）をカバーするように武器を構えることを可能にします。ただし、その武器で積極的に受け流すことはできません（部位防御を参照）。その部位を命中させた攻撃は、自動的に武器または盾の恩恵を受け、通常通りダメージを軽減します。この技術は、戦士が弱点や負傷した部位を守りたい場合に一般的に使用されます。二刀流、または武器と盾を組み合わせた戦闘員がもう一方の武器で積極的に受け流すのを妨げるものはありません。

受動的防御はカバーと同じように機能するため、近接戦闘では「部位選択」特殊効果を使ってブロックを迂回することはできませんが、相手は他のブロックされていない領域を攻撃するためにこの効果を使用することはできます。盾の陰にしゃがみ込むことで、キャラクターは受動的防御中にカバーできる部位の数を倍にすることができます。

### 奇襲

奇襲は、攻撃者の存在や意図を知らない相手に対して予期せぬ攻撃が仕掛けられたときに発生します。待ち伏せは前者の例であり、友好的な会話中に疑う余地のない味方に裏切って襲いかかることは後者を示しています。

標的への奇襲の効果は強力です。

- 標的はイニシアチブに-10のペナルティを受けます。
- イニシアチブが来るまで、彼らは不意を突かれた状態と見なされ、身を守ることができません。
- 標的への最初の攻撃が成功した場合、ボーナス特殊効果を得ます。
- そのラウンドの残りの間、いかなる攻撃アクションも実行できません。

### 広範囲攻撃

広範囲攻撃は、ゲームマスターの判断で、異常なサイズの武器やクリーチャーが密集した敵グループを攻撃する場合に発生します。例えば、巨大なドラゴンの薙ぎ払う尾や、巨大なトリケラトプスの止められない突進などが、複数の敵を同時に攻撃します。

広範囲攻撃は、武器またはクリーチャーの単一の攻撃ロールを、その経路上のすべての標的に適用して行われます。各防御者は攻撃の効果を個別に解決しなければならず、攻撃者に課せられた特殊効果は同時に発生したと見なされます。

## 遠距離戦闘

遠距離戦闘には、標的に命中させるために撃つ、投げる、投擲するなどのあらゆる形態の武器が含まれます。遠隔攻撃は近接戦闘と全く同じ方法で解決されます。ただし、遠隔武器は通常、盾でしか受け流すことができません。盾を持っていない者は、自然なカバーに頼るか、「回避」を使って射線から身をかわす必要があります。したがって、軽装甲の敵に対しては、遠隔武器は恐るべき抑止力となり得ます。

各遠隔武器には、その有効性を決定するいくつかの特定の属性があります。戦闘に重要な影響を与えるものは以下の通りです。

- **威力:** 遠隔武器版のサイズ。攻撃またはその弾薬の貫通力を抽象的に測定するもので、攻撃が（盾による）受け流しを克服するかどうかを決定します。
- **ダメージ修正値:** この属性は、攻撃者のダメージ修正値が遠隔武器のダメージロールを増幅するために使用できるかどうかを示します。一般的に、自力で引く弓や投擲武器のみが、ユーザーが自分のダメージ修正値を適用することを許可します。
- **射程:** スラッシュで区切られた3つの数字で、武器または弾薬の最大近距離、有効射程、長距離を表します。近距離は、「部位選択」特殊効果を使用できる距離であり、標的が静止しているか、差し迫った攻撃に気づいていない場合に限ります。有効射程には顕著な修正はありません。長距離では、武器は依然として危害を与えることができますが、ダメージ量は半分になり、威力は1段階減少します。
- **装填:** 弾薬を発射する武器の装填または再装填にかかるターン数です。キャラクターは、「迅速リロード」特殊効果を使用することで、装填または構える時間を短縮できます。
- **貫通サイズ:** 貫通した武器のサイズは、それが打ち込む威力とは異なります。遠隔貫通武器には、貫通特殊効果によって生じる妨害を考慮する際の実際のサイズを示す特別な列があります。

### 状況修正

近接戦闘と同様に、遠隔武器を特定の条件下で使用する際に状況修正が適用される場合があります。馬に乗って槍を投げるなど、困難な状況で武器を使用する場合、キャラクターの戦闘技能に上限が設けられることがあります。

##### 遠距離戦闘の状況修正

| 状況 | 難易度グレード |
| :-- | --- |
| 軽風\* | 困難 |
| 中風\* | 手ごわい |
| 強風\* | 超人的 |
| 突風、嵐、またはそれ以上\* | 絶望的 |
| 標的が走行中 | 困難 |
| 標的が全力疾走中 | 手ごわい |
| 標的が霧で不明瞭、または部分的な暗闇にいる | 困難 |
| 標的が濃煙で不明瞭、または暗闇にいる | 手ごわい |
| 標的が完全に不明瞭 | 超人的 |
| 標的が完全に不可視 | 不可能\*\* |
| 標的が伏せている | 手ごわい |

### 照準

遠隔武器で追加の時間をかけて照準を合わせることで、キャラクターは命中する可能性を高めることができます。照準合わせには、武器を固定し、例えば風の一時的な静寂を待ったり、標的が2つの障害物の間を移動するまで待ったりして、最高の機会を待つために戦闘ラウンド全体を費やす必要があります。照準を合わせることで、キャラクターは射程または状況修正の難易度を1段階下げることができます。追加のラウンドを照準合わせに費やしても、それ以上の利点はありません。

### 自動射撃

一部の銃器は、前回の射撃による排気ガス、反動、またはエネルギーを利用して、次の弾薬を自動的に装填します。再装填の必要がないため、ユーザーがどれだけの弾薬を発射したいかに応じて、様々な発射速度を提供します。

これらの発射モードは、セミオート、バースト、フルオートとして知られています。この能力によって一部の銃は恐ろしいダメージを与えることができますが、反動する武器で精度を維持しようとすることは問題です。以下のルールは主に投射弾を発射する自動火器に適用されると仮定してください。

#### セミオート

これは実際にはほとんどの銃のデフォルトの発射速度であり、トリガーを引くたびに一発が発射され、次の弾丸が装填されます。

#### バースト

バースト射撃の場合、トリガーを一度引くと少数の弾丸が発射されます。通常、武器にもよりますが3発から5発以内です。その本来の目的は、限定的な抑制射撃を可能にしつつ、過剰な弾薬の使用を制限することでした。これは射手の命中率を向上させることを意図していません。なぜなら、連続する高速な反動が絶えず武器を狙いからずらすからです。バーストモードで発射すると、攻撃ロールに「困難」のペナルティが課せられます。しかし、攻撃が成功した場合、射手はそのバーストで何発が標的に命中したかを決定するためにダイスをロールします。したがって、3発バーストは1d3を使用します。命中した各弾丸について、個別の命中部位とダメージをロールします。

#### フルオート

通常、複数の標的を制圧するための本格的な抑制射撃に用いられますが、フルオート射撃は密集した敵集団に発射された場合、壊滅的な威力を持つことがあります。射手は、射撃弧に含まれるすべての標的に均等に分配される弾薬の数（武器の連射速度の制限まで）を決定します。余分な弾薬は照準移動中に失われたと見なされます（したがって、6つの標的に15発発射した場合、それぞれに2発ずつ命中します）。各標的について個別の攻撃ロールが行われますが、継続的な射撃による反動効果の増加のため、難易度は「手ごわい」です。命中した標的は、バースト射撃と同様にランダムな数の弾薬のダメージを受けます。

バーストまたはフルオート射撃中の最初の命中弾のみが特殊効果の恩恵を受けることに注意してください。したがって、オートモードで射撃中にクリティカル攻撃をロールした場合、たとえ残りのすべての弾丸が命中したとしても、最初の命中弾のみで「ダメージ最大化」を選択できます。

#### なぜ自動射撃はこれほど厳しくペナルティを受けるのか？

自動射撃のルールは、現代の銃器の明らかな精度と比較して、不当に厳しいように見えるかもしれません。これにはいくつかの理由があり、主なものは合理化とゲームバランスです。キャラクターの不正確さを批判するプレイヤーは、ルールのいかなる微調整も彼らにも適用されることを思い出すべきです。これは、速射の致命性を考慮すると、身の引き締まる思いでしょう。

もちろん、これらの反動ペナルティの一部は、例えば自動火器を固定するために「武器を固定する」を使用することで軽減できます。しかし、実際には、銃撃戦は射撃場での射的とは全く異なります。弾丸が飛び交う中、カバーを取る本能、機動性を保つ必要性、そして発砲炎や音によって射手の位置が特定される一発一発の射撃は、しばしば照準や固定の機会を奪います。「撃って祈る」という言葉は、非常に正当な理由で生まれました！

### 二丁拳銃

キャラクターは二丁拳銃を使用し、自分のターン中に両方の武器を同時に発射することで、より速い連射速度を得ることができます。しかし、これは精度にペナルティをもたらします。両手にピストルを持って発射すると、標的に命中するのに「困難」のペナルティを受けます。両手にカービン銃やSMGを持って同時に射撃しようとする者は、「手ごわい」のペナルティを受けます。

### 群衆への発砲

乱戦の渦中に発砲することは危険な行為であり、そのため、もともと狙っていた標的以外の誰かに偶然当たってしまうリスクが常に存在します。群衆や乱戦の端にいる特定の標的に発砲する場合、攻撃は「困難」の難易度グレードを受けます。群衆を貫通して発砲しようとする場合は「手ごわい」に引き上げられるべきです。危険を軽減するため、照準合わせは依然として許可されます。

射撃手が難易度ペナルティにもかかわらず攻撃ロールに成功した場合、彼は狙い通りに命中します。しかし、射撃手がロールに失敗したが、「群衆への発砲」ペナルティがなければ成功していたはずの場合、隣接する犠牲者が代わりに命中します（犠牲者は通常通り回避または受け流しを行うことができます）。

複数の標的が射線上にいる場合、ゲームマスターは犠牲者をランダムに決定すべきです。攻撃によって獲得された特殊効果は、元の標的のみに適用され、偶然命中した傍観者には適用されません。

### 銃器でのファンブル

銃撃は通常、対抗技能によって抵抗されることはないため、射手が攻撃ロールをファンブルした場合、特別な規定が設けられるべきです。この場合、意図された標的であった者は、ファンブル固有の特殊効果を選択することが許可され、射手はそれを被ることになります。ファンブルのみの特殊効果である「武器故障」も参照してください。

### 盾での銃撃受け流し

前述の通り、超音速の投射物やエネルギービームを意図的に受け流すことはほぼ不可能です。しかし、盾は本質的に携帯可能な「カバー」であるため、ある程度の保護を提供します。銃撃戦で盾を使用するには、上記の[近接戦闘](0005_Combat.md?id=passive-blocking)セクションの受動的防御のルールを使用するだけです。

より近代的な銃器やSF的な性質の銃器に関しては、ゲームマスターは盾の素材固有のアーマーポイント値を割り当てることを検討してもよいでしょう。例えば、木製の盾は4アーマーポイント、ケブラー製タクティカルシールドは8AP、ユリシーズ31ブレスレット型力場シールドは15APとして扱われるかもしれません。

### 望遠照準器

望遠照準器は、ユーザーが適切に照準を合わせる時間をとる場合（照準合わせを参照）、より遠距離での銃器の精度を向上させます。最も基本的な望遠照準器は単なる光学拡大器ですが、技術が進歩するにつれて、距離計、弾道計算機、空気移動センサー、大気レンズ補償器、IRまたは低照度増幅器、その他多くのガジェットを統合し、その有効な「命中」距離を向上させます。

望遠照準器は、照準器の洗練度に応じて、技能ロールを1段階以上容易にします（[技能の修正](0003_Skills.md?id=modifying-skills)を参照）。いかなる照準器も技能ロールを自動成功にすることはできないため、達成できる最良のグレードは「非常に容易」です。
