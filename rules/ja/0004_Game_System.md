# ゲームシステム

_Mythras Imperative_のゲーム中には、様々な異なる出来事が起こり得ます。そのいくつかはスキル使用や、戦闘や魔法といった主要なゲームメカニクスを通じて解決されます。また、裁定のために独自のルールを必要とするものもあります。この章では、プレイヤーがゲーム中に遭遇する様々なシナリオに対するルールとガイダンスを提供します。

このリストは網羅的ではありませんが、最も一般的な状況を網羅しているはずです。ゲームマスターは、何らかのゲームシステムによる裁定を必要とする他の状況があり、それらの状況が頻繁に発生するように思われる場合、この章で示されたものを複製または開発の基礎として、それらの特別な状況をカバーするための独自のルールを開発することが推奨されます。

## 窒息

キャラクターは耐久スキルと同じ秒数だけ息を止めることができます。ただし、キャラクターは準備（肺にできるだけ多くの空気を満たす）をしていなければなりません。準備ができていない場合、受動的な状況では時間が半分になり、激しい活動に従事していた場合は5分の1に減少します。

息を止めている期間が過ぎたら、キャラクターは耐久ロールを行わなければなりません。

- ロールがクリティカル成功した場合、それ以上の悪化は起こりません。
- ロールが成功した場合、キャラクターは疲労レベルを1つ追加で蓄積します。
- ロールが失敗した場合、キャラクターはそのラウンドで1d2の疲労レベルを追加で受けます。
- ロールがファンブルした場合、キャラクターはそのラウンドで1d3の疲労レベルを追加で受けます。

援助がなければ、窒息による死は通常迅速です。キャラクターが死ぬ前に窒息が終わった場合、窒息によって失われた疲労レベルは比較的早く回復し、1分あたり1レベル回復します。

窒息の方法によっては、ゲームマスターは完全な回復を妨げることを望むかもしれません。その場合、煙や水の吸入による肺の損傷を表すために、永続的な疲労レベルを課します（[疲労](0004_Game_System.md?id=fatigue)のセクションを参照）。

## キャラクターの成長

すべての_Mythras Imperative_キャラクターは、時間と経験とともに成長できます。成長はいくつかの領域で実行できます。

- 既存スキルの向上
- パッションの向上
- 新しいスキルの習得
- 新しい能力の習得

#### 経験ロール

ほとんどのキャラクター成長のメカニズムは経験ロールです。ゲームマスターは、キャンペーンの適切な時期に経験ロールを付与します。各成功シナリオやストーリーラインの終了時、またはストーリーが長く完了に時間がかかる場合は、おそらく2〜3セッションのプレイ後に付与します。頻度はゲームマスターの裁量に委ねられます。経験ロールの頻度が高いほど、キャラクターはより速い速度で成長します。

経験ロールを付与するタイミングに正解も不正解もありませんが、物語の自然な区切りが適切な時期を示唆するかもしれません。しかし、決定を下すのは常にゲームマスターです。プレイヤーは経験ロールを要求することはできませんが、特定の時点での付与を期待する権利はあります。キャラクターはそれなしには成長できないからです。

付与される経験ロールの数もゲームマスターによって決定されます。ただし、前回の付与からのセッションごとに1〜3個の経験ロールが良い目安です。

#### 既存スキルの向上

キャラクターシート上の標準スキルまたは専門スキルは、経験ロールを1つ消費することで向上させることができます。プレイヤーはスキルの向上に経験ロールを費やさないことを選択し、代わりに他のもの（クラス能力など、下記参照）を向上させるために蓄積することができます。

プレイヤーは1d100をロールし、向上させるスキルと比較します。キャラクターの知力（INT）がロールに加算されます。

- ロールした数値が向上させるスキル以上の場合、スキルは1d4+1%増加します。
- ロールした数値が選択したスキル未満の場合でも、スキルは1%だけ増加します。
- キャラクターが先行するセッション（前回の経験ロール付与から現在まで）中にいずれかのスキルをファンブルした場合、ファンブルしたスキルは1%の自由な増加を得ます。私たちは成功よりも失敗から多くを学び、これは壊滅的な失敗の後にキャラクターが経験する反省を表します。同じスキルの複数回のファンブルは累積しません。ファンブルしたスキルが経験ロールで上げられる場合、ファンブルによる1%は向上ロールが行われる前に適用されます。

同じスキルは、1セッションにつき1回以上の経験ロールから恩恵を受けることはできません。

#### パッションの向上

[パッション](0002_Characters.md?id=step-8-passions)のセクションで説明されているように、パッションの値はスキルと同じ方法で経験ロールを使って向上させることができます。ゲームプレイで裏付けられる場合、ゲームマスターは経験ロールを使用してパッションを減少させることさえ許可するかもしれません。

_例えば、報われない恋からの絶え間ない拒絶は、最も熱烈な恋人であっても、過去の亡霊に囚われずに恋愛を追求したいと願う者にとっては、最終的にパッションを消耗させるかもしれません。_

#### 新しいスキルの習得

一部のキャラクターは、自分の文化から学ぶ機会がなかった新しい専門スキルを習得したいと願うかもしれません。経験ロールを投資し始める前に、まず知識源を見つける必要があります。これは、教団やギルドが提供するような専門の家庭教師のように日常的なものかもしれません。あるいは、古く朽ちた訓練の巻物のような、より異質な教育源かもしれません。

教育源が見つかれば、キャラクターはそのスキルに基本的な基礎を築く（基本レベルで学ぶ）ために、まるまる1ヶ月の学習と練習に費やさなければなりません。これには3つの経験ロールに加え、教師への支払い（もしいる場合）や必要な装備や道具の購入（またはレンタル）に必要なゲーム内コストがかかります。

## 病気と毒

キャラクターに影響を及ぼす最も恐ろしいものの中には、世界の暗く陰鬱な場所にはびこる無数の病気と毒があります。キャラクターの故郷の都市を疫病が襲い、彼らがまだその門内にいるときほど不安なものはありません。同様に、ベッドで毒蛇を見つけたり、巨大なサソリに襲われたりすることは、高いドラマの瞬間となり得ます。

病気と毒はその種類においてほぼ無限ですが、ゲーム目的では同じ方法で扱われます。

#### 病気と毒の特性

すべての病気と毒は、その効果にとって重要な多くの特性を示します。

**適用:** 病気または毒が犠牲者に導入される方法。

- **摂取:** 飲食を通じて効果が現れます。
- **吸入:** 呼吸または鼻から吸い込むことによって効果が現れます。
- **接触:** 皮膚を通じた吸収によって効果が現れます。
- **注射:** 物質が刺し傷を通じて体内に注射されるときに効果が現れます。

**効力:** 病気または毒の毒性。この値は、対抗ロールで適切な抵抗スキル（通常は耐久または意志力）と対比されます。キャラクターがロールに勝てば、その効果を跳ね返します。ロールに負ければ、病気または毒のすべての条件を、適切な時期にそれぞれ受けます。

**抵抗:** 病気または毒にどのように抵抗するか – 通常は耐久または意志力ですが、より異質な抵抗要件を持つ場合があります。抵抗は曝露時にロールするか、感染または中毒の事実を隠すために発症時間まで延期することができます。

**発症時間:** 多くの病気や毒は即座には効果を発揮しません。遅延は発症時間と呼ばれ、これは数秒、数分、数時間、あるいはそれ以上になることがあります。複数の効果を持つ毒や病気は、条件で説明されているように、それぞれ異なる発症時間を持つ場合があります。

**持続時間:** 病気または毒の条件が続く期間。

**条件:** すべての毒素には1つ以上の条件があります。これらは以下の条件表に記載されている特定の効果を持ち、犠牲者が抵抗に失敗した場合、記載されているすべての条件に苦しみます。

**解毒剤/治療法:** 毒素が治療可能であれば、ここに記されます。治療に成功すれば、それ以上の条件の発生は防げますが、状況によっては既に与えられた永続的な効果に苦しむことがあります。

##### 条件表

| 条件 | 効果 |
| :-- | :-- |
| 激痛 | 犠牲者は激しい痛みによって妨げられます。特定の部位であろうと全身であろうと、影響を受けた部位の使用を伴ういかなるスキルロールも、キャラクターの意志力以下でなければならず、そうでなければ試みは失敗し、呻き声や叫び声を上げます。 |
| 窒息 | 犠牲者は窒息します – 彼は意識を失い、呼吸できなくなります。[窒息](0004_Game_System.md?id=asphyxiation)セクションのルールが使用されます。窒息はぜんそく性の性質を持つ場合があり、犠牲者は息切れのみを経験するか、完全な呼吸不全で死に至る場合があります。後者の場合、応急処置スキルと病気または毒の効力との対抗テストに勝つことで、犠牲者を生き延びさせることができます。 |
| 出血 | 犠牲者は内出血または表面的な出血に苦しみ、下記の[疲労](0004_Game_System.md?id=fatigue)および[失血](0004_Game_System.md?id=blood-loss)セクションで説明されている疲労および回復時間を引き起こします。 |
| 伝染性 | 犠牲者は触れることによって毒または病気を移すことができます。 |
| 死 | 犠牲者は意識を失って倒れ、肉体（CON）特性値と同じ数のラウンド後に死亡します。 |
| 発熱 | 犠牲者の体温は急激に変動し（熱くなったり冷たくなったり）、筋肉が痛みます。すべてのスキルは難易度「困難」のペナルティを受けます。 |
| 狂気 | 犠牲者は、水を避ける、偏執、自傷行為などの強迫観念に従うよう駆り立てられます。病気または毒によって引き起こされる狂気はその記述に記載されます。 |
| 吐き気 | 犠牲者は食事をとることができず、ストレスの多い身体活動を行うたびに耐久力でロールし、物理的に気分が悪くなるのを避ける必要があります。嘔吐は1d3ラウンド続き、その間は行動できません。吐き気が長時間続くと飢餓を引き起こす可能性があります。 |
| 麻痺 | 犠牲者は物理的に動けません。影響を受けた部位が特定の場所であれば、その間使用できません。全身に影響する場合は、キャラクターは全く動けません。 |

#### 病気の例：赤斑病

赤斑病は、感染者との皮膚接触、または彼らが触れたものとの接触によって広がる厄介な病気です。物体から病気に感染する可能性は接触後1d3日間持続するため、一度発生すると封じ込めるのが非常に困難です。

犠牲者は最初に軽度の発疹に苦しみますが、1日以内に重い発熱に見舞われ、通常は病気の全期間にわたって寝たきりになります。この間、発疹は恐ろしくかゆい膿疱に発展し、わずかな動きで破裂し、特に脇の下や股間から少量の血が滲み出します。この段階は1週間以上続くことがあり、致命的になる可能性があります。

- **適用:** 接触
- **効力:** 50
- **抵抗:** 耐久
- **発症時間:** 1d6+6時間
- **持続時間:** 1d6+3日
- **条件:** 病気が発現すると、犠牲者は伝染性になります。1日後、発熱と出血に苦しみ始めます。膿疱から血が滲み出るごとに、犠牲者は疲労レベルを1つ失い、この疲労は病気の期間が完了するまで回復しません。
- **解毒剤/治療法:** 治癒スキルで治療可能です。

## 落下

落下によって受けるダメージの量は、落下距離に依存します。アーマーポイントは落下ダメージを軽減しませんが、落下したキャラクターまたはクリーチャーのダメージ修正値が落下によって受けるダメージに適用されます。

##### 落下距離表

| 落下距離 | 受けるダメージ |
| :-- | :-- |
| 1メートル以下 | ダメージなし。 |
| 2-5メートル | ランダムな1部位に1d6点のダメージ。 |
| 6-10メートル | ランダムな2部位に2d6点のダメージ。 |
| 11-15メートル | ランダムな3部位に3d6点のダメージ。 |
| 16-20メートル | ランダムな4部位に4d6点のダメージ。 |
| 5メートル増すごと | +1d6ダメージ。 |

#### 落下物

落下する物体は、そのSIZと落下距離に基づいてダメージを与えます。物体はSIZの6点ごと（またはその端数）に1d6ダメージを与え、さらに落下距離表の「受けるダメージ」列に示されたダメージ量を与えます（小型の物体に対する軽減を含む）。

## 疲労

疲労は疲れとその段階的な影響を測ります。これは、激しい活動から病気の衰弱効果まで、様々なことを追跡するために使用されます。

疲労を蓄積する主な方法は、何らかの身体活動に従事することです。運動や仕事が厳しければ厳しいほど、キャラクターは早く疲れます。ゲームマスターの裁量により、キャラクターは疲労レベルの増加に抵抗するために、適切なスキルロール（運動、腕力、または耐久）を行わなければなりません。

失敗したロールごとに疲労レベルが1つ蓄積されます。各疲労レベルは、スキル使用、移動、イニシアチブ、およびアクションポイントにペナルティをもたらします。窒息、失血、およびその他の設定依存の効果も疲労の蓄積に寄与します。

ほとんどのキャラクターにとって、「行動不能」レベルに達すると、あらゆる種類の活動がほぼ不可能になります。この段階では、キャラクターはまだ意識がありますが、最も絶望的な活動以外は何もできません。

行動不能を超えると、キャラクターは全く行動できません。「半意識」、「昏睡」、「死亡」の疲労レベルは、一般的に窒息、病気、失血、飢餓、露出などによる最も極端な影響を測定するために予約されています。

#### 疲労からの回復

キャラクターは、治癒率に応じて疲労から回復します。蓄積された各疲労レベルから回復するのに必要な完全な休息の量は、回復期間をキャラクターの治癒率で割ったものに等しくなります。

以下の表は身体的疲労からの回復を示しています。窒息からの回復ははるかに速い場合がありますし、失血からの回復は遅い場合があります。

| :-- | :-: | :-: | :-: | :-: | :-: |
| 新鮮 | – | – | ペナルティなし | – | – |
| 息切れ | 困難 | ペナルティなし | ペナルティなし | ペナルティなし | 15分 |
| 疲労 | 困難 | -1メートル | ペナルティなし | ペナルティなし | 3時間 |
| 消耗 | 強大 | -2メートル | -2 | ペナルティなし | 6時間 |
| 疲弊 | 強大 | 半減 | -4 | -1 | 12時間 |
| 衰弱 | 超人的 | 半減 | -6 | -2 | 18時間 |
| 行動不能 | 超人的 | 不動 | -8 | -3 | 24時間 |
| 半意識 | 絶望的 | 活動不能 | 活動不能 | 活動不能 | 36時間 |

## 火災

火は、武器として使用される場合や制御不能に燃え盛る場合、常に危険の元です。下記の火災強度表は、熱ダメージに対する5つの異なる強度を示しており、いくつかの例も挙げています。ラウンドごとに与えられるダメージはダメージ列に記載されています。比較的小さいため、強度1および2の火源はそのダメージを単一のヒットロケーションに適用し、通常は火源に触れている部分に与えます。強度3および4の火源はより大きく、最も近い1d4+1のヒットロケーションにダメージを適用し、輻射熱の程度を示します。強度5の火源はすべてのヒットロケーションに同時に影響を与えます。

火は可燃物を発火させます。すぐに消火されない場合、そのような物質は火災の強度によって示される数のラウンド内に燃焼します。一度発火すると、可燃物は物理的に消火されるまで燃え続けます。ダメージは、アーマーポイントを無視して、直接物質のヒットポイントに、そしてその下の肉にも適用されます。制御されない場合、それはその強度に等しい数のヒットロケーションに、ラウンドごとに広まります。

##### 火災強度表

| 強度 | 例 | 発火までの時間 | ダメージ |
| :-: | :-- | :-: | :-: |
| 1 | キャンドル | 1d4 | 1d2 |
| 2 | 松明 | 1d3 | 1d4 |
| 3 | キャンプファイヤー | 1d2 | 1d6 |
| 4 | 部屋全体を埋める大火事 | 1d2 | 2d6 |
| 5 | 火山の溶岩 | 即時 | 3d6 |

火を基にした攻撃は、通常1ラウンドで終わるため、可燃物を発火させる確率が典型的にはパーセンテージで示されます。特に指定がない限り、この確率は火を基にした攻撃の場合はダメージロール値×5%、電気を基にした攻撃の場合はダメージロール値×1%です。一度発火した可燃物は、物理的に消火されるまで毎ラウンド1d3-1のダメージを与え続けます。このダメージは、アーマーポイントを無視して、直接物質のヒットポイントに、そしてその下の肉にも適用されます。ただし、クリーチャーの自然の鎧は通常の防御を提供します。犠牲者は、伏せることで炎を消そうと試みることができ、成功した運動ロールが必要です。これは1アクションポイントを要し、各ターンに試みることができます。他の者が何らかの方法で炎の消火を助ける場合、ロールの難易度は1段階下がります。ダメージロールがゼロダメージになった場合、その場所の火は自然に消えました。自然に消えた各場所について、炎を消すロールも1段階難易度が下がります。

## ゲーム時間

_Mythras Imperative_では、何がどの順序で起こるかを決定し、ゲームメカニクスが適切に適用されるようにするために、時間が重要な要素となります。ゲーム内の時間は、通常、プレイに費やされる時間とは異なることに注意してください。時にはゲームマスターは「アンタックス城に到達するのに一週間かかった」のように、何日もの出来事をたった一文で要約する必要があるかもしれません。また、特に戦闘中には、わずか数秒間の行動の解決に数分あるいはそれ以上の時間がかかることもあります。

以下にゲーム時間の重要な区別を示します。

#### 戦闘ラウンド

戦闘ラウンドは現実の5秒間を表します。これは、完了に数秒かかる短く激しい活動のバーストを測るため、または細部にわたる解決を必要とする非常に詳細な活動を測るために使用されます。例えば、ハンターと獲物の追跡で、各ラウンドが追跡の紆余曲折を決定する場合などです。

#### スキル時間

スキル時間は数分から数時間を表し、戦闘ラウンドの詳細な注意を必要としないが、明確な結果を伴う特定の熟考された努力を必要とする活動を測定するために使用されます。鍵開けには5分かかるかもしれませんが、城壁を巡回する警備チームの巡回を観察するには数時間かかるかもしれません。これらはどちらもスキル時間の例です。

#### ナラティブ時間

これはゲームマスターがプレイヤーに語りかけたり、プレイヤーが話し合ったりする際に費やされる時間です。特定の理由がない限り、実際のロールプレイングのほとんどはナラティブ時間で行われます。ロールプレイングでは、ナラティブ時間は現実時間と似ており、会話はプレイするのにかかる時間と同じだけかかります。ゲームセッションに長距離の移動や、正確な時間が重要でない活動期間が含まれる場合、時間は大幅に圧縮され、ゲーム時間の数瞬に短縮されます。

## ヒットロケーション

ほとんどの成功した（武器による）攻撃は、特定のヒットロケーションに着弾します。これはランダムに決定されることもあれば、「部位選択」などのスペシャルエフェクトによって選択されることもあります。どの部位にヒットしたかをランダムに計算するには、1d20をロールし、ロールした数字を関連する人型またはクリーチャーのヒットロケーション表と比較します。

動物やモンスターは、その独自の生理機能を反映するため、通常、人型とはわずかに異なるヒットロケーション表を持っています。これらは各存在の説明の一部としてリストされており、通常はゲームマスターのみが使用します。

人型ヒットロケーション表を参考のためにここに再掲します。

##### 人型ヒットロケーション

| 1d20 | ヒットロケーション |
| :-: | :-- |
| 1-3 | 右足 |
| 4-6 | 左足 |
| 7-9 | 腹部 |
| 10-12 | 胸部 |
| 13-15 | 右腕 |
| 16-18 | 左腕 |
| 19-20 | 頭部 |

特に魔法、テクノロジー、または特定のモンスター（ドラゴンの火の息や爆発する手榴弾など）によって引き起こされる一部の攻撃は、複数のヒットロケーションに同時に命中する可能性があることに注意してください。個々のヒットロケーションは、着用している防具またはクリーチャーの外面の自然な頑丈さによって装甲されている場合があります。これは、下記のダメージと負傷レベルを考慮する際に重要です。

## ダメージと負傷レベル

攻撃が成功した場合、武器のダメージ評価に応じてダメージを与えます。例えば、ショートソードは1d6ダメージを与えます。

武器または攻撃によってロールされたダメージは、特定の以下の順序で特定の条件によって修正されます。

1. 攻撃を行うキャラクターのダメージ修正値を適用します。
2. 武器強化（または軽減）魔法によるダメージを修正します。
3. 受け流された場合、比較する武器のサイズに応じて適切にダメージを軽減します。
4. ヒットロケーションが装甲されている場合、アーマーポイントの値だけダメージを軽減します。

攻撃によって与えられたダメージがゼロを超えている場合、その量のダメージをヒットロケーションが受け、そのダメージ合計は直ちにロケーションのヒットポイントから差し引かれます。ロケーションに残っているヒットポイントと、その結果生じる負傷カテゴリを記録します。

- **軽傷:** ヒットロケーションにまだ正のヒットポイントがある
- **重傷:** ヒットロケーションがゼロまたはそれ以下になった
- **致命傷:** ヒットロケーションが初期ヒットポイントと同じかそれ以上のマイナス値になった

部位が重傷または致命傷を受けた場合は、以下のルールを使用してください。耐久力チェックを要する負傷は、敵の成功した攻撃ロール値に対する対抗ロールでテストされることに注意してください。これは、熟練した戦士がより致命的な攻撃を行う能力を反映しています。耐久力ロールは、その部位が再び負傷しない限り繰り返されません。

### 軽傷

軽傷とは、切り傷、擦り傷、打撲傷、捻挫のことです。痛みがあり、出血するかもしれませんが、犠牲者の速度を落としたり、妨げたりするほど深刻ではありません。

### 重傷

部位のヒットポイントがゼロ以下になった場合、犠牲者は重傷を負います。その部位は永久的な傷跡が残り、傷の痛みによって気絶または気が散るため、次の1d3ターンは攻撃したり呪文を唱え始めたりすることはできません（ただし、受け流しや回避は可能です）。

手足に重傷を負ったキャラクターは、直ちに敵の成功した攻撃ロールに対する自身の耐久力の対抗ロールを行わなければなりません。失敗すると、その手足はヒットポイントが正の値に戻るまで使用不能になります。足の場合、犠牲者はうつ伏せに倒れます。腕の場合、持っていたものは落ちますが、物が縛り付けられている場合は除きます（常識で判断してください）。

腹部、胸部、または頭部に重傷を負ったキャラクターは、直ちに敵の成功した攻撃ロールに対する自身の耐久力の対抗ロールを行わなければなりません。失敗すると、重傷の原因となった攻撃で受けたダメージの量と同じ分だけ、数分間意識を失います。応急処置または治癒スキルを使って、意識不明の犠牲者を意識を取り戻させることができます。

ゲームマスターの裁量により、キャラクターが機能し続けたとしても、そのヒットロケーションの使用を必要とするすべてのタスクは、その負傷が軽傷になるまで、難易度を1段階下げる継続的なペナルティを受けます。

### 致命傷

部位が初期ヒットポイントと同じかそれ以上のマイナス値になった場合、キャラクターは致命傷を負います。キャラクターは直ちに行動不能となり、戦闘を継続できません。致命傷により手足は切断され、貫通され、粉砕され、または引き裂かれます。キャラクターはうつ伏せに倒れ、物理的に行動不能となり、直ちに敵の成功した攻撃ロールに対する自身の耐久力の対抗ロールを行わなければなりません。失敗すると、激痛により意識を失います。切断された、貫通された、または引き裂かれた部位が、自身の治癒率の5倍の分数以内に治療されない場合、キャラクターは失血とショックにより死亡します。

腹部、胸部、または頭部に致命傷を負った犠牲者は意識を失い、完全に活動不能となり、直ちに敵の成功した攻撃ロールに対する自身の耐久力の対抗ロールを行わなければなりません。失敗すると、即座に悲惨な死を遂げます（斬首、真っ二つにされる、心臓を貫かれる、引き裂かれるなど）。もし生き延びたとしても、その部位がキャラクターの治癒率の2倍の戦闘ラウンド数以内に治療されない場合、失血とショックによりやはり死亡します。

ほとんどの致命傷は、何らかの手術や強力な魔法による治療を必要とするため、負傷者が戦闘に復帰できるほど早く致命傷から回復することは非常に稀でしょう。利用可能な治療法にもよりますが、負傷した部位は恒久的な損傷を負う可能性があります（下記の[負傷からの回復](0004_Game_System.md?id=healing-from-injury)を参照）。

### 失血

外部および内部の負傷による失血は、通常戦闘中に負うものであり、キャラクターを非常に迅速に疲弊させることができます。その影響は、窒息と同様に疲労レベルの蓄積によって測定されます。無制限の失血は、何らかの医療処置がなければ通常死に至ります。窒息とは異なり、重度の失血が疲労に与える影響は永続的です。キャラクターは、出血が止まった翌日から1日あたり1レベルの割合で、出血によって失われた疲労を回復します。

### 英雄的最後の行動

主要なキャラクターが致命傷を負った特殊なケースでは、ゲームマスターは、最後の呪文を吐き出す、殺害者を斬り倒そうとする、隠れるために這いずり回る、自分の傷に止血帯を巻くなど、英雄的な最後の行動を許可するかもしれません。彼らはその行動に適用できる動く手足を持ち、幸運ポイントを消費する必要があります。彼らはその行動を試みる一度の機会を得て、その後すぐに意識を失い、おそらく死に至ります。

## 負傷からの回復

傷や怪我からの自然治癒は、キャラクターの治癒率に基づいています。治癒率は、怪我の性質に応じて、部位が回復するヒットポイントの量を決定します。

- **軽傷:** 日単位
- **重傷:** 週単位
- **致命傷:** 月単位

したがって、治癒率が3のキャラクターがヒットロケーションで-3のダメージを受け、重傷を負った場合、傷がゼロを超えて治癒するまでは週に3ヒットポイントの割合で自然治癒し、その後は完全に回復するまで1日あたり3ヒットポイントの割合で治癒します。

自然治癒には特定の制限があります。

- 治癒中のキャラクターは激しい活動に従事することはできません。そうしないと、治癒率が1d3減少します。したがって、軽傷から回復中のキャラクターでさえ、怪我を悪化させる可能性のある身体活動に従事することを決定した場合、進行が停止する可能性があります。
- 自然治癒は、犠牲者が成功した治癒ロールで治療されるか、何らかの治癒魔法がかけられるまで、致命傷の治癒を開始しません。切断を伴わない致命傷で、治癒スキルの20分の1の日数以内に治療されないものは、損傷を受け、その部位のヒットポイントが永続的に減少します。

## 無生物

すべての無生物は、ダメージと破壊への抵抗力を決定するために、アーマーポイントとヒットポイントを持っています。アーマーポイントはヒットポイントが影響を受ける前にダメージを軽減します。物体のヒットポイントがゼロになると、それは無用になります。

破壊の試みには、キャラクターが適切に応じて腕力、素手、または戦闘スタイルロールを行い、ダメージを与える必要があります。成功するとアイテムにダメージを与えます。

#### 武器 vs 無生物

アーマーポイントが武器のアーマーポイントと同じかそれ以上の無生物に対して武器を使用すると、物体と武器の両方にダメージを与えます。例えば、斧で鉄の扉を切ると、扉にダメージを与えるかもしれませんが、斧自体は鉄の扉よりもずっと早く破壊されるでしょう。ゲームマスターは、どの武器や道具がどの物体に影響を与えることができるかについて裁量を行使すべきです。例えば、ノミは木や石を彫るために明確に設計されていますが、剣は鋭いとはいえ、そうではありません。したがって、ノミは彫ろうとしているものからダメージを受けませんが、剣はほとんどの場合ダメージを受けるでしょう。

## 幸運ポイント

幸運ポイントは、英雄を一般人から区別するのに役立ちます。彼らは、失敗を成功に変え、さらには死を欺く可能性のあるキャラクターの能力を表します。

幸運ポイントはプレイ中に使用でき、次のセッションの開始時には通常の値に補充されます。

#### 幸運ポイントの使用

幸運ポイントは様々な方法で使用できます。特定のアクションをサポートするために使用できる幸運ポイントは1つだけです。経験ロールや訓練ロールに幸運ポイントを費やすことはできません。以下の各オプションは1つの幸運ポイントを消費します。

#### 運命を欺く

キャラクターは幸運ポイントを使って、自分が行う任意のダイスロール（スキルロール、ダメージロール、または何らかの効果を持つもの）を再ロールまたは交換（例えば75は57になる）することができます。キャラクターは、自分に対する敵の攻撃ロールやダメージロールを再ロールさせることさえできます。

#### 絶望的な努力

キャラクターが戦闘中にアクションポイントを使い果たし、例えば悲惨な最期を避けるために最後の絶望的なエネルギーの爆発を見つける必要がある場合、幸運ポイントを消費して追加のアクションポイントを1つ得ることができます。

#### ダメージの軽減

致命傷を受けたキャラクターは、幸運ポイントを消費して負傷を重傷に格下げすることができます。これにより、致命傷を与えるのに必要なヒットポイントより1点少ないダメージに軽減されます。
