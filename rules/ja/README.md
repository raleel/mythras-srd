# Mythras Imperative システムリソース文書

---
## 導入

_Mythras Imperative_ は、_Mythras_ ルールセットのオープンなORCライセンス版です。この文書は、キャラクター作成、技能、特定の典型的な状況に関するルール、戦闘、そして魔法を含む、包括的ではあるものの網羅的ではないルールセットを提供します。これは、プレイヤーとゲームマスターの両方がゲームを始めるのに十分であり、さらなる発展のための堅固な基盤を形成します。_Mythras Imperative_ は、Mythras のコアルールおよび他のORCライセンス準拠のパーセンタイルゲームシステムと互換性があります。その姉妹編である _Classic Fantasy Imperative_ も完全に互換性があり、ORCの下でリリースされているため、これら2つのImperativeは容易に組み合わせることができます。

英語版以外の翻訳はAIによって行われます。コミュニティからのご意見を歓迎します！

## ORCライセンス

_Mythras Imperative_ はORCライセンスを使用して発行されています。ORCライセンスは、ゲームクリエイターがゲームのメカニクスをオープンに共有し、後続の利用者がこれらのゲームメカニクスを自由に利用、修正、再共有できるようにする方法を提供します。

ORCの下でオープンに共有されるものは _**ライセンス対象素材**_ として知られており、ステータスブロック、ゲームルール、キャラクター属性、ゲームをプレイする上で不可欠なメソッドやシステムといったゲームの広範な機能要素、およびライセンサーが明示的に共有したいその他のすべてを含みます。このライセンスは広範であり、印刷された書籍、ビデオゲーム、ポッドキャスト、AI、または将来存在する、あるいは作成される可能性のあるその他のテクノロジーと関連付けて、ライセンス対象素材をグローバルに利用することを許可します。『Mythras Imperative』に見られるコンテンツの大部分は、ライセンス対象素材を構成します。

ORCの下で共有_されない_ものは _**予約済み素材**_ として知られており、商標、ワールドの伝承、ストーリーアーク、特徴的なキャラクター、およびビジュアルアートを含みます。予約済み素材については、以下のORC告知に記載しています。

ORCライセンスの下では、公開された作品に適切な帰属表示を行う限り、ライセンス対象素材を自由に利用、共有、適合させ、それに基づいて構築することができます。この表示は、先行するライセンサーとその作品への貢献をクレジットするとともに、後続の利用者が、その表示がある作品の著者に対してクレジットを付与する簡単な方法を提供します。ORCの下でライセンスされた素材を使用することにより、あなたは自動的に、公開する作品内のゲームメカニクスを同じ条件でライセンス供与することに同意します。あなたが独自の予約済み素材を開発した場合、それはあなたの独占的財産となります。

## d100初心者の方へ

_Mythras Imperative_ はd100またはパーセンタイルシステムです。パーセンタイルダイス、つまり1d100は、技能の使用、戦闘、魔法の詠唱など、主要な行動の解決に使用されます。これはロールアンダーシステムであり、目標値がパーセンテージ（例：Athletics 65%）で表され、ゲームマスターとプレイヤーの両方が1d100を使用して、この目標値以下を振ることで成功を達成します。必要であれば1d100をd20に簡単に置き換えることができ、すべてのパーセンテージを5で割って（切り上げ）d20の目標値を得ることができます。先の例では、Athletics 65%はAthletics 13になります。

しかし、1d100はゲーム全体を支える多くの柔軟性とニュアンスを提供するため、まずは1d100のアプローチを試すことをお勧めします。何かより_少ない_数を振らなければならないことは直感に反するように思えるかもしれませんが（多くのゲームでは通常、より高い方が良い！）、目標値がパーセンテージ範囲で示されているため、成功のチャンスを簡単かつ単純に確認でき、対抗ロール、クリティカル、ファンブル、そしてこれらのルールで後で説明されるいくつかの「ダイス技」を可能にします。パーセンタイルロールプレイングは、ロールプレイングの初期の頃から存在しており、柔軟で直感的で、プレイ中に背景に溶け込みがちな、実績のある中核メカニクスを持つ確立されたゲームシステムです。

## 簡略化

_**Mythras**_ を長年所有している方は、一部で _Mythras Imperative_ のルールが簡略化されていることに気づかれるでしょう。これは、ゲームマスターが最小限の手間で新しいプレイヤーにゲームをより迅速に紹介できるようにするために行われました。

---
## 数値と結果の切り上げ

場合によっては、数値を割る必要があります。通常は、技能の評価（クリティカル成功を決定する場合など、技能値の1/10）です。割り算の結果が端数を生じる場合は、常に切り上げて整数にします。例えば、63%の1/10は6.3ですが、これは7に切り上げられます。

## 使用するダイス

_Mythras Imperative_ は、標準的な多面体ダイスセット：d4、d6、d8、d10、d12、d20、およびd100を使用します。加えて、以下のダイスが求められます：

d2：任意のダイスを振る。奇数なら1、偶数なら2とする。またはコインを投げる。

d3：6面体ダイスを振る。1-2 = 1、3-4 = 2、5-6 = 3
