# पूर्वतयारी

1. [मार्कडाउन भाषेचे](https://www.markdownguide.org/basic-syntax/) मूलभूत ज्ञान असणे.

2. तुमच्याकडे github खाते नसल्यास, तुम्हाला एक विनामूल्य [github](https://github.com/) खाते तयार करावे लागेल. तुमच्या RPG चे नाव वापरण्याची शिफारस केली जाते कारण हे तुमच्या SRD मध्ये प्रवेश करण्यासाठी लोक वापरणार असलेल्या URL चा भाग बनेल.

3. तुम्ही git चे ज्ञान नसतानाही ही संपूर्ण प्रक्रिया ब्राउझरद्वारे करू शकता. तरीही, github डेस्कटॉप शिकल्याने तुम्हाला कोणतेही बदल जिवंत जगाला दिसण्यापूर्वी स्थानिक पातळीवर तपासण्याची अनुमती मिळेल.

> तुमच्याकडे डोमेन नाव नोंदणीकृत असल्यास, मी जसे https://srd.7thextinctionrpg.com केले आहे, त्याप्रमाणे तुम्ही या प्रक्रियेचा वापर करून SRD तयार करू शकता.

# प्रक्रिया

## Github वर रिपॉझिटरी तयार करणे

1. Github मध्ये लॉग इन करा.

2. सार्वजनिक टेम्प्लेट https://github.com/7thExtinctionrpg/RPG_SRD_Template वर नेव्हिगेट करा.

3. "Use this template" वर क्लिक करा.

![Step3](/_media/Step3.png)

4. "Create a new repository" निवडा.

5. रिपॉझिटरीचे नाव टाका.

![Step5](/_media/Step5.png)

6. 'Private' निवडा.

7. 'Create repository' निवडा.

टेम्प्लेट कॉपी होण्यासाठी 5-30 सेकंद लागतील, त्यानंतर तुमच्याकडे स्वतःची कॉपी असेल.

8. तुमच्याकडे ".nojekyll" ही फाइल असल्याची खात्री करा. ही एक रिकामी फाइल आहे जी Github वर वेबसाइट होस्ट करताना आवश्यक असते, अन्यथा SRD योग्यरित्या प्रदर्शित होणार नाही. *(हे GitHub Pages ला प्रकाशित फाइल्स Jekyll द्वारे चालवू नका असे सांगते).*


## तुमची सामग्री जोडणे

### *.md फाइल्स

.md फाइल्स म्हणजे मार्कडाउन फाइल्स. मार्कडाउन ही एक हलकी मार्कर भाषा आहे जी तुम्ही प्लेनटेक्स्ट मजकूर दस्तऐवजांमध्ये स्वरूपण घटक जोडण्यासाठी वापरू शकता. तुमच्या सामग्रीच्या आकारानुसार, तुम्ही तुमच्या गेमची सर्व माहिती एकाच *.md फाइलमध्ये ठेवू शकता, किंवा मी [7thextinctionrpg.com](https://srd.7thextinctionrpg.com) वर केले आहे त्याप्रमाणे, प्रत्येक अध्यायासाठी स्वतंत्र फाइल्स तयार करू शकता. "sample.md" काही स्वरूपण उदाहरणे प्रदान करते.

मार्कडाउन भाषा शिकायला खूप सोपी आहे, सिंटॅक्सच्या पूर्ण सूचीसाठी वर नमूद केलेल्या पूर्वतयारीतील गुणधर्म 2 अंतर्गत दिलेली लिंक वापरा.

### _Sidebar.md

हा तुमचा मेनू आहे, वेबसाइटच्या डाव्या बाजूला असलेले नेव्हिगेशन पेन. एकदा तुम्ही तुमच्या रिपॉझिटरीमध्ये मार्कडाउन फाइल जोडल्यानंतर, तिची सामग्री समाविष्ट करण्यासाठी तुम्हाला ती फाइल येथे जोडणे आवश्यक आहे. सध्याचे स्वरूपण फॉलो करा.


## रिपॉझिटरीला वेबसाइटमध्ये रूपांतरित करण्यासाठी Github Pages सक्षम करा

1. तुमच्या रिपॉझिटरीमध्ये 'Settings' वर क्लिक करा.

![Website-Step1](/_media/Website-Step1.png)

2. डाव्या हाताच्या मेनूमधून 'Pages' निवडा.

![Website-Step2](/_media/Website-Step2.png)

3. जर तुम्ही रिपॉझिटरी 'public' केली नसेल, तर तुमच्याकडे दोन पर्याय आहेत:
    a. रिपॉझिटरी सार्वजनिक करा, ज्यामुळे तुम्हाला Github pages वापरून तुमची SRD वेबसाइट विनामूल्य होस्ट करण्याची अनुमती मिळेल.
    b. तुमचे Github खाते अपग्रेड करा (हे लिहित असताना याची किंमत दरमहा $4 किंवा वार्षिक $48 आहे).

4. 'Branch' अंतर्गत 'main' निवडा आणि सेव्ह करा.

5. 10-20 सेकंद द्या आणि नंतर पृष्ठ रीफ्रेश करा.

6. तुम्हाला आता तुमची URL दिसेल जसे की https://coweater.github.io/mysrd/

![Website-Step6](/_media/Website-Step6.png)

## तुमच्या SRD मध्ये विनामूल्य चिन्ह (icons) प्रदर्शित करण्यासाठी Fontawesome आणि/किंवा Material Icons प्लगइन वापरणे

### Fontawesome

विनामूल्य चिन्ह आणि ॲनिमेटेड चिन्ह मिळवण्यासाठी Fontawesome वापरा. सबस्क्राईब केलेले वापरकर्ते प्रो आयकॉन वापरू शकतात.

1. [Fontawesome](https://fontawesome.com/icons) वर चिन्ह शोधा.
2. चिन्ह निवडा.
3. पॉपअप विंडोमध्ये, HTML टॅब अंतर्गत " " मधील सर्वकाही कॉपी करा.

```html
<i class="fa-solid fa-cloud"></i>
```

4. कॉपी केलेला मजकूर `<i>` `</i>` मध्ये कोणत्याही जागेविना ठेवा.

```html
<span class="fas fa-hand-rock"></span>
```

### Material Icons

अधिक विनामूल्य चिन्हांसाठी Material Icons वापरा.

1. [Material Icons](https://fonts.google.com/icons) वर चिन्ह शोधा.
2. वेब स्पॅन लिंक थेट तुमच्या मार्कडाउन फाइल्समध्ये कॉपी आणि इन्सर्ट करा.

```html
<span class="material-symbols-outlined">
  cloud
</span>
```
