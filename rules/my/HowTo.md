# ကြိုတင်လိုအပ်ချက်များ

1. [မာခ်ဒေါင်းဘာသာစကား](https://www.markdownguide.org/basic-syntax/) ကို အခြေခံနားလည်မှုရှိခြင်း။

2. သင့်တွင် [github](https://github.com/) အကောင့်မရှိသေးပါက အခမဲ့အကောင့်တစ်ခု ဖန်တီးရန် လိုအပ်မည်ဖြစ်သည်။ သင့် RPG ၏အမည်ကို အသုံးပြုရန် အကြံပြုလိုပါသည်၊ အဘယ်ကြောင့်ဆိုသော် ၎င်းသည် လူများသင့် SRD ကို ဝင်ရောက်ကြည့်ရှုရန် အသုံးပြုမည့် URL ၏ အစိတ်အပိုင်းတစ်ခု ဖြစ်လာမည်ဖြစ်သောကြောင့် ဖြစ်သည်။

3. သင်သည် git ကိုမသိဘဲ ဘရောက်ဆာမှတစ်ဆင့် ဤလုပ်ငန်းစဉ်တစ်ခုလုံးကို လုပ်ဆောင်နိုင်သည်။ သို့သော် github desktop ကို လေ့လာခြင်းက သင့်အား မည်သည့်ပြောင်းလဲမှုများကိုမဆို ကမ္ဘာသို့ တိုက်ရိုက်မဖြန့်ချိမီ ဒေသတွင်း၌ စမ်းသပ်ခွင့်ပြုမည်ဖြစ်သည်။

> သင့်တွင် မှတ်ပုံတင်ထားသော ဒိုမိန်းအမည်ရှိပါက၊ ကျွန်ုပ်ပြုလုပ်ထားသည့် https://srd.7thextinctionrpg.com ကဲ့သို့ SRD တစ်ခုကို ဖန်တီးရန် ဤလုပ်ငန်းစဉ်ဖြင့် အသုံးပြုနိုင်ပါသည်။

# လုပ်ဆောင်ရမည့်အဆင့်များ

## သင့်အကြောင်းအရာများ ထည့်သွင်းခြင်း

### *.md Files

.md ဖိုင်များသည် markdown ဖိုင်များဖြစ်ပြီး၊ Markdown သည် စာသားဖိုင်များတွင် formatting elements များထည့်သွင်းရန် သင်အသုံးပြုနိုင်သော ပေါ့ပါးသည့် markup ဘာသာစကားတစ်ခုဖြစ်သည်။ သင့်အကြောင်းအရာ၏ အရွယ်အစားပေါ်မူတည်၍ သင်၏ဂိမ်းအချက်အလက်အားလုံးကို *.md ဖိုင်တစ်ခုတည်းတွင် ထည့်သွင်းနိုင်သည် သို့မဟုတ် ကျွန်ုပ် [7thextinctionrpg.com](https://srd.7thextinctionrpg.com) တွင် ပြုလုပ်ထားသကဲ့သို့ အခန်းတစ်ခုစီအတွက် သီးခြားဖိုင်များ ဖန်တီးနိုင်သည်။ "sample.md" သည် formatting ဥပမာအချို့ကို ပေးထားသည်။

Markdown ဘာသာစကားသည် လေ့လာရန် အလွန်လွယ်ကူပါသည်၊ syntax စာရင်းအပြည့်အစုံအတွက် အပေါ်ရှိ ကြိုတင်လိုအပ်ချက်များ၏ အမှတ် ၂ တွင် ပေးထားသည့် link ကို အသုံးပြုပါ။

### _Sidebar.md

၎င်းသည် သင့်ဝက်ဘ်ဆိုက်၏ ဘယ်ဘက်ခြမ်းရှိ သင့်မီနူး၊ navigation pane ဖြစ်သည်။ သင့် repository သို့ markdown ဖိုင်တစ်ခုထည့်ပြီးသည်နှင့် ၎င်း၏အကြောင်းအရာများ ပါဝင်လာစေရန် ဤနေရာတွင် ဖိုင်ကို ထည့်သွင်းရန် လိုအပ်ပါသည်။ ရှိပြီးသား formatting ကို လိုက်နာပါ။

## ဖောင့်များ ပြောင်းလဲခြင်း

1. /_assets/style.css ဖိုင်ကို ပြင်ဆင်ပါ။

2. 'body section' အောက်တွင် siteFont & headingFont ကို တွေ့ရမည်။

3. ၎င်းတို့ကို သင်အသုံးပြုလိုသော ဖောင့်များသို့ ပြောင်းပါ။

## Fontawesome နှင့်/သို့မဟုတ် Material Icons plugin ကို အသုံးပြု၍ သင့် SRD တွင် အခမဲ့ icon များပြသခြင်း

### Fontawesome

အခမဲ့ icon များနှင့် animated icon များရရှိရန် Fontawesome ကို အသုံးပြုပါ။ Subcribed လုပ်ထားသော အသုံးပြုသူများသည် Pro Icons ကို အသုံးပြုနိုင်သည်။

1. [Fontawesome](https://fontawesome.com/icons) တွင် icon ကို ရှာပါ။
2. icon ကို ရွေးပါ။
3. popup window တွင်၊ HTML tab အောက်ရှိ " " ကြားရှိ အရာအားလုံးကို ကူးယူပါ။

```EXAMPLE
<i class="fa-solid fa-dice-d20"></i>
```

4. ကူးယူထားသော စာသားကို `:` နေရာ၌ ကပ်ထည့်ပါ (ကြားထဲတွင် နေရာလွတ်မထားရ)။

```EXAMPLE
fa-solid fa-dice-d20
```

### Material Icons

ပိုမိုများပြားသော အခမဲ့ icon များအတွက် Material Icons ကို အသုံးပြုပါ။

1. [Material Icons](https://fonts.google.com/icons) တွင် icon များကို ရှာပါ။
2. web span link ကို တိုက်ရိုက်ကူးယူပြီး သင့် markdown ဖိုင်များထဲသို့ ထည့်သွင်းပါ။

```markup
<span class="material-icons">face</span>
```
