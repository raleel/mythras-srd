# Предварителни изисквания

1. Основно разбиране на [езика Markdown](https://www.markdownguide.org/basic-syntax/).

2. Ще трябва да си създадете безплатен [github](https://github.com/) акаунт, ако нямате такъв. Препоръчително е да използвате името на вашето RPG, тъй като то ще стане част от URL адреса, който хората ще използват за достъп до вашия SRD.

3. Можете да извършите целия този процес през браузъра, без да знаете git. Въпреки това, научаването на github desktop ще ви позволи да тествате всякакви промени локално, преди да ги разположите на живо, за да ги види светът.

> Ако имате регистрирано домейн име, можете да го използвате с този процес, за да създадете SRD като моя https://srd.7thextinctionrpg.com

# Процесът

## Създаване на хранилището в Github

1. Влезте в Github

2. Навигирайте до публичния шаблон https://github.com/7thExtinctionrpg/RPG_SRD_Template

3. Кликнете върху "Use this template" (Използвайте този шаблон)

![Step3](/_media/Step3.png)

4. Изберете "Create a new repository" (Създайте ново хранилище)

5. Въведете име на хранилище

![Step5](/_media/Step5.png)

6. Изберете "Private" (Частно)

7. Изберете "Create repository" (Създайте хранилище)

Ще отнеме 5-30 секунди за копиране на шаблона, след което ще имате свое собствено копие.

8. Уверете се, че имате файла ".nojekyll", това е празен файл, който е необходим при хостване на уебсайт в Github, в противен случай SRD няма да се показва правилно. *(Казва на GitHub Pages да не обработва публикуваните файлове през Jekyll).*


## Добавяне на вашето съдържание

### *.md Файлове

.md файловете са Markdown файлове. Markdown е олекотен език за маркиране, който можете да използвате, за да добавяте елементи за форматиране към текстови документи с обикновен текст. В зависимост от размера на вашето съдържание, можете или да поставите цялата информация за вашата игра в един *.md файл, или както аз съм направил на [7thextinctionrpg.com](https://srd.7thextinctionrpg.com), да създадете отделни файлове за всяка глава. "sample.md" предоставя някои примери за форматиране.

Езикът Markdown е много лесен за научаване, използвайте линка, предоставен в точка 2 под "Предварителни изисквания" по-горе, за пълен списък със синтаксис.

### _Sidebar.md

Това е вашето меню, навигационен прозорец отляво на уебсайта. След като сте добавили Markdown файл към вашето хранилище, трябва да добавите файла тук, така че съдържанието му да бъде включено. Следвайте съществуващото форматиране.


## Активиране на Github Pages за превръщане на хранилището в уебсайт

1. Във вашето хранилище кликнете върху "Settings" (Настройки)

![Website-Step1](/_media/Website-Step1.png)

2. Изберете "Pages" (Страници) от менюто отляво

![Website-Step2](/_media/Website-Step2.png)

3. Ако не сте направили хранилището "public" (публично), имате два избора
    а. Направете хранилището публично, което след това ще ви позволи да използвате Github Pages за хостване на вашия SRD уебсайт безплатно
    б. Надстройте вашия Github акаунт (към момента на писане това струва $4 на месец или $48 годишна такса)

4. Под "Branch" (Клон) изберете "main" и кликнете върху "Save" (Запазване)

5. Изчакайте 10-20 секунди, след което обновете страницата

6. Сега ще видите вашия URL адрес, като например https://coweater.github.io/mysrd/

![Website-Step6](/_media/Website-Step6.png)

## Използване на плъгина Fontawesome и/или Material Icons за показване на безплатни икони във вашия SRD

### Fontawesome

Използвайте Fontawesome, за да получите безплатни икони и анимирани икони. Абонираните потребители могат да използват Pro икони.

1. Намерете иконата на [Fontawesome](https://fontawesome.com/icons)
2. Изберете иконата
3. В изскачащия прозорец, под раздела HTML, копирайте всичко между " "

```html
<i class="fa-solid fa-hat-wizard"></i>
```

4. Поставете копирания текст между ` `: без интервали

```
<i class="fa-solid fa-hat-wizard"></i>
```

### Material Icons

Използвайте Material Icons за още безплатни икони

1. Намерете иконите на [Material Icons](https://fonts.google.com/icons)
2. Копирайте и поставете web span линка директно във вашите Markdown файлове

```html
<span class="material-symbols-outlined">
star
</span>
```
