# پێش مەرجەکان

1. تێگەیشتنێکی بنەڕەتی لە [زمانی مارکداون](https://www.markdownguide.org/basic-syntax/).

2. پێویستە هەژمارێکی [github](https://github.com/) بە خۆڕایی دروست بکەیت ئەگەر نەت بوو. پێشنیار دەکرێت ناوی RPGەکەت بەکاربهێنیت چونکە ئەمە دەبێتە بەشێک لەو URLـەی کە خەڵک بۆ گەیشتن بە SRDـەکەت بەکاریدەهێنن.

3. دەتوانیت ئەم هەموو پرۆسەیە لە ڕێگەی وێبگەڕەوە ئەنجام بدەیت بەبێ ئەوەی گیت بزانیت. هەرچەندە فێربوونی github desktop ڕێگەت پێدەدات کە هەر گۆڕانکارییەک بە شێوەیەکی خۆجێی تاقیبکەیتەوە پێش ئەوەی بۆ جیهان بڵاوی بکەیتەوە.

> ئەگەر ناوی دۆمەینێکت تۆمار کردووە، دەتوانیت ئەمە لەگەڵ ئەم پرۆسەیەدا بەکاربهێنیت بۆ دروستکردنی SRD وەک ئەوەی من هەمە https://srd.7thextinctionrpg.com

## زیادکردنی ناوەڕۆکەکەت

### فایلەکانی *.md

فایلەکانی .md فایلەکانی مارکداونن، مارکداون زمانێکی مارکئەپ سووکە کە دەتوانیت بەکاریبهێنیت بۆ زیادکردنی توخمەکانی فۆرماتکردن بۆ دۆکومێنتەکانی دەقی سادە. بەپێی قەبارەی ناوەڕۆکەکەت دەتوانیت هەموو زانیارییەکانی یارییەکەت لە یەک فایلێکی *.md دابنێیت، یان وەک ئەوەی من لە [7thextinctionrpg.com](https://srd.7thextinctionrpg.com) کردوومە، فایلە جیاوازەکان بۆ هەر بەشێک دروست بکەیت. "sample.md" چەند نموونەیەکی فۆرماتکردن نیشان دەدات.

فێربوونی زمانی مارکداون زۆر ئاسانە، بەستەرەکەی خاڵی 2 لەژێر پێش مەرجەکاندا بەکاربهێنە بۆ لیستێکی تەواوی رێزمان.

### _Sidebar.md

ئەمە مێنیوەکەتە، پانێڵی گەشتیارییە لە لای چەپی وێبسایتەکە. کاتێک فایلێکی مارکداونت بۆ کۆگاکەت زیاد کرد، پێویستە فایلەکە لێرە زیاد بکەیت تاوەکو ناوەڕۆکەکەی بخرێتە ناوی. شوێن فۆرماتکردنی ئێستا بکەوە.

## چالاککردنی Github Pages بۆ گۆڕینی کۆگاکە بۆ وێبسایتێک

1. لە کۆگاکەت کرتە لەسەر 'Settings' بکە

![Website-Step1](/_media/Website-Step1.png)

2. 'Pages' لە لیستی لای چەپدا هەڵبژێرە

![Website-Step2](/_media/Website-Step2.png)

3. ئەگەر کۆگاکەت نەکردووەتە 'public'، دوو هەڵبژاردەت هەیە

5. 10-20 چرکە پێ بدە پاشان پەیجەکە نوێ بکەرەوە

## گۆڕینی فۆنتەکان

1. فایلی /_assets/style.css دەستکاری بکە

2. لەژێر بەشی 'body'دا siteFont و headingFont دەدۆزیتەوە

3. ئەمانە بگۆڕە بۆ ئەو فۆنتانەی کە دەتەوێت بەکاریان بهێنیت

## بەکارهێنانی Fontawesome و/یان Material Icons plugin بۆ نمایشکردنی ئایکۆنە خۆڕاییەکان لە SRDەکەتدا

### Fontawesome

Fontawesome بەکاربهێنە بۆ بەدەستخستنی ئایکۆنە خۆڕاییەکان و ئایکۆنە جووڵاوەکان. بەکارهێنەرانی بەشداربوو دەتوانن Pro Icons بەکاربهێنن.

1. ئایکۆنەکە لە [Fontawesome](https://fontawesome.com/icons) بدۆزەرەوە
2. ئایکۆنەکە هەڵبژێرە
3. لە پەنجەرەی پاپئەپدا، لەژێر تابی HTMLدا هەموو شتێک لەنێوان " " کۆپی بکە

```EXAMPLE
<i class="fa-solid fa-dice-d20"></i>
```

4. دەقە کۆپیکراوەکە لەنێوان `:` دابنێ بەبێ بۆشایی

```EXAMPLE
fa-solid fa-dice-d20
```

### Material Icons

Material Icons بەکاربهێنە بۆ ئایکۆنەکانی خۆڕایی زیاتر

1. ئایکۆنەکان لە [Material Icons](https://fonts.google.com/icons) بدۆزەرەوە
2. بەستەری web spanەکە ڕاستەوخۆ کۆپی و بچووک بکە بۆ فایلەکانی مارکداونەکەت

```markup
<span class="material-icons">face</span>
```
