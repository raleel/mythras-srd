# Предварительные условия

1. Базовое понимание [языка разметки markdown](https://www.markdownguide.org/basic-syntax/).

2. Вам потребуется создать бесплатную учетную запись [github](https://github.com/), если у вас ее нет. Рекомендуется использовать название вашей НРИ, так как это станет частью URL-адреса, который люди будут использовать для доступа к вашему SRD.

3. Вы можете выполнить весь этот процесс через браузер, не зная git. Хотя изучение github desktop позволит вам тестировать любые изменения локально, прежде чем разворачивать их для всеобщего обозрения.

> Если у вас зарегистрировано доменное имя, вы можете использовать его в этом процессе для создания SRD, как у меня: https://srd.7thextinctionrpg.com

# Процесс

## Создание репозитория на Github

1. Войдите в Github

2. Перейдите к общедоступному шаблону https://github.com/7thExtinctionrpg/RPG_SRD_Template

3. Нажмите "Use this template" (Использовать этот шаблон)

![Step3](/_media/Step3.png)

4. Выберите "Create a new repository" (Создать новый репозиторий)

5. Введите имя репозитория

![Step5](/_media/Step5.png)

6. Выберите 'Private' (Приватный)

7. Выберите 'Create repository' (Создать репозиторий)

Копирование шаблона займет 5-30 секунд, после чего у вас будет собственная копия.

8. Убедитесь, что у вас есть файл ".nojekyll". Это пустой файл, который необходим при хостинге веб-сайта на Github, иначе SRD не будет отображаться должным образом. *(Указывает GitHub Pages не обрабатывать опубликованные файлы через Jekyll).*

## Добавление вашего контента

### Файлы *.md

Файлы .md — это файлы markdown. Markdown — это легковесный язык разметки, который вы можете использовать для добавления элементов форматирования в простые текстовые документы. В зависимости от объема вашего контента вы можете либо поместить всю информацию о вашей игре в один файл *.md, либо, как я сделал на [7thextinctionrpg.com](https://srd.7thextinctionrpg.com), создать отдельные файлы для каждой главы. "sample.md" содержит несколько примеров форматирования.

Язык Markdown очень прост в изучении, используйте ссылку, приведенную в пункте 2 раздела "Предварительные условия" выше, для полного списка синтаксиса.

### _Sidebar.md

Это ваше меню, панель навигации в левой части веб-сайта. После того как вы добавили файл markdown в свой репозиторий, вам нужно добавить файл сюда, чтобы его содержимое было включено. Следуйте существующему форматированию.

## Включение Github Pages для превращения репозитория в веб-сайт

1. В вашем репозитории нажмите 'Settings' (Настройки)

![Website-Step1](/_media/Website-Step1.png)

2. Выберите 'Pages' (Страницы) в меню слева

![Website-Step2](/_media/Website-Step2.png)

3. Если вы не сделали репозиторий 'public' (общедоступным), у вас есть два варианта:
    a. Сделать репозиторий общедоступным, что позволит вам использовать Github Pages для бесплатного хостинга вашего SRD-сайта.
    b. Обновить свою учетную запись Github (на момент написания это стоит 4 доллара в месяц или 48 долларов в год).

4. В разделе 'Branch' (Ветка) выберите 'main' и нажмите 'Save' (Сохранить).

5. Подождите 10-20 секунд, затем обновите страницу.

6. Теперь вы увидите свой URL, например https://coweater.github.io/mysrd/

![Website-Step6](/_media/Website-Step6.png)

## Изменение шрифтов

1. Отредактируйте файл /_assets/style.css

2. В разделе 'body section' вы найдете siteFont и headingFont

3. Измените их на шрифты, которые вы хотите использовать.

## Использование плагина Fontawesome и/или Material Icons для отображения бесплатных иконок в вашем SRD

### Fontawesome

Используйте Fontawesome для получения бесплатных и анимированных иконок. Подписчики могут использовать Pro Icons.

1. Найдите иконку на [Fontawesome](https://fontawesome.com/icons)
2. Выберите иконку
3. Во всплывающем окне, на вкладке HTML, скопируйте всё между " "

```html
<i class="fa-solid fa-dice-d20"></i>
```

4. Вставьте скопированный текст между `<span class="icon"> </span>` без пробелов

```html
<span class="icon"><i class="fa-solid fa-dice-d20"></i></span>
```

### Material Icons

Используйте Material Icons для большего количества бесплатных иконок

1. Найдите иконки на [Material Icons](https://fonts.google.com/icons)
2. Скопируйте и вставьте веб-ссылку span непосредственно в ваши файлы markdown

```html
<span class="material-symbols-outlined">casino</span>
```
