## உங்கள் உள்ளடக்கத்தைச் சேர்த்தல்

### *.md கோப்புகள்

.md கோப்புகள் மார்க்டவுன் கோப்புகள் ஆகும், மார்க்டவுன் என்பது ஒரு இலகுரக மார்க்அப் மொழி ஆகும், இது வெற்று உரை ஆவணங்களுக்கு வடிவமைக்கும் கூறுகளைச் சேர்க்க பயன்படுத்தலாம். உங்கள் உள்ளடக்கத்தின் அளவைப் பொறுத்து, உங்கள் விளையாட்டின் அனைத்து தகவல்களையும் ஒரு *.md கோப்பில் வைக்கலாம், அல்லது [7thextinctionrpg.com](https://srd.7thextinctionrpg.com) இல் நான் செய்தது போல் ஒவ்வொரு அத்தியாயத்திற்கும் தனித்தனி கோப்புகளை உருவாக்கலாம். "sample.md" சில வடிவமைக்கும் எடுத்துக்காட்டுகளை வழங்குகிறது.

மார்க்டவுன் மொழி கற்றுக்கொள்ள மிகவும் எளிதானது, முழுமையான தொடரியல் பட்டியலுக்கு மேலே உள்ள முன்நிபந்தனைகளின் புள்ளி 2-ல் கொடுக்கப்பட்டுள்ள இணைப்பைப் பயன்படுத்தவும்.

## உங்கள் SRD-யில் இலவச ஐகான்களைக் காட்ட Fontawesome மற்றும்/அல்லது Material Icons ப்ளக்இன்னைப் பயன்படுத்துதல்

### ஃபான்ட்ஆசம்

இலவச ஐகான்கள் மற்றும் அனிமேட்டட் ஐகான்களைப் பெற ஃபான்ட்ஆசத்தைப் பயன்படுத்தவும். சந்தாதாரர்கள் புரோ ஐகான்களைப் பயன்படுத்தலாம்.

1.  [ஃபான்ட்ஆசத்தில்](https://fontawesome.com/icons) ஐகானைக் கண்டறியவும்
2.  ஐகானைத் தேர்ந்தெடுக்கவும்
3.  பாப்-அப் சாளரத்தில், HTML தாவலின் கீழ் " " க்கு இடையில் உள்ள அனைத்தையும் நகலெடுக்கவும்

```EXAMPLE
<i class="fa-solid fa-dice-d20"></i>
```

4.  நகலெடுக்கப்பட்ட உரையை `:` இடையில் இடைவெளிகள் இல்லாமல் வைக்கவும்

```EXAMPLE
fa-solid fa-dice-d20
```

### மெட்டீரியல் ஐகான்ஸ்

மேலும் இலவச ஐகான்களுக்கு மெட்டீரியல் ஐகான்ஸைப் பயன்படுத்தவும்

1.  [மெட்டீரியல் ஐகான்ஸில்](https://fonts.google.com/icons) ஐகான்களைக் கண்டறியவும்
2.  வெப் ஸ்பான் இணைப்பை நேரடியாக உங்கள் மார்க்டவுன் கோப்புகளில் நகலெடுத்து செருகவும்

```markup
<span class="material-icons">face</span>
```
