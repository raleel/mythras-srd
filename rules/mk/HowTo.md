# Предуслови

1. Основно разбирање на [markdown јазикот](https://www.markdownguide.org/basic-syntax/).

2. Ќе треба да креирате бесплатна [github](https://github.com/) сметка доколку немате. Препорачливо е да го користите името на вашето RPG бидејќи тоа ќе стане дел од URL-то што луѓето ќе го користат за пристап до вашиот SRD.

3. Целиот овој процес можете да го направите преку прелистувачот без да го знаете git. Иако учењето на github desktop ќе ви овозможи да ги тестирате сите промени локално пред да ги објавите во живо за светот да ги види.

> Ако имате регистрирано име на домен, можете да го користите со овој процес за да креирате SRD како што јас имам https://srd.7thextinctionrpg.com

# Процесот

## Креирање на репозиториумот на Github

1. Најавете се на Github

2. Одете до јавниот шаблон https://github.com/7thExtinctionrpg/RPG_SRD_Template

3. Кликнете „Use this template“ (Користи го овој шаблон)

![Step3](/_media/Step3.png)

4. Изберете „Create a new repository“ (Креирај нов репозиториум)

5. Внесете име на репозиториумот

![Step5](/_media/Step5.png)

6. Изберете „Private“ (Приватен)

7. Изберете „Create repository“ (Креирај репозиториум)

Ќе бидат потребни 5-30 секунди за да се копира шаблонот, потоа ќе имате своја копија.

8. Осигурајте се дека ја имате датотеката ".nojekyll", ова е празна датотека која е потребна при хостирање на веб-страница на Github, во спротивно SRD нема правилно да се прикажува. *(Му кажува на GitHub Pages да не ги извршува објавените датотеки преку Jekyll).*

## Додавање на вашата содржина

### *.md Датотеки

.md датотеките се markdown датотеки, Markdown е лесен јазик за означување што можете да го користите за додавање елементи за форматирање на текстуални документи. Во зависност од големината на вашата содржина, можете да ги поставите сите информации за вашата игра во една *.md датотека, или како што јас имам на [7thextinctionrpg.com](https://srd.7thextinctionrpg.com) да креирате посебни датотеки за секое поглавје. "sample.md" дава некои примери за форматирање.

Markdown јазикот е многу лесен за учење, користете ја врската дадена на точка 2 под Предуслови погоре за целосен список на синтакса.

### _Sidebar.md

Ова е вашето мени, панел за навигација на левата страна на веб-страницата. Откако ќе додадете markdown датотека во вашиот репозиториум, треба да ја додадете датотеката овде за да биде вклучена нејзината содржина. Следете го постоечкиот формат.

## Овозможете Github Pages за да го претворите репозиториумот во веб-страница

1. Во вашиот репозиториум кликнете на „Settings“ (Поставки)

![Website-Step1](/_media/Website-Step1.png)

2. Изберете „Pages“ (Страници) од левото странично мени

![Website-Step2](/_media/Website-Step2.png)

3. Ако не сте го направиле репозиториумот „public“ (јавен), имате два избора
    а. Направете го репозиториумот јавен што потоа ќе ви овозможи да користите Github Pages за бесплатно хостирање на вашата SRD веб-страница
    б. Надградете ја вашата Github сметка (во моментот на пишување ова чини 4 долари месечно или 48 долари годишна такса)

4. Под „Branch“ (Грана) изберете „main“ и кликнете „save“ (зачувај)

5. Почекајте 10-20 секунди, а потоа освежете ја страницата

6. Сега ќе го видите вашето URL како на пр. https://coweater.github.io/mysrd/

![Website-Step6](/_media/Website-Step6.png)

## Менување фонтови

1. Уредете ја датотеката /_assets/style.css

2. Под делот „body section“ ќе ги најдете siteFont и headingFont

3. Променете ги овие на фонтовите што сакате да ги користите

## Користење на Fontawesome и/или Material Icons додаток за прикажување бесплатни икони во вашиот SRD

### Fontawesome

Користете Fontawesome за да добиете бесплатни икони и анимирани икони. Корисниците со претплата можат да користат Pro Icons

1. Најдете ја иконата на [Fontawesome](https://fontawesome.com/icons)
2. Изберете ја иконата
3. Во скокачкиот прозорец, под табулаторот HTML, копирајте сè помеѓу " "

```html
<i class="fa-solid fa-dice-d20"></i>
```

4. Поставете го копираниот текст помеѓу `` без празни места

```markdown
<i class="fa-solid fa-dice-d20"></i>
```

### Material Icons

Користете Material Icons за повеќе бесплатни икони

1. Најдете ги иконите на [Material Icons](https://fonts.google.com/icons)
2. Копирајте и вметнете ја веб-врската `<span>` директно во вашите markdown датотеки

```html
<span class="material-symbols-outlined">
auto_stories
</span>
```
