# Προαπαιτούμενα

1.  Βασική κατανόηση της [γλώσσας markdown](https://www.markdownguide.org/basic-syntax/).

2.  Θα χρειαστεί να δημιουργήσετε έναν δωρεάν λογαριασμό [github](https://github.com/) αν δεν έχετε. Συνιστάται να χρησιμοποιήσετε το όνομα του RPG σας, καθώς αυτό θα γίνει μέρος της διεύθυνσης URL που θα χρησιμοποιούν οι χρήστες για να έχουν πρόσβαση στο SRD σας.

3.  Μπορείτε να κάνετε όλη αυτή τη διαδικασία μέσω του προγράμματος περιήγησης χωρίς να γνωρίζετε το git. Ωστόσο, η εκμάθηση του github desktop θα σας επιτρέψει να δοκιμάσετε τυχόν αλλαγές τοπικά πριν τις αναπτύξετε ζωντανά για να τις δει ο κόσμος.

> Αν έχετε κατοχυρωμένο όνομα τομέα (domain name), μπορείτε να το χρησιμοποιήσετε με αυτή τη διαδικασία για να δημιουργήσετε ένα SRD όπως έχω εγώ https://srd.7thextinctionrpg.com

# Η Διαδικασία

## Δημιουργία του Repository στο Github

1.  Συνδεθείτε στο Github

2.  Μεταβείτε στο δημόσιο πρότυπο https://github.com/7thExtinctionrpg/RPG_SRD_Template

3.  Κάντε κλικ στο "Use this template"

![Βήμα 3](/_media/Step3.png)

4.  Επιλέξτε "Create a new repository"

5.  Εισάγετε ένα όνομα repository

![Βήμα 5](/_media/Step5.png)

6.  Επιλέξτε 'Private'

7.  Επιλέξτε 'Create repository'

Θα χρειαστούν 5-30 δευτερόλεπτα για να αντιγραφεί το πρότυπο, και μετά θα έχετε το δικό σας αντίγραφο.

8.  Βεβαιωθείτε ότι έχετε το αρχείο ".nojekyll" αυτό είναι ένα κενό αρχείο που χρειάζεται κατά τη φιλοξενία μιας ιστοσελίδας στο Github, διαφορετικά το SRD δεν θα εμφανίζεται σωστά. *(Λέει στις GitHub Pages να μην εκτελέσουν τα δημοσιευμένα αρχεία μέσω του Jekyll).*

## Προσθήκη Περιεχομένου

### Αρχεία *.md

Τα αρχεία .md είναι αρχεία markdown, το Markdown είναι μια ελαφριά γλώσσα σήμανσης που μπορείτε να χρησιμοποιήσετε για να προσθέσετε στοιχεία μορφοποίησης σε έγγραφα απλού κειμένου. Ανάλογα με το μέγεθος του περιεχομένου σας, μπορείτε είτε να τοποθετήσετε όλες τις πληροφορίες του παιχνιδιού σας σε ένα μόνο αρχείο *.md, είτε όπως έχω κάνει εγώ στο [7thextinctionrpg.com](https://srd.7thextinctionrpg.com) να δημιουργήσετε ξεχωριστά αρχεία για κάθε κεφάλαιο. Το "sample.md" παρέχει ορισμένα παραδείγματα μορφοποίησης.

Η γλώσσα Markdown είναι πολύ εύκολη στην εκμάθηση, χρησιμοποιήστε τον σύνδεσμο που παρέχεται στο σημείο 2 υπό τα Προαπαιτούμενα παραπάνω για μια πλήρη λίστα σύνταξης.

### _Sidebar.md

Αυτό είναι το μενού σας, το πλαίσιο πλοήγησης στην αριστερή πλευρά της ιστοσελίδας. Μόλις προσθέσετε ένα αρχείο markdown στο repository σας, πρέπει να προσθέσετε το αρχείο εδώ ώστε το περιεχόμενό του να συμπεριληφθεί. Ακολουθήστε την υπάρχουσα μορφοποίηση.

## Ενεργοποίηση Github Pages για τη μετατροπή του Repository σε Ιστοσελίδα

1.  Στο repository σας κάντε κλικ στο 'Settings'

![Ιστοσελίδα-Βήμα 1](/_media/Website-Step1.png)

2.  Επιλέξτε 'Pages' από το μενού στην αριστερή πλευρά

![Ιστοσελίδα-Βήμα 2](/_media/Website-Step2.png)

3.  Αν δεν έχετε κάνει το repository 'public', έχετε δύο επιλογές
    α. Κάντε το repository δημόσιο, το οποίο θα σας επιτρέψει στη συνέχεια να χρησιμοποιήσετε τις Github pages για να φιλοξενήσετε την ιστοσελίδα του SRD σας δωρεάν
    β. Αναβαθμίστε τον λογαριασμό σας στο Github (τη στιγμή που γράφεται αυτό κοστίζει 4$ το μήνα ή 48$ ετήσια συνδρομή)

4.  Κάτω από το 'Branch' επιλέξτε 'main' και κάντε κλικ στην αποθήκευση

5.  Δώστε του 10-20 δευτερόλεπτα και στη συνέχεια ανανεώστε τη σελίδα

6.  Τώρα θα δείτε την URL σας, όπως https://coweater.github.io/mysrd/

![Ιστοσελίδα-Βήμα 6](/_media/Website-Step6.png)

## Χρήση του plugin Fontawesome και/ή Material Icons για την εμφάνιση δωρεάν εικονιδίων στο SRD σας

### Fontawesome

Χρησιμοποιήστε το Fontawesome για να αποκτήσετε δωρεάν εικονίδια και κινούμενα εικονίδια. Οι συνδρομητές μπορούν να χρησιμοποιούν Pro Icons

1.  Βρείτε το εικονίδιο στο [Fontawesome](https://fontawesome.com/icons)
2.  Επιλέξτε το εικονίδιο
3.  Στο αναδυόμενο παράθυρο, στην καρτέλα HTML αντιγράψτε οτιδήποτε μεταξύ " "

```EXAMPLE
<i class="fa-solid fa-dice-d20"></i>
```

4.  Τοποθετήστε το αντιγραμμένο κείμενο μεταξύ `:` χωρίς κενά

```EXAMPLE
fa-solid fa-dice-d20
```

### Material Icons

Χρησιμοποιήστε τα Material Icons για περισσότερα δωρεάν εικονίδια

1.  Βρείτε τα εικονίδια στο [Material Icons](https://fonts.google.com/icons)
2.  Αντιγράψτε & εισάγετε τον σύνδεσμο web span απευθείας στα αρχεία markdown σας

```markup
<span class="material-icons">face</span>
```
