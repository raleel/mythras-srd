# Μάχη

Η μάχη είναι ένα σημαντικό χαρακτηριστικό των παιχνιδιών ρόλων. Η βίαιη αντιπαράθεση ήταν ανέκαθεν ένας τρόπος επίλυσης προβλημάτων, από τις μάχες εκδίκησης μεταξύ εχθρών, μέχρι τη σύγκρουση στρατών σε καιρό πολέμου. Ωστόσο, η μάχη δεν χρειάζεται να είναι μια άγρια μάχη μέχρι θανάτου. Ορισμένες μάχες μπορεί να είναι ξιφασκίες που τελειώνουν χωρίς τραυματισμό, ή ίσως ελαφριές συμπλοκές σε μπαρ. Το _Mythras Imperative_ χρησιμοποιεί διάφορους όρους για να εξηγήσει τα βασικά στοιχεία της μάχης. Αυτοί εξηγούνται στις ακόλουθες ενότητες, αλλά σε γενικές γραμμές, είναι:
- **Στυλ Μάχης:** Η δεξιότητα χειρισμού μιας ομάδας όπλων που μαθαίνεται ως μέρος μιας κουλτούρας ή επαγγέλματος.
- **Γύροι Μάχης:** Λογιστικοί χρονικοί τομείς διάρκειας πέντε δευτερολέπτων ο καθένας.
- **Ενέργειες Μάχης:** Πιθανές ενέργειες που μπορεί να εκτελέσει ένας χαρακτήρας κατά τη διάρκεια ενός Γύρου με την κατανάλωση ενός Πόντου Ενέργειας.
- **Μέγεθος Όπλου:** Το πόσο δύσκολο είναι να αποκρούσεις ένα συγκεκριμένο όπλο.
- **Εμπλοκή:** Όταν οι μαχητές μπορούν να χτυπήσουν ή να χτυπηθούν από έναν άμεσο αντίπαλο, ανάλογα με τις περιστάσεις.
- **Ειδικά Εφέ:** Ελιγμοί μάχης που μπορούν να εκτελεστούν εάν ένας μαχητής αποκτήσει το πάνω χέρι.

## Στυλ Μάχης

Το Στυλ Μάχης αντιπροσωπεύει ένα «πακέτο» πολλαπλών όπλων που σχετίζονται με τη φύση τους, το οποίο αποφεύγει την αναγκαιότητα εκμάθησης κάθε μεμονωμένου όπλου ξεχωριστά.

Η πιο σημαντική πτυχή της Δεξιότητας Μάχης είναι ότι ο χαρακτήρας μαθαίνει πώς να χρησιμοποιεί κάθε όπλο της δεξιότητας, τόσο μεμονωμένα όσο και σε συνδυασμό, έτσι ώστε να μπορούν να εναλλάσσονται, όπως απαιτείται. Η εκπαίδευση μάχης δεν επικεντρώνεται μόνο στη χρήση ενός συγκεκριμένου όπλου ή όπλων υπό τις καλύτερες συνθήκες, αλλά καλύπτει επίσης τι πρέπει να γίνει όταν βρίσκεται σε σοβαρό μειονέκτημα. Μια τέτοια διασταυρούμενη εκπαίδευση είναι ένα πρωταρχικό μέρος της προετοιμασίας ενός μαχητή για τα απρόβλεπτα γεγονότα του πεδίου μάχης, όπου η θραύση όπλου ή ο αφοπλισμός είναι πάντα μια πιθανή δυνατότητα.

Έτσι, ένας Υπερβόρειος βάρβαρος μπορεί να αναμένεται να γνωρίζει πώς να χρησιμοποιεί δόρυ, τσεκούρι χειρός, ακόντιο και ασπίδα, ενώ ένας Σουμεριανός πολεμιστής μπορεί να είναι εκπαιδευμένος σε κοντόξιφο, ρόπαλο, ασπίδα και τόξο αντ' αυτού. Η απόφαση για το πόσα όπλα πρέπει να περιλαμβάνονται σε ένα μόνο στυλ είναι τελικά μια επιλογή που καθορίζεται από τον Games Master και το σκηνικό του παιχνιδιού.

Οι χαρακτήρες που επιχειρούν να χρησιμοποιήσουν ένα όπλο που δεν καλύπτεται από το στυλ τους θα πρέπει να υποστούν ποινή δεξιότητας μιας ή περισσότερων βαθμίδων δυσκολίας.

Τα στυλ μάχης έχουν και μια άλλη λειτουργία πέρα από την απλή εκμάθηση μιας ομάδας δεξιοτήτων όπλων. Δεδομένου ότι κάθε στυλ διδάσκεται από μια συγκεκριμένη κουλτούρα ή καριέρα, είναι εγγενώς βελτιστοποιημένα για το περιβάλλον ή τις στρατιωτικές τακτικές με τις οποίες συνήθως μάχεται αυτή η ομάδα. Για παράδειγμα, ένας νομάς ιππέας μαθαίνει να μάχεται από τη σέλα, ή ένας πεζικάριος εκπαιδεύεται να μάχεται σε κλειστό σχηματισμό. Επομένως, ως πρόσθετο μπόνους, ορισμένα Στυλ Μάχης μπορεί να αποδοθούν με ένα ή περισσότερα χαρακτηριστικά που του παρέχουν ένα πλεονέκτημα υπό ορισμένες συνθήκες. Τα χαρακτηριστικά προορίζονται να χρησιμοποιούνται μόνο με τα όπλα και τις καταστάσεις αυτού του συγκεκριμένου στυλ. Προφανώς, υπάρχουν αμέτρητα χαρακτηριστικά, πολλά από τα οποία είναι ειδικά για μια συγκεκριμένη περίοδο ή κόσμο εκστρατείας.

## Διεξαγωγή Μάχης

Για να τονιστεί η ενστικτώδης φύση της, η μάχη παρακολουθείται χτύπημα προς χτύπημα. Για να διευκολυνθεί η τήρηση αρχείων, οι μάχες χωρίζονται σε Γύρους Μάχης διάρκειας πέντε δευτερολέπτων ο καθένας. Κατά τη διάρκεια αυτής της περιόδου, οι μαχητές μπορούν να πραγματοποιήσουν επιθέσεις σώμα με σώμα ή να αμυνθούν εναντίον τους, να ελιχθούν μέσα και έξω από τη μάχη, να πυροβολήσουν ή να ρίξουν ένα Όπλο Εμβέλειας, και ούτω καθεξής. Παρόμοια με την πραγματική ζωή, οι περισσότερες εμπλοκές μάχης – μόλις ξεκινήσουν – ολοκληρώνονται σε δευτερόλεπτα παρά σε λεπτά. Σε όρους παιχνιδιού αυτό σημαίνει αρκετοί Γύροι Μάχης, εκτός εάν οι συμμετέχοντες αποσυρθούν για να επαναφέρουν ίσως την Πρωτοβουλία, να επανεκτιμήσουν την τακτική τους κατάσταση ή απλά να πάρουν μια στιγμιαία ανάπαυση πριν επανενεργοποιηθούν. Μόνο μάχες μεταξύ καλά ταιριασμένων αντιπάλων ή ακολουθίες αντιπάλων τείνουν να διαρκούν περισσότερο.

## Πυροβόλα Όπλα

Εάν χρησιμοποιείτε πυροβόλα όπλα σε ένα παιχνίδι _Mythras Imperative_, υπάρχουν μερικά πράγματα που πρέπει να λάβετε υπόψη, δεδομένης της θανατηφόρας φύσης μιας ανταλλαγής πυρών. Πρώτον, είναι σχεδόν αδύνατο να αποφύγετε ακτίνες λέιζερ ή σφαίρες που ταξιδεύουν ταχύτερα από τα ανθρώπινα αντανακλαστικά. Χρησιμοποιώντας τους κανόνες μάχης όπως παρουσιάζονται, αυτό σημαίνει ότι εκτός εάν κάποιος προσπαθήσει να βουτήξει από την περιοχή χρησιμοποιώντας την Αποφυγή, οποιοδήποτε χτύπημα θα εκθέσει αμέσως το θύμα σε ένα ή περισσότερα Ειδικά Εφέ. Αλλάζει επίσης την οικονομία των Πόντων Ενέργειας. Εάν δεν μπορείτε να χρησιμοποιήσετε πόντους για να αμυνθείτε αντιδραστικά, τότε οι περισσότεροι χαρακτήρες θα καταλήξουν να χρησιμοποιούν όλες τους τις ενέργειες για να πυροβολούν, ποντάροντας στην ανικανότητα των εχθρών τους. Για να διαχειριστείτε αυτή τη μετατόπιση παραδείγματος, τόσο οι Game Masters όσο και οι Παίκτες μπορεί να χρειαστεί να αλλάξουν τον τρόπο με τον οποίο βλέπουν – και προσεγγίζουν – τη μάχη με όπλα. Κάποια πράγματα που πρέπει να σημειωθούν είναι:

- Η μάχη με όπλα είναι επικίνδυνη!
- Γενικά, οι άνθρωποι δίνουν προτεραιότητα στο να μην πυροβοληθούν παρά στο να πυροβολούν άλλους.
- Η πιο κοινή μέθοδος προστασίας είναι η κάλυψη.
- Τα κατασταλτικά πυρά είναι ο καλύτερος τρόπος για να αποτρέψετε τους εχθρούς να πυροβολήσουν πίσω.
- Τα πυροβόλα όπλα είναι εκπληκτικά ανακριβή, εκτός εάν αφιερωθεί χρόνος για στήριξη ή/και στόχευση.
- Όταν σας πυροβολήσουν, συμβαίνουν άσχημα πράγματα.

Η άμυνα έναντι των πυροβολισμών γίνεται σε μεγάλο βαθμό ζήτημα θωράκισης σώματος, χρήσης κάλυψης και προσπάθειας να αποτρέψει τον αντίπαλο από το να μπορεί να ανταποδώσει τα πυρά. Συνεπώς, οι περισσότερες ανταλλαγές πυροβολισμών τείνουν να γίνονται πολύ τακτικές – πολλά κατασταλτικά πυρά, κινήσεις πλευρικής υπερκέρασης και εκδίωξη εχθρών από βαθιά κάλυψη – κατά τη διάρκεια των οποίων οι χαρακτήρες μπορεί να περνούν ολόκληρους γύρους μάχης καθηλωμένοι και ανίκανοι να ανταποκριθούν έως ότου η αντίσταση σταματήσει να πυροβολεί... κάτι που ορισμένοι παίκτες μπορεί να το βρουν εξαιρετικά κουραστικό.

Είναι απολύτως πιθανό μια ολόκληρη ανταλλαγή πυροβολισμών να μην οδηγήσει σε τραυματισμούς· απλά η μία πλευρά να παραδοθεί λόγω εξάντλησης πυρομαχικών ή να εγκαταλείψει εντελώς τη μάχη. Άλλωστε, λίγοι άνθρωποι θέλουν να διακινδυνεύσουν να τρυπηθούν από μια ριπή από ένα όπλο σε πλήρη αυτόματη λειτουργία.

Σε αυτούς τους κανόνες μάχης, ορισμένοι τύποι ενεργειών, Ειδικά Εφέ και καταστάσεις ισχύουν μόνο για μάχες με πυροβόλα όπλα, και αυτά σημειώνονται αναλόγως.

## Γύροι Μάχης και Πρωτοβουλία

Οι Γύροι Μάχης χρησιμοποιούν διάφορους σημαντικούς όρους:

- **Πρωτοβουλία:** Η σειρά με την οποία ενεργεί ο καθένας κατά τη διάρκεια μιας Στροφής ενός Γύρου Μάχης. Ο Games Master μετράει αντίστροφα τις τιμές Πρωτοβουλίας από την υψηλότερη προς τη χαμηλότερη, έτσι ώστε κάθε συμμετέχων να έχει την ευκαιρία να πραγματοποιήσει τη Στροφή του όταν φτάσει ο αριθμός του.
- **Στροφές:** Κατά τη διάρκεια κάθε Γύρου Μάχης 5 δευτερολέπτων, κάθε συμμετέχων έχει την ευκαιρία να πραγματοποιήσει μία ή περισσότερες Στροφές, ανάλογα με τον αριθμό των Πόντων Ενέργειας που διαθέτει και τον τρόπο που τους χρησιμοποιεί. Κατά τη Στροφή του, οι συμμετέχοντες εκτελούν μία Ενέργεια Μάχης, η οποία περιλαμβάνει μια δήλωση, πιθανή κίνηση, τυχόν απαραίτητες ρίψεις ζαριών (συμπεριλαμβανομένων των Αντιδράσεων), και την επίλυση της Ενέργειας. Εάν οποιοσδήποτε συμμετέχων έχει Πόντους Ενέργειας που του απομένουν, οι παίκτες μπορούν να κάνουν άλλη μια Στροφή κατά σειρά Πρωτοβουλίας. Μόλις δεν απομείνουν Πόντοι Ενέργειας, ή τυχόν εναπομείναντες κρατούνται για Αντιδράσεις, το παιχνίδι προχωρά στον επόμενο Γύρο Μάχης.

Κάθε όρος περιγράφεται, λεπτομερέστερα, παρακάτω.

#### Πρωτοβουλία

Η Πρωτοβουλία καθορίζει τη σειρά των ενεργειών των συμμετεχόντων σε έναν Γύρο Μάχης. Ρίχνεται στην αρχή μιας μάχης και καθορίζει πότε ενεργεί κάθε χαρακτήρας. Εκτός εάν συμβεί κάτι που αλλάζει την κατάσταση, όπως ορισμένες Ενέργειες Μάχης ή Ειδικά Εφέ, η Πρωτοβουλία παραμένει σε ισχύ μέχρι να αναγκαστεί να ξαναριχτεί.

Η Πρωτοβουλία υπολογίζεται με κάθε συμμετέχοντα να ρίχνει 1d10 και να προσθέτει τον Τροποποιητή Πρωτοβουλίας του. Όποιος πέτυχε το υψηλότερο αποτέλεσμα ενεργεί πρώτος, ακολουθούμενος από τον δεύτερο υψηλότερο και ούτω καθεξής. Όταν δύο ή περισσότεροι συμμετέχοντες ισοβαθμούν, αυτός με την υψηλότερη DEX θα ενεργήσει πρώτος. Εάν αυτό εξακολουθεί να οδηγεί σε ισοπαλία, αφήστε τον καθένα να ρίξει ένα ζάρι με την υψηλή ρίψη να προηγείται του άλλου.

#### Ενέργειες σε Μάχη

Μόλις καθοριστεί η Πρωτοβουλία, οι συμμετέχοντες έχουν τη δυνατότητα να εκτελέσουν διάφορες Ενέργειες Μάχης κατά τη διάρκεια κάθε Γύρου Μάχης. Όλοι οι χαρακτήρες αποκτούν 2 Πόντους Ενέργειας ανά Γύρο, αλλά το πότε μπορούν να ενεργήσουν περιορίζεται στο αν η Ενέργεια είναι Προληπτική ή Αντιδραστική.

Οι Προληπτικές Ενέργειες μπορούν να επιχειρηθούν μόνο στην δική του Στροφή του χαρακτήρα, δηλαδή, κατά την Πρωτοβουλία του. Οι Προληπτικές Ενέργειες είναι αυτές στις οποίες ο χαρακτήρας είναι ο υποκινητής, όπως η πραγματοποίηση μιας επίθεσης με ένα όπλο.

Οι Αντιδραστικές Ενέργειες είναι αυτές που αναλαμβάνει ένας χαρακτήρας για να αντιμετωπίσει ή να αντισταθεί σε μια πράξη που στρέφεται εναντίον του. Επιτρέπεται μόνο μία προσπάθεια αντίδρασης για κάθε απειλή, ένα παράδειγμα θα ήταν η προσπάθεια απόκρουσης μιας επίθεσης.

Οι αχρησιμοποίητοι Πόντοι Ενέργειας δεν μεταφέρονται από τον έναν Γύρο στον επόμενο.

## Ενέργειες Μάχης

Οι Ενέργειες Μάχης είναι οι πράξεις που μπορούν να εκτελεστούν κατά τη διάρκεια της μάχης. Η ανάσυρση ενός όπλου, το ρίξιμο μιας μαγείας ή η αποφυγή μιας επίθεσης είναι όλα παραδείγματα Ενεργειών Μάχης. Οι περισσότερες αφορούν στοιχεία της ίδιας της μάχης, αλλά ορισμένες σχετίζονται με δραστηριότητες εκτός του πεδίου των μαγικών και των όπλων.

Όπως αναφέρθηκε προηγουμένως, το πόσο συχνά ένας χαρακτήρας μπορεί να ενεργεί ανά Γύρο περιορίζεται από τους διαθέσιμους Πόντους Ενέργειας. Αυτοί δαπανώνται κατά τη διάρκεια κάθε Γύρου Μάχης για την εκτέλεση διαφόρων Ενεργειών Μάχης. Μόλις εξαντληθούν οι Πόντοι Ενέργειας ενός μαχητή, δεν μπορεί πλέον να ενεργήσει για το υπόλοιπο αυτού του Γύρου και πρέπει να περιμένει μέχρι να επανέλθουν οι πόντοι του στην αρχή του επόμενου.

Εκτός αν ορίζεται διαφορετικά, κάθε Ενέργεια Μάχης (εκτός από τις "Ελεύθερες" Ενέργειες) κοστίζει έναν Πόντο Ενέργειας. Έτσι, οι μαχητές πρέπει να εξετάσουν προσεκτικά πώς και πότε χρησιμοποιούν τους Πόντους Ενέργειας τους. Μπορούν να ενεργούν τόσο επιθετικά ή αμυντικά όσο επιθυμούν, αντιδρώντας δυναμικά στις εξελισσόμενες συνθήκες της μάχης.

### Προληπτικές Ενέργειες

Οι παρακάτω είναι δραστηριότητες που ένας χαρακτήρας μπορεί να επιχειρήσει στη Στροφή του ξοδεύοντας έναν Πόντο Ενέργειας. Σημειώστε ότι ορισμένες Ενέργειες, όπως η εκτέλεση μαγείας ή η επαναφόρτωση, μπορεί να χρειαστούν αρκετές Στροφές για να ολοκληρωθούν. Κάθε Στροφή κοστίζει τον δικό της Πόντο Ενέργειας.

#### Επίθεση

Ο χαρακτήρας μπορεί να επιχειρήσει να χτυπήσει με όπλο σώμα με σώμα ή να χρησιμοποιήσει Όπλο Εμβέλειας.

#### Στήριξη

Ο χαρακτήρας στηρίζεται παίρνοντας μια σταθερή στάση και γέρνοντας προς την κατεύθυνση μιας επερχόμενης επίθεσης. Για τους σκοπούς της αντίστασης σε Χτύπημα Πίσω (Knockback) ή Επιθέσεις Άλματος (Leaping Attacks), το SIZ του χαρακτήρα αντιμετωπίζεται ως 50% μεγαλύτερο. Εναντίον του Ειδικού Εφέ Μπάσιμο (Bash), το SIZ διπλασιάζεται.

#### Κάστα Μαγείας/Χρήση Δύναμης

Ο χαρακτήρας μπορεί να επιχειρήσει να εκτελέσει ένα ξόρκι, να επικαλεστεί ορισμένες δυνάμεις, και ούτω καθεξής. Οι σύνθετες μαγείες μπορεί να απαιτήσουν αρκετές Ενέργειες για να ολοκληρωθεί η εκτέλεση. Μόλις ολοκληρωθεί, η μαγεία μπορεί να απελευθερωθεί οποιαδήποτε στιγμή μέχρι την επόμενη Στροφή του εκτελεστή – οπότε μπορεί να κρατηθεί για μεταγενέστερο αποτέλεσμα, αλλά αυτό απαιτεί την Ενέργεια Κράτημα Μαγείας/Δύναμης (βλ. παρακάτω) για να τη διατηρήσει σε προετοιμασία για μεταγενέστερη απελευθέρωση.

#### Αλλαγή Εμβέλειας

Ο χαρακτήρας μπορεί να επιχειρήσει να πλησιάσει ή να αποσυρθεί από έναν αντίπαλο, αρκεί να μην εμπλέκεται σε μάχη σώμα με σώμα με τον αντίπαλο ή να μην καθηλώνεται από εχθρικά πυρά σε κατάσταση μάχης εμβέλειας.

#### Έφοδος

Η ενέργεια Έφοδος επιτρέπει σε έναν χαρακτήρα να εισέλθει σε εμβέλεια εμπλοκής με ταχύτητα τρεξίματος ή σπριντ, χρησιμοποιώντας την ορμή της εφόδου για να κάνει μια πιο δυνατή επίθεση κοντινής μάχης. Αυτοί οι κανόνες καλύπτονται λεπτομερέστερα στην ενότητα [Έφοδος](0005_Combat.md?id=charging) παρακάτω.

#### Καθυστέρηση

Ο χαρακτήρας διατηρεί μία ή περισσότερες Ενέργειες για να εκτελέσει Αντιδραστικές Ενέργειες αργότερα, όπως Διακοπή ή Απόκρουση. Το κόστος Πόντων Ενέργειας της καθυστέρησης καλύπτεται από όποιες πράξεις εκτελεστούν τελικά. Εάν οι καθυστερημένες Ενέργειες δεν πραγματοποιηθούν πριν από την επόμενη Στροφή του χαρακτήρα, τότε ο χαρακτήρας θεωρείται ότι έχει Διστάσει (βλ. παρακάτω), και ο Πόντος Ενέργειας χάνεται.

#### Δισταγμός

Η προεπιλεγμένη επιλογή εκτός αν επιλεγεί άλλη Ενέργεια, ο χαρακτήρας απλώς σπαταλά τη σειρά του χωρίς να κάνει τίποτα χρήσιμο.

#### Κράτημα Μαγείας/Δύναμης

Μόλις ολοκληρωθεί η εκτέλεση, ο χαρακτήρας μπορεί να κρατήσει ένα ξόρκι ή Δύναμη σε προσωρινό έλεγχο, περιμένοντας την καλύτερη στιγμή για να το απελευθερώσει. Η μαγεία μπορεί να κρατηθεί για όσο χρονικό διάστημα ο χαρακτήρας συνεχίζει να κάνει αυτή την Ενέργεια σε διαδοχικές Στροφές, αλλά αυτό επιτρέπει την ελεύθερη χρήση της αντίδρασης Αντι-ξόρκι από έναν εχθρό, εάν σχετίζεται με το ξόρκι. Η πραγματική ρίψη δεξιότητας για την εκτέλεση του κρατημένου ξορκιού δεν γίνεται μέχρι να εκτελεστεί.

#### Ιππεία

Ο χαρακτήρας μπορεί να ανέβει ή να κατέβει από ένα ιππεύσιμο ζώο ή όχημα. Ιδιαίτερα μεγάλα οχήματα μπορεί να απαιτήσουν αρκετές Στροφές για να ολοκληρωθούν.

#### Κίνηση

Εφόσον δεν είναι εμπλεκόμενος με αντίπαλο, ο χαρακτήρας μπορεί να κινηθεί σε οποιαδήποτε απόσταση ο Games Master κρίνει κατάλληλη για την περίσταση.

#### Υπερκερασμός

Ο χαρακτήρας μπορεί να εμπλακεί με πολλούς αντιπάλους σε μια ομαδική Αντιτιθέμενη Ρίψη δεξιοτήτων Αποφυγής. Όσοι αποτυγχάνουν να ξεπεράσουν τη ρίψη του δεν μπορούν να του επιτεθούν σε αυτόν τον Γύρο Μάχης. Δείτε την ενότητα [Υπερκερασμός](0005_Combat.md?id=outmaneuvering) παρακάτω.

#### Προετοιμασία Όπλου

Ο χαρακτήρας μπορεί να ανακτήσει, να τραβήξει, να θηκεύσει, να αποσύρει ή να επαναφορτώσει ένα όπλο ή άλλο αντικείμενο. Η ανάκτηση ενός κοντινού πεσμένου αντικειμένου απαιτεί 2 Ενέργειες: μία για να κινηθεί και να σκύψει για το αντικείμενο και μια δεύτερη για να επανέλθει σε ετοιμότητα. Ορισμένα όπλα βολής απαιτούν αρκετές Ενέργειες για να επαναφορτωθούν.

#### Επαναφόρτωση (Πυροβόλα Όπλα)

Όπως και η Προετοιμασία Όπλου. Οι χρόνοι επαναφόρτωσης μπορεί να ποικίλλουν ευρέως ανάλογα με τον τύπο του όπλου. Τα μονοσφαιρα μουσκέτα μαύρης πυρίτιδας θα είναι πολύ πιο αργά στην επαναφόρτωση από την αλλαγή γεμιστήρα πυρομαχικών ή πηγής ενέργειας.

#### Ανάκτηση Στήριξης

Εάν δεν είναι εμπλεκόμενος με αντίπαλο, ο χαρακτήρας μπορεί αυτόματα να ανακτήσει τη στήριξή του από το να σκοντάψει ή να πέσει. Εάν είναι Εμπλεκόμενος, ο χαρακτήρας πρέπει να κερδίσει μια Αντιτιθέμενη Ρίψη Δύναμης ή Αθλητισμού με τον αντίπαλο πριν σηκωθεί. Αυτός ο διαγωνισμός δεν κοστίζει Πόντους Ενέργειας για να επιλυθεί.

#### Κρυφή Ματιά (Πυροβόλα Όπλα)

Η Κρυφή Ματιά επιτρέπει σε κάποιον που βρίσκετai αυτή τη στιγμή πίσω από κάλυψη να ρίξει μια γρήγορη ματιά για να κρίνει την τακτική κατάσταση. Η ενέργεια είναι φαινομενικά ασφαλής, επιτρέποντας στον παρατηρητή μια μοναδική ρίψη Αντίληψης πριν ξανακρυφτεί. Εάν όμως ο έλεγχος Αντίληψης είναι μια αποτυχία (fumble), υπερεκτείνει ανόητα το κεφάλι του ή παραμένει ορατός αρκετά ώστε ένας εχθρός να του ρίξει μια βολή.

#### Σταθεροποίηση Όπλου (Πυροβόλα Όπλα)

Επιτρέπει στον χαρακτήρα να υιοθετήσει μια σταθερή στάση βολής και να υποστηρίξει ένα αυτόματο όπλο ώστε να παραμείνει σταθερό. Απαιτεί ο χαρακτήρας να είναι τουλάχιστον μερικώς εκτεθειμένος και να μην προστατεύεται πλήρως από κάλυψη. Το κύριο όφελος είναι ότι μειώνει την ποινή ανάκρουσης των ριπών ή της πλήρως αυτόματης βολής κατά ένα βήμα. Αυτό το αποτέλεσμα είναι σαν το Στήριξη, καθώς τα οφέλη του συνεχίζονται μέχρι ο χαρακτήρας να κινηθεί.

#### Αγωνία

Εάν ο χαρακτήρας είναι θύμα ορισμένων τύπων επίθεσης ή Ειδικών Εφέ, μπορεί να επιχειρήσει να απεγκλωβιστεί από την κατάσταση. Για παράδειγμα, να ξεφύγει από το κράτημα ενός εχθρού.

#### Αλλαγή Λειτουργίας Πυρός (Πυροβόλα Όπλα)

Η αλλαγή λειτουργίας πυρός επιτρέπει στον χρήστη ενός αυτόματου ή διπλού σκοπού πυροβόλου όπλου να αλλάξει τον επιλογέα στο όπλο του για να εκμεταλλευτεί άλλες λειτουργίες πυρός. Έτσι, ένα τυφέκιο εφόδου θα μπορούσε να αλλάξει από ημι-αυτόματο σε ριπή, να επιλέξει διαφορετικούς τύπους πυρομαχικών, ή να ενεργοποιηθεί ένα συμπληρωματικό σύστημα όπως ένας ενσωματωμένος εκτοξευτής χειροβομβίδων. Ενώ τα σύγχρονα όπλα απαιτούν έναν Πόντο Ενέργειας στη Στροφή του χαρακτήρα για να αλλάξουν λειτουργίες, τα επαρκώς προηγμένα όπλα υψηλής τεχνολογίας μπορεί να το επιτρέπουν ως Ελεύθερη Ενέργεια, ενεργοποιούμενη από ηλεκτρονικά εμφυτεύματα ή διαβάζοντας ψυχικά το μυαλό του χρήστη.

#### Κάλυψη

Η Κάλυψη είναι μια προληπτική ενέργεια που επιτρέπει σε έναν σκοπευτή να σκύψει πίσω από όποια κάλυψη είναι διαθέσιμη στην άμεση γειτονιά του, κερδίζοντας έτσι έναν βαθμό προστασίας από τα εισερχόμενα πυρά. Σε αντίθεση με την Αποφυγή, δεν αφήνει τον χρήστη πρηνή, αλλά βασίζεται στο να υπάρχει κάποια μορφή κάλυψης, για παράδειγμα, να σκύψει πίσω από μια γωνία σε ένα διάδρομο ή να σκύψει πίσω από ένα τραπέζι σε ένα μπαρ. Ανάλογα με τις περιστάσεις, η διαθέσιμη κάλυψη μπορεί να είναι ή να μην είναι επαρκής για να προστατεύσει πλήρως τον χαρακτήρα. Ο τύπος της κάλυψης θα καθορίσει επίσης τις προστατευτικές της ιδιότητες· μια παχιά χαλύβδινη πόρτα, για παράδειγμα, μπορεί να αποδειχθεί αδιαπέραστη από σφαίρες, ενώ ένας λεπτός ξύλινος τοίχος μπορεί να παρέχει μόνο 4 Πόντους Θωράκισης. Για γενικές οδηγίες σχετικά με τις προστατευτικές ιδιότητες ορισμένων υλικών, δείτε τον πίνακα στην ενότητα [Άψυχα Αντικείμενα](0004_Game_System.md?id=inanimate-objects).

### Αντιδραστικές Ενέργειες

Αυτή η λίστα προσδιορίζει τις Αντιδράσεις που μπορούν να χρησιμοποιηθούν ανά πάσα στιγμή κατά τη διάρκεια του Γύρου Μάχης ως απάντηση σε μια επικείμενη απειλή. Όπως και στην προηγούμενη λίστα, μια αντίδραση κοστίζει έναν Πόντο Ενέργειας για να εκτελεστεί.

#### Αντι-ξόρκι

Ο χαρακτήρας μπορεί να επιχειρήσει να διαλύσει ή να αντιμετωπίσει ένα εισερχόμενο ξόρκι. Αυτό προϋποθέτει ότι η αντιθετική μαγεία έχει Χρόνο Εκτέλεσης μιας Στροφής, διαφορετικά πρέπει να προετοιμαστεί εκ των προτέρων και να συγκρατηθεί προσωρινά χρησιμοποιώντας την Ενέργεια Κράτημα Μαγείας. Η επιτυχής αναχαίτιση της μαγείας με αυτόν τον τρόπο θεωρείται ότι αναιρεί ολόκληρο το ξόρκι, ακόμη και αυτά με πολλαπλούς στόχους ή περιοχές επίδρασης.

#### Αποφυγή

Ο χαρακτήρας μπορεί να χρησιμοποιήσει τη δεξιότητα Αποφυγής σε μια προσπάθεια να αποφύγει απειλές όπως εισερχόμενα βλήματα ή μια έφοδο επίθεση. Αυτό αφήνει τον χαρακτήρα πρηνή, εκτός εάν μετριαστεί από μια Ικανότητα ή ειδική περίσταση. Έτσι, η επόμενη σειρά του χαρακτήρα συνήθως δαπανάται για την Ενέργεια Ανάκτηση Στήριξης για να σηκωθεί ξανά.

#### Διακοπή

Μόνο για χαρακτήρες που καθυστερούν. Αυτή η Αντιδραστική Ενέργεια σταματά τη Στροφή ενός αντιπάλου σε οποιοδήποτε σημείο για να πραγματοποιήσει μια καθυστερημένη Ενέργεια Στροφής. Υποθέτοντας ότι δεν υπάρχει αλλαγή στην τακτική κατάσταση, ο αντίπαλος συνεχίζει τη Στροφή μετά την ολοκλήρωση αυτής του χαρακτήρα. Εάν δεν είναι ακόμη δυνατή η επίτευξη της αρχικής δήλωσης, ο Πόντος Ενέργειας του αντιπάλου χάνεται. Μια διακοπή μπορεί επίσης να χρησιμοποιηθεί για την εκτέλεση επίθεσης ευκαιρίας εναντίον οποιουδήποτε περνά κοντά στον χαρακτήρα που καθυστερεί και βρίσκεται σε εμβέλεια όπλου.

#### Απόκρουση

Ο χαρακτήρας μπορεί να επιχειρήσει να αμυνθεί ενάντια σε μια εισερχόμενη επίθεση χρησιμοποιώντας συνδυασμό απόκρουσης, κλίσης, σκυψίματος και πλαγιολίσθησης για να ελαχιστοποιήσει το χτύπημα.

### Ελεύθερες Ενέργειες

Οι Ελεύθερες Ενέργειες (που αναφέρονται παρακάτω) μπορούν να εκτελεστούν οποιαδήποτε στιγμή κατά τη διάρκεια του Γύρου Μάχης και δεν κοστίζουν Πόντους Ενέργειας.

#### Αξιολόγηση Κατάστασης

Εάν δεν είναι εμπλεκόμενος, ένας χαρακτήρας μπορεί να κάνει μια ρίψη Αντίληψης χωρίς κόστος Πόντων Ενέργειας. Μια Επιτυχία αποκαλύπτει τυχόν σχετικές αλλαγές στην τακτική κατάσταση (όπως το να εντοπίσει έναν εχθρό που αρχίζει μια έφοδο). Η αξιολόγηση κατά το Τρέξιμο οδηγεί σε μια Δύσκολη ρίψη Αντίληψης, ενώ το Σπριντ είναι Δύσκολο (Formidable).

#### Απόρριψη Αντικειμένου

Η απόρριψη ενός αντικειμένου είναι μια Ελεύθερη Ενέργεια.

#### Σήμα

Εάν δεν είναι απασχολημένος, η χειρονομία ή η σήμανση σε έναν ή περισσότερους συμμετέχοντες (εάν μπορούν να αντιληφθούν το σήμα) είναι μια Ελεύθερη Ενέργεια.

#### Ομιλία

Ένας χαρακτήρας μπορεί να μιλήσει οποιαδήποτε στιγμή κατά τη διάρκεια της μάχης, αλλά αυτά που λέγονται πρέπει να περιορίζονται σε σύντομες φράσεις που μπορούν να ειπωθούν σε πέντε δευτερόλεπτα ή λιγότερο· για παράδειγμα, «Πρόσεχε πίσω σου!» ή «Καταραμένε!».

#### Χρήση Πόντου Τύχης

Η χρήση ενός Πόντου Τύχης – για παράδειγμα, για επανάληψη μιας συγκεκριμένης ρίψης – είναι μια Ελεύθερη Ενέργεια.

#### Φύλαξη Τοποθεσίας

Ο χαρακτήρας φυλάει μια συγκεκριμένη Σημείο Πλήγματος από το να χτυπηθεί, αφιερώνοντας ένα από τα όπλα του για να μπλοκάρει παθητικά την περιοχή. Κάθε χτύπημα που προσγειώνεται σε αυτή την τοποθεσία υποβαθμίζεται αυτόματα η ζημιά του όπως είναι φυσιολογικό για ένα όπλο απόκρουσης του Μεγέθους του. Η κάλυψη συνεχίζεται μέχρι το αφιερωμένο όπλο να χρησιμοποιηθεί για επίθεση ή ενεργή Απόκρουση. Η καθιέρωση ή αλλαγή του καλυμμένου Σημείου Πλήγματος πρέπει να πραγματοποιηθεί πριν ο αντίπαλος ρίξει για να επιτεθεί στον χαρακτήρα. Λόγω του σχεδιασμού τους, οι ασπίδες μπορούν να καλύψουν πολλαπλές περιοχές.

Για περαιτέρω επεξηγήσεις, δείτε [Παθητικό Μπλοκάρισμα](0005_Combat.md?id=passive-blocking) παρακάτω.

## Κίνηση

Οι ακόλουθοι κανόνες διέπουν την εμπλοκή στη μάχη:

- Η κίνηση πραγματοποιείται είτε με την Ενέργεια Κίνηση είτε με την Ενέργεια Έφοδος.
- Η Ενέργεια Κίνηση επιτρέπει στον χαρακτήρα να μετακινηθεί σε απόσταση ίση με το Βασικό του Ρυθμό Κίνησης.
- Η κίνηση μπορεί να πραγματοποιηθεί μόνο από μη εμπλεκόμενους χαρακτήρες (εκτός από ειδικές περιπτώσεις όπως ο Υπερκερασμός).
- Εκτός από ορισμένες περιπτώσεις (όπως η έφοδος), όλες οι σημαντικές κινήσεις σταματούν μόλις ένας χαρακτήρας εισέλθει στην εμβέλεια εμπλοκής του επιδιωκόμενου αντιπάλου.
- Εκτός εάν γίνεται έφοδος μέσω επαφής, η είσοδος στη ζώνη εμπλοκής του αντιπάλου θέτει τον έναν σε εμπλοκή με τον άλλο.
- Μόλις εμπλακούν, οι χαρακτήρες δεν μπορούν να απομακρυνθούν από έναν αντίπαλο, εκτός εάν πρώτα αποσυρθούν από τη στενή μάχη.
- Η προσπάθεια να περάσει κανείς έναν μη εμπλεκόμενο εχθρό που χρησιμοποιεί την Ενέργεια Μάχης Καθυστέρηση, επιτρέπει (ως ειδική εξαίρεση) σε αυτόν τον αντίπαλο να χτυπήσει τους κινούμενους χαρακτήρες καθώς περνούν ή να μπλοκάρει την πρόοδο ενός χαρακτήρα, προκαλώντας στενή μάχη στην επόμενη Στροφή ή Γύρο.

### Ρυθμοί Κίνησης ανά Κατάσταση

Αυτοί οι ρυθμοί κίνησης είναι καταστασιακοί και δεν αποτελούν πάντα επιλογή. Πολλοί από αυτούς αναφέρονται αλλού, αλλά περιλαμβάνονται εδώ για ευκολία. Οι περισσότεροι από αυτούς τους καταστασιακούς ρυθμούς μπορούν να χρησιμοποιηθούν επιπρόσθετα σε έναν κανονικό Ρυθμό Κίνησης υπό καθορισμένους περιορισμούς. Τέτοιες πληροφορίες αναφέρονται στις ειδικές περιγραφές παρακάτω.

#### Αναρρίχηση

Ένας χαρακτήρας που κινείται δίπλα ή ξεκινά τη Στροφή δίπλα σε μια αναρριχητική επιφάνεια ή αντικείμενο μπορεί να αναρριχηθεί με ρυθμό ίσο με τον Βασικό Ρυθμό Κίνησης. Ο χαρακτήρας δεν μπορεί να έχει κινηθεί γρηγορότερα από το Βάδισμα και πρέπει να του έχει απομείνει κίνηση, με την απόσταση που αναρριχήθηκε να μειώνεται κατά τυχόν κίνηση που έχει ήδη χρησιμοποιηθεί. Η φορεμένη πανοπλία εμποδίζει τους αναρριχητές με βάση το σύνολο των Πόντων Θωράκισης που φορούν.

##### Πίνακας Ποινής Κίνησης με Θωράκιση

| 16-23 | Μεσαίο | -2 |
| 24-31 | Βαρύ | -3 |
| 32-39 | Δύσβατο | -4 |

#### Συρμός

Ένας χαρακτήρας που ξεκινά τη Στροφή πρηνής μπορεί να συρθεί με μέγιστο ρυθμό δύο μέτρων ανά Στροφή αντί για μια κανονική Κίνηση. Πλάσματα που συρθήκουν φυσικά ως ο κανονικός τους τρόπος κίνησης χρησιμοποιούν αντ' αυτού τον πλήρη Ρυθμό Κίνησης τους τροποποιημένο από τις Βαδίσεις - Βάδισμα, Τρέξιμο και Σπριντ - δείτε την ενότητα [Ρυθμός Κίνησης](0002_Characters.md?id=movement-rate) παραπάνω.

Οι κανόνες κίνησης καλύπτονται πλήρως στο βιβλίο κανόνων του _Mythras_.

#### Άλμα

Ένας χαρακτήρας που κινείται δίπλα ή ξεκινά τη Στροφή δίπλα σε ένα εμπόδιο που μπορεί να πηδήξει, μπορεί να επιχειρήσει να πηδήξει πάνω από αυτό. Ο χαρακτήρας μπορεί να κινείται με οποιοδήποτε ρυθμό μέχρι το σημείο του άλματος.

Μια επιτυχημένη ρίψη Αθλητισμού επιτρέπει σε κάποιον να πηδήξει έως και διπλάσιο του ύψους του οριζόντια ή έως και το μισό του ύψους του κάθετα (εάν ο χαρακτήρας έχει τουλάχιστον πέντε μέτρα φόρα διαθέσιμα). Εάν πηδάει από στάση, τότε αυτές οι αποστάσεις μειώνονται στο μισό.

Μειώστε την απόσταση του άλματος σε μέτρα κατά το ένα πέμπτο του συνόλου των 'Πραγμάτων' που μεταφέρονται. Η φορεμένη πανοπλία εμποδίζει τους άλτες, με κάθε θέση που καλύπτεται από ελαφριά πανοπλία να μετράται ως 1 'Πράγμα' στον προαναφερθέντα υπολογισμό, και η βαριά πανοπλία μετράται ως 2 'Πράγματα'.

## Πώς Λειτουργεί η Μάχη

Η μάχη στο _Mythras Imperative_ επιλύεται με κάθε επίθεση ή απόκρουση να αντιπροσωπεύει ένα και μόνο χτύπημα, κίνηση, εκτέλεση ξορκιού ή βολή όπλου. Οποιαδήποτε επιθετική Ενέργεια επιτρέπει την ευκαιρία αντίστασης με μια αντίδραση. Έτσι, ακόμη και αν ένας χαρακτήρας καταφέρει να χτυπήσει έναν αντίπαλο, ο εχθρός επιτρέπεται η δική του ρίψη για να δει αν μπορεί να αποκρούσει το χτύπημα πριν αυτό προσγειωθεί. Η ίδια φιλοσοφία χρησιμοποιείται είτε οι επιθέσεις είναι όπλα βολής εναντίον στόχων που βουτούν για κάλυψη είτε προσπαθούν να ξεφύγουν από το σιδερένιο κράτημα ενός γιγάντιου χταποδιού.

### Επιθέσεις και Αποκρούσεις

Η στενή μάχη χειρίζεται με τον ακόλουθο βήμα προς βήμα τρόπο:

1.  Στη σειρά του, ο επιτιθέμενος ξοδεύει έναν Πόντο Ενέργειας, ρίχνει ενάντια στη Δεξιότητα Μάχης του και σημειώνει το αποτέλεσμα.
2.  Εάν επιθυμεί, ο αμυνόμενος ξοδεύει έναν Πόντο Ενέργειας για να αποκρούσει, ρίχνει ενάντια στη Δεξιότητα Μάχης του και σημειώνει το αποτέλεσμα.
3.  Το επίπεδο επιτυχίας των αποτελεσμάτων συγκρίνεται σύμφωνα με μια Διαφορική Ρίψη.
4.  Οποιαδήποτε διαφορά παρέχει στον επιτυχημένο μαχητή με την ανώτερη ρίψη ένα ή περισσότερα Ειδικά Εφέ.
5.  Εάν ο επιτιθέμενος πέτυχε Επιτυχία ή Κρίσιμο, μπορεί να ρίξει ζημιά όπλου και να εφαρμόσει τον Τροποποιητή Ζημιάς του (εάν υπάρχει). Εάν ισχύει, καθορίζεται ένα Σημείο Πλήγματος για το χτύπημα.
6.  Εάν ο αμυνόμενος πέτυχε Επιτυχία ή Κρίσιμο, μειώστε τυχόν ζημιές που προκλήθηκαν σύμφωνα με τα συγκριτικά μεγέθη των όπλων που χρησιμοποιήθηκαν.
7.  Μειώστε τυχόν εναπομείναντες ζημιές κατά τους Πόντους Θωράκισης φυσικής ή φορετής προστασίας.

Σημειώστε ότι οποιαδήποτε Ειδικά Εφέ που παράγονται από την ανταλλαγή είναι ανεξάρτητα από το αν προκαλείται ζημιά. Είναι απολύτως πιθανό ένας αμυνόμενος να αποκτήσει υψηλότερο επίπεδο επιτυχίας, αλλά να εξακολουθεί να υφίσταται τραυματισμό.

### Μέγεθος Όπλου

Κάθε όπλο στο _Mythras Imperative_ διαθέτει διάφορες διαφορετικές ιδιότητες, από τη ζημιά που προκαλεί μέχρι το πόσα χέρια απαιτούνται για τον χειρισμό του. Το Μέγεθος είναι ένας συνδυασμός της μάζας, της μόχλευσης και της σταθερότητας ενός όπλου, και χρησιμοποιείται για τον προσδιορισμό της ικανότητας του όπλου να επιβάλλει και να αποκρούει ζημιά. Οι κατηγορίες Μεγέθους είναι Μικρό, Μεσαίο, Μεγάλο, Τεράστιο και Εξαιρετικά Μεγάλο. Οι άοπλες επιθέσεις των πλασμάτων εκχωρούνται σε παρόμοιες κατηγορίες ανάλογα με τα φυσικά τους χαρακτηριστικά.

### Μείωση Ζημιάς

Εάν ένας αμυνόμενος καταφέρει να αποκρούσει, τότε μπορεί να μειώσει τη ζημιά του επιτιθέμενου, εάν υπάρχει, σύμφωνα με το συγκριτικό Μέγεθος των όπλων που χρησιμοποιήθηκαν.

- Η απόκρουση μιας επίθεσης με όπλο ή ασπίδα ίσου ή μεγαλύτερου Μεγέθους εκτρέπει όλη τη ζημιά.
- Η απόκρουση με όπλο ή ασπίδα ενός Μεγέθους λιγότερο εκτρέπει μόνο τη μισή ζημιά.
- Η απόκρουση με όπλο ή ασπίδα δύο ή περισσότερων Μεγεθών λιγότερο αποτυγχάνει να εκτρέψει οποιαδήποτε ζημιά.

_Για παράδειγμα, η απόκρουση ενός μεγάλου τσεκουριού (Τεράστιο) με μια ασπίδα χαρταετού (επίσης Τεράστιο) θα μπλόκαρε όλη τη ζημιά· η απόκρουσή του με ένα μακρύ ξίφος (Μεγάλο) θα μείωνε στο μισό τη ζημιά, και η απόκρουσή του με ένα κοντό ξίφος (Μεσαίο) δεν θα σταματούσε καθόλου ζημιά._

### Θωράκιση

Κάθε ζημιά που διαπερνά την Απόκρουση του αμυνόμενου μειώνεται περαιτέρω από τους Πόντους Θωράκισης οποιασδήποτε φυσικής προστασίας (όπως λέπια ή κέλυφος) ή φορετής πανοπλίας που διαθέτει. Εάν ο αμυνόμενος έχει και τα δύο, τότε οι δύο συνδυάζονται για να μειώσουν τη ζημιά.

Λόγω της μάζας της και του περιορισμού στην κίνηση, η υψηλότερη τιμή Πόντων Θωράκισης της πανοπλίας που φοριέται από έναν χαρακτήρα λειτουργεί ως ποινή στην ρίψη Πρωτοβουλίας του.

### Απόκρουση μιας Άστοχης Επίθεσης

Εάν ο επιτιθέμενος αστοχήσει στην αρχική του ρίψη επίθεσης, ο αμυνόμενος έχει την επιλογή να ξοδέψει έναν Πόντο Ενέργειας για Απόκρουση. Αν και μπορεί να φαίνεται παράλογο να αποκρούσεις μια επίθεση που θα αστοχήσει ούτως ή άλλως, ένας έμπειρος αμυνόμενος μπορεί να το χρησιμοποιήσει προς όφελός του για να αποκτήσει ένα ή περισσότερα Ειδικά Εφέ, αποδυναμώνοντας ή αχρηστεύοντας ενδεχομένως έναν εχθρό και διαφυλάσσοντας την επόμενη σειρά του για κάποια άλλη Ενέργεια.

### Ανίκανος ή Απρόθυμος να Αποκρούσει

Σε μια κατάσταση όπου ένας αμυνόμενος είναι ανίκανος να Αποκρούσει λόγω έλλειψης εναπομεινάντων Πόντων Ενέργειας, ή, σίγουρος για την ικανότητά του να αντέξει το χτύπημα, επιλέγει να μην Αποκρούσει, αντιμετωπίζεται σαν να έχει αυτόματα ρίξει μια Αποτυχία. Αυτό έχει ως συνέπεια την παροχή στον επιτυχημένο επιτιθέμενο ενός ή περισσότερων Ειδικών Εφέ.

### Αποτυχημένες Ρίψεις και Αδέξιες Κινήσεις

Εάν και οι δύο μαχητές Αποτύχουν στις 2 ρίψεις τους, ή ο αμυνόμενος αποφασίσει να μην εκμεταλλευτεί μια χαμένη επίθεση, τότε η ακολουθία επίθεση – Απόκρουση τελειώνει, και η μάχη συνεχίζεται στον συμμετέχοντα με την επόμενη υψηλότερη Πρωτοβουλία.

## Ειδικά Εφέ

Η μάχη είναι πολύ περισσότερο από το να τραυματίζεις ή να σκοτώνεις έναν αντίπαλο. Οι πολεμικές τέχνες διδάσκουν πολλές μεθόδους για να νικήσεις έναν εχθρό, ίσως αφήνοντάς τον ανήμπορο ή αναγκάζοντάς τον σε καταστάσεις όπου πρέπει να υποχωρήσει, χωρίς να χρειάζεται απαραίτητα να του προκαλέσεις βλάβη. Τα Ειδικά Εφέ αντιπροσωπεύουν αυτές τις τεχνικές και ελέγχουν πώς εμφανίζονται στο παιχνίδι.

Όποτε οι αντίπαλοι εμπλέκονται σε μια Διαφορική Ρίψη των αντίστοιχων δεξιοτήτων μάχης τους, οποιαδήποτε προκύπτουσα διαφορά στα επίπεδα επιτυχίας υποδεικνύει μια ευκαιρία για την εμφάνιση Ειδικών Εφέ. Αυτό αντικατοπτρίζει έναν μαχητή που ελιχθεί τον αντίπαλό του σε μια δυσμενή κατάσταση που μπορεί να εκμεταλλευτεί χρησιμοποιώντας ένα έξυπνο κόλπο ή τακτική.

Ο αριθμός των Ειδικών Εφέ που λαμβάνονται εξαρτάται από τη διαφορά μεταξύ των Επιπέδων Επιτυχίας, όπως απεικονίζεται στον πίνακα Διαφορικών Επιπέδων Επιτυχίας.

Εάν κερδηθούν Ειδικά Εφέ κατά τη διάρκεια μιας ανταλλαγής, πρέπει να επιλεγούν πριν ριχτεί η Ζημιά και η Τοποθεσία Πλήγματος, εάν υπάρχουν. Σε περιπτώσεις όπου ο δέκτης τραυματίζεται σοβαρά επιπρόσθετα στην λήψη Ειδικού Εφέ, οι ρίψεις Αντοχής επιλύονται μετά την εφαρμογή του εφέ.

Τα Ειδικά Εφέ καλύπτουν ένα ευρύ φάσμα καταστάσεων και τεχνικών. Ορισμένα είναι σχεδιασμένα ειδικά για επιθέσεις, ενώ άλλα προορίζονται για άμυνα, και λίγα είναι προσαρμόσιμα και για τις δύο περιπτώσεις. Ορισμένα Ειδικά Εφέ μπορούν επίσης να περιορίζονται σε συγκεκριμένους τύπους όπλων ή συγκεκριμένες ρίψεις ζαριών, απαιτώντας, για παράδειγμα, ένα Κρίσιμο ή Άστοχο αποτέλεσμα στον έλεγχο δεξιότητας τους.

Όταν αποκτηθούν δύο ή περισσότερα Ειδικά Εφέ, ο μαχητής μπορεί ελεύθερα να συνδυάσει ποια επιλέγει, εφόσον πληρούνται οι προαπαιτούμενες συνθήκες για το καθένα. Ορισμένα εφέ μπορούν να στοιβάζονται. Για παράδειγμα, ένας επιτιθέμενος που ρίχνει Κρίσιμη Επιτυχία και κερδίζει δύο Ειδικά Εφέ θα μπορούσε να επιλέξει Μεγιστοποίηση Ζημιάς δύο φορές αντί να επιλέξει δύο ξεχωριστούς επιθετικούς ελιγμούς.

### Περιγραφές Ειδικών Εφέ

#### Ακούσιος Τραυματισμός

Ο αμυνόμενος εκτρέπει ή στρεβλώνει την επίθεση ενός αντιπάλου με τέτοιο τρόπο ώστε ο επιτιθέμενος να σκοντάφτει, τραυματίζοντας τον εαυτό του. Ο επιτιθέμενος πρέπει να ρίξει ζημιά εναντίον του εαυτού του σε μια τυχαία Τοποθεσία Πλήγματος χρησιμοποιώντας το όπλο που χρησιμοποίησε για να χτυπήσει. Εάν είναι άοπλος, ο επιτιθέμενος σκίζει ή σπάει κάτι εσωτερικό, η ρίψη ζημιάς αγνοεί οποιαδήποτε πανοπλία.

#### Ανάσταση

Επιτρέπει στον αμυνόμενο να χρησιμοποιήσει ένα στιγμιαίο άνοιγμα για να σηκωθεί ξανά στα πόδια του.

#### Μπάσιμο

Ο επιτιθέμενος χτυπά εσκεμμένα τον αντίπαλο εκτός ισορροπίας. Το πόσο μακριά ο αμυνόμενος παραπαίει προς τα πίσω ή προς τα πλάγια εξαρτάται από το όπλο που χρησιμοποιείται. Οι ασπίδες σπρώχνουν έναν αντίπαλο πίσω ένα μέτρο για κάθε 2 σημεία ζημιάς που ρίχνονται (πριν από τυχόν αφαιρέσεις λόγω πανοπλίας, αποκρούσεων και ούτω καθεξής), ενώ τα ροπαλοφόρα όπλα σπρώχνουν πίσω ένα μέτρο για κάθε τρία σημεία. Το μπάσιμο λειτουργεί μόνο σε πλάσματα έως και διπλάσιο του SIZ του επιτιθέμενου. Εάν ο αποδέκτης αναγκαστεί να οπισθοχωρήσει σε ένα εμπόδιο, τότε πρέπει να κάνει μια ρίψη δεξιότητας Ακροβατικών – ή Αθλητισμού σε Δύσκολη δυσκολία – για να αποφύγει την πτώση ή το σκάλωμα. Για τα πυροβόλα όπλα, δεδομένου ότι το Μπάσιμο βασίζεται περισσότερο στο να πιάνει έναν αντίπαλο εκτός ισορροπίας, παρά στο να τον πετάει προς τα πίσω με καθαρή δύναμη, είναι απολύτως λογικό οι εκρηκτικές σφαίρες να προκαλούν την παραπάτηση ενός στόχου εάν χτυπήσει ένα άκρο. Τα ενεργειακά όπλα είναι λιγότερο πιθανό να προκαλέσουν τέτοιο αποτέλεσμα, εκτός εάν η φυσική της ακτίνας οδηγήσει σε κάποιο είδος επιφανειακής έκρηξης.

#### Αιμορραγία

Η επίθεση κόβει ένα μεγάλο αιμοφόρο αγγείο. Εάν το χτύπημα υπερνικήσει τους Πόντους Θωράκισης και τραυματίσει τον στόχο, ο αμυνόμενος πρέπει να κάνει μια Αντιτιθέμενη Ρίψη Αντοχής εναντίον της αρχικής ρίψης επίθεσης. Εάν ο αμυνόμενος Αποτύχει, τότε αρχίζει να αιμορραγεί άφθονα. Στην αρχή κάθε Γύρου Μάχης, ο αποδέκτης χάνει ένα επίπεδο Κόπωσης, μέχρι να καταρρεύσει και πιθανώς να πεθάνει. Οι αιμορραγικές πληγές μπορούν να σταματήσουν με την επιτυχία μιας ρίψης δεξιότητας Πρώτων Βοηθειών, αλλά ο αποδέκτης δεν μπορεί πλέον να εκτελέσει καμία επίπονη ή βίαιη ενέργεια χωρίς να ξανανοίξει την πληγή. Δείτε Απώλεια Αίματος, στην ενότητα Σύστημα Παιχνιδιού. Για τα πυροβόλα όπλα, δεδομένου ότι τα περισσότερα πυροβόλα όπλα παράγουν πολύ στενές διαδρομές τραύματος, είναι στην πραγματικότητα πιο δύσκολο να πληγωθεί ή να κοπεί ένα μεγάλο αιμοφόρο αγγείο από ό,τι με ένα όπλο κοπής ή μαχαιρώματος. Λόγω αυτού, η Αιμορραγία θα πρέπει να είναι μόνο Κρίσιμη Επιτυχία – εκτός εάν το όπλο είναι κάποιο παράξενο όπλο που εκτοξεύει σούρικεν, ξυραφένια φλεξέτες, επίπεδα πεδία δύναμης ή κάποιου τέτοιου εξωτικού πυρομαχικού.

#### Τύφλωση Αντιπάλου

Σε Κρίσιμο ο αμυνόμενος τυφλώνει στιγμιαία τον αντίπαλό του πετώντας άμμο, αντανακλώντας το φως του ήλιου στην ασπίδα του, ή με κάποια άλλη τακτική που παρεμβαίνει στιγμιαία στην όραση του επιτιθέμενου. Ο επιτιθέμενος πρέπει να κάνει μια Αντιτιθέμενη Ρίψη της δεξιότητας Αποφυγής του (ή δεξιότητα Όπλου αν χρησιμοποιεί ασπίδα) ενάντια στην αρχική ρίψη Απόκρουσης του αμυνόμενου. Εάν ο επιτιθέμενος Αποτύχει, υποφέρει μια καταστασιακή τροποποίηση Δύσκολη ή Δύσβατη για τις επόμενες 1d3 Στροφές, ανάλογα με τη μέθοδο τύφλωσης.

#### Παράκαμψη Θωράκισης

Σε Κρίσιμο ο επιτιθέμενος βρίσκει ένα κενό στη φυσική ή φορετή πανοπλία του αμυνόμενου. Εάν ο αμυνόμενος φοράει πανοπλία πάνω από φυσική προστασία, τότε ο επιτιθέμενος πρέπει να αποφασίσει ποια από τις δύο παρακάμπτεται. Αυτό το εφέ μπορεί να στοιβάζεται για να παρακάμψει και τις δύο. Για τους σκοπούς αυτού του εφέ, η φυσική προστασία που αποκτάται από μαγεία θεωρείται ως φορετή πανοπλία. Αν και η Παράκαμψη Θωράκισης μπορεί φαινομενικά να χρησιμοποιηθεί με οποιοδήποτε είδος επίθεσης, ορισμένοι τύποι πυρομαχικών μπορεί είτε να εμποδίζονται από την πρόσβαση σε αυτό το εφέ (βλήματα κυνηγετικού όπλου λόγω της διασποράς τους) είτε να είναι ενσωματωμένοι στην βασική τους απόδοση, παρέχοντάς το ως μπόνους ειδικό εφέ επιπρόσθετα σε τυχόν άλλα που κερδίζονται (βολές τηλεμεταφοράς). Πολλά εξαρτώνται από το πώς ο Game Master επιθυμεί να μοντελοποιήσει το όπλο.

#### Επιλογή Τοποθεσίας

Όταν χρησιμοποιείτε όπλα κοντινής μάχης, ο επιτιθέμενος μπορεί να επιλέξει ελεύθερα την τοποθεσία όπου προσγειώνεται το χτύπημα, αρκεί αυτή η τοποθεσία να είναι κανονικά προσβάσιμη. Εάν χρησιμοποιείτε Όπλα Εμβέλειας, συμπεριλαμβανομένων των πυροβόλων όπλων, η Επιλογή Τοποθεσίας είναι μόνο Κρίσιμη Επιτυχία, εκτός εάν ο στόχος βρίσκεται σε Κοντινή Εμβέλεια και είναι είτε ακίνητος είτε ανυποψίαστος για τον επιτιθέμενο. Για τα πυροβόλα όπλα, οι τηλεσκοπικοί σκοπευτές μπορούν να επεκτείνουν αυτή την εμβέλεια.

#### Υπερκερασμός Απόκρουσης

Σε Κρίσιμο ο επιτιθέμενος μπορεί να παρακάμψει πλήρως μια κατά τα άλλα επιτυχημένη Απόκρουση.

#### Ζημιά Όπλου

Επιτρέπει στον χαρακτήρα να προκαλέσει ζημιά στο όπλο του αντιπάλου του ως μέρος μιας Επίθεσης ή Απόκρουσης. Εάν επιτίθεται, ο χαρακτήρας στοχεύει συγκεκριμένα το όπλο Απόκρουσης του αμυνόμενου και εφαρμόζει τη ρίψη ζημιάς του σε αυτό, αντί στον χειριστή. Το στοχευμένο όπλο χρησιμοποιεί τους δικούς του Πόντους Θωράκισης για να αντισταθεί στη ζημιά. Εάν μειωθεί στους μηδέν Πόντους Ζωής, το όπλο σπάει. Για τα πυροβόλα όπλα, δεδομένου ότι το χτύπημα ενός όπλου χειρός από απόσταση είναι ακόμη πιο δύσκολο από το να στοχεύσεις μια συγκεκριμένη θέση σώματος, αυτό θα πρέπει γενικά να προορίζεται μόνο για Κρίσιμες Επιτυχίες. Υπάρχουν όμως αρκετές εξαιρέσεις, η πρώτη είναι ότι μια ασπίδα, λόγω της φύσης της, θα πρέπει να είναι πολύ πιο εύκολο να χτυπηθεί. Η δεύτερη αφορά εξωτικά όπλα που εκτοξεύουν μια ακτίνα σε σχήμα κώνου ικανή να στοχεύει αντικείμενα χειρός εντός της περιοχής επίδρασής της· για παράδειγμα, ένα όπλο επαγωγής που θερμαίνει μεταλλικά αντικείμενα.

#### Αφοπλισμός Αντιπάλου

Ο χαρακτήρας χτυπά, τραβάει ή περιστρέφει το όπλο του αντιπάλου από το χέρι του. Ο αντίπαλος πρέπει να κάνει μια Αντιτιθέμενη Ρίψη του Στυλ Μάχης του εναντίον της αρχικής ρίψης του χαρακτήρα. Εάν ο αποδέκτης του αφοπλισμού χάσει, το όπλο του εκτοξεύεται σε απόσταση ίση με τη ρίψη του Τροποποιητή Ζημιάς του αφοπλιστή σε μέτρα. Εάν δεν υπάρχει Τροποποιητής Ζημιάς, τότε το όπλο πέφτει στα πόδια του αφοπλισμένου ατόμου. Το συγκριτικό μέγεθος των όπλων επηρεάζει τη ρίψη. Κάθε βήμα που το όπλο του αφοπλίζοντος χαρακτήρα είναι μεγαλύτερο αυξάνει τη δυσκολία της ρίψης του αντιπάλου κατά έναν βαθμό. Αντίθετα, κάθε βήμα που το όπλο του αφοπλίζοντος χαρακτήρα είναι μικρότερο καθιστά τη δυσκολία έναν βαθμό ευκολότερη. Ο αφοπλισμός λειτουργεί μόνο σε πλάσματα έως και διπλάσιο του STR του επιτιθέμενου.

#### Ρίψη Εχθρού

Επιθετικό, Μόνο Πυροβόλα Όπλα. Υποθέτοντας ότι ο στόχος υφίσταται τουλάχιστον ένα μικρό τραύμα από τον πυροβολισμό, αναγκάζεται να κάνει μια Αντιτιθέμενη Δοκιμασία Αντοχής ενάντια στην ρίψη χτυπήματος του επιτιθέμενου. Η αποτυχία υποδηλώνει ότι ο στόχος υποκύπτει σε σοκ και πόνο, καθιστάμενος ανίκανος και ανίκανος να συνεχίσει τη μάχη. Η ανάρρωση από την ανικανότητα μπορεί να πραγματοποιηθεί με έναν επιτυχημένο έλεγχο Πρώτων Βοηθειών ή χρησιμοποιώντας κάποια μορφή τεχνολογικού ή ναρκωτικού ενισχυτικού εάν υπάρχει τέτοιο στην εκστρατεία. Διαφορετικά, η προσωρινή ανικανότητα διαρκεί για μια περίοδο ίση με μία ώρα διαιρεμένη με τον Ρυθμό Θεραπείας του στόχου.

#### Υποχώρηση

Επιθετικό, Μόνο Πυροβόλα Όπλα. Αυτό το ειδικό εφέ επιτρέπει στον σκοπευτή να υποχωρήσει αμέσως στην κάλυψη, χωρίς να χρειάζεται να περιμένει την επόμενη σειρά του για να χρησιμοποιήσει την ενέργεια Κάλυψη. Ο χαρακτήρας πρέπει να στέκεται ή να σκύβει ήδη δίπλα σε κάποια μορφή κάλυψης για να χρησιμοποιήσει την Υποχώρηση.

#### Ενίσχυση Απόκρουσης

Σε Κρίσιμο ο αμυνόμενος καταφέρνει να εκτρέψει ολόκληρη τη δύναμη μιας επίθεσης, ανεξαρτήτως του Μεγέθους του όπλου του.

#### Εμπλοκή

Επιτρέπει σε έναν χαρακτήρα που κρατά ένα όπλο εμπλοκής, όπως ένα μαστίγιο ή ένα δίχτυ, να ακινητοποιήσει την τοποθεσία που χτυπήθηκε. Ένα εμπλεγμένο χέρι δεν μπορεί να χρησιμοποιήσει οτιδήποτε κρατάει. ένα παγιδευμένο πόδι εμποδίζει τον στόχο να κινηθεί. ενώ ένα μπλεγμένο κεφάλι, στήθος ή κοιλιά καθιστά όλες τις ρίψεις δεξιοτήτων ένα βαθμό δυσκολότερες. Στην επόμενη σειρά του, ο κάτοχος μπορεί να ξοδέψει έναν Πόντο Ενέργειας για να κάνει μια αυτόματη προσπάθεια Σκάλωμα Αντιπάλου. Ένα εμπλεγμένο θύμα μπορεί να επιχειρήσει να απελευθερωθεί στη σειρά του είτε επιχειρώντας μια Αντιτιθέμενη Ρίψη χρησιμοποιώντας Δύναμη για να τραβήξει ελεύθερα, είτε κερδίζοντας ένα Ειδικό Εφέ και επιλέγοντας Ζημιά Όπλου, Αφοπλισμός Αντιπάλου ή Απεγκλωβισμός.

#### Αναγκαστική Αποτυχία

Χρησιμοποιείται όταν ένας αντίπαλος Αδέξια, ο χαρακτήρας μπορεί να συνδυάσει την Αναγκαστική Αποτυχία με οποιοδήποτε άλλο Ειδικό Εφέ που απαιτεί Αντιτιθέμενη Ρίψη για να λειτουργήσει. Η Αναγκαστική Αποτυχία αναγκάζει τον αντίπαλο να αποτύχει τη ρίψη αντίστασής του εξ ορισμού – έτσι αυτομάτως αφοπλίζεται, σκοντάφτει, και ούτω καθεξής.

#### Κράτημα

Εφόσον ο αντίπαλος βρίσκεται εντός της εμβέλειας Άοπλης Μάχης του επιτιθέμενου, μπορεί να χρησιμοποιήσει ένα άδειο χέρι (ή παρόμοιο άκρο ικανό για κράτημα) για να κρατήσει τον αντίπαλο, εμποδίζοντάς τον να απεμπλακεί από τη μάχη. Ο αντίπαλος μπορεί να επιχειρήσει να απελευθερωθεί στη σειρά του, απαιτώντας μια Αντιτιθέμενη Ρίψη είτε Δύναμης είτε Άοπλης Μάχης ενάντια σε όποια από τις δύο δεξιότητες προτιμά ο κρατών.

#### Διαπέραση

Ρίξτε τη ζημιά του όπλου δύο φορές, με τον επιτιθέμενο να επιλέγει ποιο από τα δύο αποτελέσματα θα χρησιμοποιήσει για την επίθεση. Υποθέτοντας ότι το όπλο είναι ένα όπλο κοντινής μάχης, εάν η πανοπλία διαπεραστεί και προκληθεί πληγή, ο επιτιθέμενος έχει την επιλογή να αφήσει το όπλο στην πληγή ή να το τραβήξει ελεύθερο στην επόμενη σειρά του. Αφήνοντας το όπλο στην πληγή επιβάλλεται μια Βαθμίδα Δυσκολίας στις μελλοντικές προσπάθειες δεξιοτήτων του θύματος. Η σοβαρότητα της ποινής εξαρτάται από το μέγεθος τόσο του πλάσματος όσο και του όπλου που το διαπερνά, όπως αναφέρεται στον Πίνακα Επιδράσεων Διαπέρασης παρακάτω. Για λόγους απλότητας, περαιτέρω διαπεράσεις με το ίδιο μέγεθος όπλου δεν επιβάλλουν πρόσθετες ποινές. Για την απόσυρση ενός διαπερασμένου όπλου κατά τη διάρκεια μάχης σώμα με σώμα απαιτείται η χρήση της Ενέργειας Μάχης Προετοιμασία Όπλου. Ο χειριστής πρέπει να περάσει μια αναντίθετη ρίψη Δύναμης (ή να κερδίσει μια Αντιτιθέμενη ρίψη Δύναμης εάν ο αντίπαλος αντισταθεί). Η επιτυχία τραβάει το όπλο ελεύθερο, προκαλώντας περαιτέρω τραυματισμό στην ίδια τοποθεσία ίσο με το μισό της κανονικής ρίψης ζημιάς για αυτό το όπλο, αλλά χωρίς κανέναν Τροποποιητή Ζημιάς. Η αποτυχία συνεπάγεται ότι το όπλο παραμένει κολλημένο στην πληγή χωρίς περαιτέρω αποτέλεσμα, αν και ο χειριστής μπορεί να προσπαθήσει ξανά στην επόμενη σειρά του. Τα ειδικά αγκυλωτά όπλα (όπως οι λόγχες) προκαλούν κανονική ζημιά. Η πανοπλία δεν μειώνει τη ζημιά από την απόσυρση. Ενώ παραμένει διαπερασμένο, ο επιτιθέμενος δεν μπορεί να χρησιμοποιήσει το διαπεραστικό του όπλο για Απόκρουση.

| Κάθε +10 | Ακολουθήστε την πρόοδο του πίνακα |

Για τα πυροβόλα όπλα, τα πυρομαχικά των περισσότερων πυροβόλων όπλων με βάση τα βλήματα θα μπορούν να διαπεράσουν. Οι Game Masters μπορεί να θέλουν να περιορίσουν ή να απαγορεύσουν αυτό το εφέ για ενεργειακά όπλα με βάση το πλάσμα ή το λέιζερ. Οι σφαίρες, οι φλέττες και παρόμοια βλήματα είναι σαφώς πολύ μικρά για να επιβάλουν οποιαδήποτε δευτερεύοντα διαπεραστικά εφέ.

#### Σκοπευτής

Επιτρέπει στον σκοπευτή να μετακινήσει την Τοποθεσία Πλήγματος που χτυπήθηκε από τη βολή του κατά ένα βήμα, σε μια άμεσα γειτονική περιοχή του σώματος. Η φυσιολογία έχει επιρροή στο τι μπορεί να αναπροσανατολιστεί και πρέπει να εφαρμόζεται η κοινή λογική. Έτσι, χρησιμοποιώντας αυτό το Ειδικό Εφέ σε έναν ανθρωποειδή θα επέτρεπε σε έναν επιτιθέμενο που έριξε ένα χτύπημα στο πόδι, να το μετακινήσει στην κοιλιά αντ' αυτού. Αντίθετα, ο πυροβολισμός ενός γρύπα στο στήθος θα επέτρεπε την επιλογή των μπροστινών ποδιών, των φτερών ή του κεφαλιού.

#### Μεγιστοποίηση Ζημιάς

Σε Κρίσιμο ο χαρακτήρας μπορεί να αντικαταστήσει ένα από τα ζάρια ζημιάς του όπλου του με την πλήρη του αξία. Για παράδειγμα, ένα Τσεκούρι Χειρός που κανονικά προκαλεί 1d6 ζημιά θα αντιμετωπιζόταν αντ' αυτού ως 6, ενώ ένα μεγάλο ρόπαλο με 2d6 ζημιά θα προκαλούσε αντ' αυτού 1d6+6 ζημιά. Αυτό το Ειδικό Εφέ μπορεί να στοιβάζεται. Αν και μπορεί να χρησιμοποιηθεί και για φυσικά όπλα, η Μεγιστοποίηση Ζημιάς δεν επηρεάζει τον Τροποποιητή Ζημιάς του επιτιθέμενου, ο οποίος πρέπει να ριχθεί κανονικά.

#### Υπέρ-διείσδυση

Κρίσιμο, Μόνο Πυροβόλα Όπλα. Εάν πυροβολείτε σε γραμμικά τοποθετημένους αντιπάλους ή σε μια πυκνά συγκεντρωμένη ομάδα, αυτό το ειδικό εφέ επιτρέπει στη βολή να περάσει εντελώς μέσα από το πρώτο θύμα για να χτυπήσει ένα δεύτερο πίσω του, υποθέτοντας ότι υπερνικά την πανοπλία του πρώτου στόχου. Το δεύτερο θύμα, ωστόσο, υφίσταται μόνο τη μισή ζημιά λόγω εξασθένησης ή επιβράδυνσης της βολής. Η Υπέρ-διείσδυση είναι γενικά πιο χρήσιμη με όπλα υψηλής ισχύος που προκαλούν μεγάλες ποσότητες ζημιάς ή αυτά που έχουν κάποια ικανότητα διάτρησης πανοπλίας. Τυχόν ειδικά εφέ που επιβάλλονται στον πρώτο στόχο δεν εφαρμόζονται στον δεύτερο.

#### Καθήλωση

Μόνο Πυροβόλα Όπλα. Παρόμοιο με την Πίεση Πλεονεκτήματος, αυτό το ειδικό εφέ αναγκάζει τον στόχο να κάνει μια Αντιτιθέμενη Δοκιμασία της Δύναμης Θέλησής του εναντίον της ρίψης χτυπήματος του επιτιθέμενου. Η αποτυχία σημαίνει ότι ο στόχος κρύβεται πίσω από όποια κάλυψη είναι διαθέσιμη και δεν μπορεί να ανταποδώσει πυρά στην επόμενη σειρά του. Σημειώστε ότι η Καθήλωση λειτουργεί ακόμα κι αν δεν προκληθεί πραγματική ζημιά στον στόχο (ίσως λόγω επιτυχημένης αποφυγής ή χτυπημάτων που προσκρούουν στην κάλυψή του), καθώς βασίζεται στο εκφοβιστικό αποτέλεσμα των πυροβολισμών που περνούν πολύ κοντά.

Αν και ένα καθηλωμένο θύμα δεν μπορεί να ανταποδώσει πυρά για τον απαιτούμενο χρόνο, μπορεί να εκτελέσει άλλες ενέργειες υπό την προϋπόθεση ότι δεν εκθέτει τον εαυτό του σε πυρά, όπως το να συρθεί σε νέα κάλυψη, να επικοινωνήσει με άλλους, να επαναφορτώσει ένα όπλο και ούτω καθεξής.

#### Προετοιμασία Αντεπίθεσης

Ο αμυνόμενος διαβάζει τα μοτίβα του εχθρού του και προετοιμάζει μια αντεπίθεση εναντίον ενός συγκεκριμένου Ειδικού Εφέ (το οποίο θα πρέπει να σημειωθεί κρυφά). Εάν ο αντίπαλός του επιχειρήσει να του επιβάλει το επιλεγμένο Ειδικό Εφέ κατά τη διάρκεια της μάχης, ο αμυνόμενος αντικαθιστά αμέσως το εφέ του επιτιθέμενου με ένα δικό του, το οποίο επιτυγχάνει αυτόματα.

#### Γρήγορη Επαναφόρτωση

Όταν χρησιμοποιείτε ένα Όπλο Εμβέλειας, ο επιτιθέμενος μειώνει τον χρόνο επαναφόρτωσης για την επόμενη βολή κατά ένα. Αυτό το εφέ μπορεί να στοιβάζεται. Για τα πυροβόλα όπλα, ενώ είναι χρήσιμο για όπλα με μαύρη πυρίτιδα ή μονής βολής, αυτό το εφέ έχει λιγότερο εμφανή εφαρμογή όταν εφαρμόζεται σε αυτόματα όπλα, όπου η επόμενη βολή γεμίζεται (ή φορτίζεται) αμέσως. Ωστόσο, η Γρήγορη Επαναφόρτωση θα μπορούσε να χρησιμοποιηθεί αντ' αυτού για να αλλάξετε γρήγορα γεμιστήρες όπλων. Θα μπορούσε επίσης να εφαρμόζεται σε ενεργειακά όπλα που διαθέτουν μακροχρόνιους κύκλους για την επαναφόρτιση πυκνωτών ή την ψύξη της κάννης.

#### Ουλή στον Εχθρό

Ο μαχητής προκαλεί στον αντίπαλό του μια ουλή που θα είναι το θέμα συζήτησης όλων για το υπόλοιπο της ζωής του – όπως ένα σχεδόν κομμένο λαιμό, ή το γράμμα «Ζ» χαραγμένο με τέχνη στο στήθος.

#### Επιλογή Στόχου

Όταν ένας επιτιθέμενος Αδέξια, ο αμυνόμενος μπορεί να ελιχθεί ή να εκτρέψει το χτύπημα με τέτοιο τρόπο ώστε να χτυπήσει αντ' αυτού έναν παρακείμενο περαστικό. Αυτό απαιτεί ο νέος στόχος να βρίσκεται σε εμβέλεια του όπλου κοντινής μάχης του επιτιθέμενου, ή στην περίπτωση επίθεσης εμβέλειας, να στέκεται κατά μήκος της γραμμής βολής. Το νέο θύμα εκπλήσσεται εντελώς από το απρόσμενο ατύχημα και δεν έχει καμία ευκαιρία να αποφύγει την επίθεση που χτυπά αυτόματα. Σε αντάλλαγμα, ωστόσο, δεν υφίσταται κανένα Ειδικό Εφέ.

#### Απεγκλωβισμός

Σε Κρίσιμο ο αμυνόμενος μπορεί αυτόματα να ξεφύγει από το να είναι Εμπλεγμένος, Κρατημένος ή Καθηλωμένος.

#### Αναισθησία Τοποθεσίας

Ο επιτιθέμενος μπορεί να χρησιμοποιήσει ένα ροπαλοφόρο όπλο για να αναισθητοποιήσει προσωρινά το χτυπημένο σημείο του σώματος. Εάν το χτύπημα υπερνικήσει τους Πόντους Θωράκισης και τραυματίσει τον στόχο, ο αμυνόμενος πρέπει να κάνει μια Αντιτιθέμενη Ρίψη Αντοχής εναντίον της αρχικής ρίψης επίθεσης. Εάν ο αμυνόμενος Αποτύχει, τότε η Τοποθεσία Πλήγματος είναι Ανίκανος για ένα αριθμό Στροφών ίσο με τη ζημιά που προκλήθηκε. Ένα χτύπημα στον κορμό κάνει τον αμυνόμενο να παραπαίει λαχανιασμένος, ικανός μόνο να αμυνθεί. Ένα χτύπημα στο κεφάλι καθιστά τον εχθρό στιγμιαία ανεπίδεκτο αντίδρασης και ανίκανο να ενεργήσει.

#### Σπάσιμο

Ο επιτιθέμενος μπορεί να χρησιμοποιήσει ένα κατάλληλο όπλο, όπως ένα τσεκούρι, για να προκαλέσει ζημιά στην πανοπλία ή τη φυσική προστασία ενός αντιπάλου. Οποιαδήποτε ζημιά από όπλο, μετά τις μειώσεις για απόκρουση ή μαγεία, εφαρμόζεται στην τιμή των Πόντων Θωράκισης της προστασίας – σκίζοντας ιμάντες, σπάζοντας κρίκους, τσαλακώνοντας πλάκες ή σχίζοντας το δέρμα, τα λέπια ή το χιτίνη των τεράτων. Εάν παραμείνει κάποια ζημιά αφού η προστασία έχει μειωθεί σε μηδέν AP, μεταφέρεται στους Πόντους Ζωής της χτυπημένης τοποθεσίας. Για τα πυροβόλα όπλα, το Σπάσιμο θα ήταν κατάλληλο για ενεργειακά πυροβόλα όπλα σχεδιασμένα να καταστρέφουν θωρακίσεις αναπλήρωσης ή πιο εξωτικές έννοιες όπως όπλα βαρυτικής κύμα.

#### Σκάλωμα Αντιπάλου

Ο χαρακτήρας επιχειρεί να αποσταθεροποιήσει ή να ρίξει τον αντίπαλό του στο έδαφος. Ο αντίπαλος πρέπει να κάνει μια Αντιτιθέμενη Ρίψη της Δύναμης, της Αποφυγής ή των Ακροβατικών του εναντίον της αρχικής ρίψης του χαρακτήρα. Εάν ο στόχος αποτύχει, πέφτει πρηνής. Οι τετράποδοι αντίπαλοι (ή πλάσματα με ακόμα περισσότερα πόδια) μπορούν να αντικαταστήσουν τη δεξιότητα Αθλητισμού τους με την Αποφυγή και να αντιμετωπίσουν τη ρίψη ως ένα βαθμό δυσκολίας ευκολότερη. Για τα πυροβόλα όπλα, υποθέτοντας ότι το πυροβόλο όπλο εκτοξεύει πυρομαχικά με μάζα, δεν υπάρχει λόγος αυτό το εφέ να μην μπορεί να χρησιμοποιηθεί κατά την βολή στα πόδια ενός αντιπάλου, ακόμα κι αν δεν διεισδύσει καμία ζημιά σε καμία φορετή προστασία.

#### Δυσλειτουργία Όπλου

Ο Επιτιθέμενος Αδέξιος, Μόνο Πυροβόλα Όπλα. Το πυροβόλο όπλο του επιτιθέμενου δυσλειτουργεί με τέτοιο τρόπο ώστε να καθίσταται άχρηστο μέχρι να αφιερωθεί χρόνος για την αποσυναρμολόγηση και τον καθαρισμό του.

#### Αποχώρηση

Ο αμυνόμενος μπορεί αυτόματα να αποσυρθεί εκτός εμβέλειας, διακόπτοντας την εμπλοκή με αυτόν τον συγκεκριμένο αντίπαλο.

## Στενή Μάχη

Η στενή μάχη είναι η μάχη σώμα με σώμα με την κλασική της έννοια, όταν οι μαχητές χτυπούν ο ένας τον άλλο με χειριζόμενα ή φυσικά όπλα. Οι ακόλουθες οδηγίες προσφέρουν τρόπους για να γίνει η στενή μάχη πιο ενδιαφέρουσα και συμβάλλουν στην περαιτέρω επεξήγηση ορισμένων εννοιών που αναφέρθηκαν νωρίτερα σε αυτό το κεφάλαιο.

### Καταστασιακοί Τροποποιητές

Καταστασιακοί τροποποιητές μπορούν να εφαρμοστούν κατά τη μάχη σε συγκεκριμένες συνθήκες ή ως αποτέλεσμα ενός Ειδικού Εφέ. Για παράδειγμα, η μάχη στο σκοτάδι μπορεί να συνεπάγεται μια Βαθμίδα Δυσκολίας Ηράκλεια. Εκτός αν αναφέρεται διαφορετικά, οι τροποποιητές αποφασίζονται από τον Games Master. Όταν δύο ή περισσότερες καταστάσεις είναι σχετικές με τον χαρακτήρα, χρησιμοποιήστε την πιο σοβαρή.

Ορισμένοι τροποποιητές εξαρτώνται από τις περιστάσεις, αλλά αντί να εφαρμόζουν μια Βαθμίδα Δυσκολίας, η Δεξιότητα Μάχης περιορίζεται από μια σχετική δεξιότητα. Για παράδειγμα, η μάχη από την πλάτη ενός αλόγου θα περιόριζε τη Δεξιότητα Μάχης του μαχητή από τη δεξιότητα Ιππασίας του, ενώ η μάχη στο νερό θα μπορούσε να την περιορίσει στην τιμή της δεξιότητας Κολύμβησης του χαρακτήρα.

#### Καταστασιακοί Τροποποιητές Στενής Μάχης

| Κατάσταση | Βαθμίδα Δυσκολίας |
| :-- | :-: |
| Επίθεση σε ανήμπορο στόχο | Αυτόματη |
| Επίθεση σε περιορισμένη κατάσταση | Δύσκολη |
| Άμυνα ενώ βρίσκεστε σε χαμηλότερο έδαφος ή εναντίον έφιππου εχθρού | Δύσκολη |
| Μάχη σε ασταθές έδαφος | Δύσκολη |
| Μάχη ενώ σκύβετε ή από το ένα γόνατο | Δύσκολη |
| Μάχη σε κακή ορατότητα (πυκνή ομίχλη, χιονοθύελλα) | Δύσκολη |
| Άμυνα ενάντια σε επίθεση από πίσω | Δύσβατη |
| Μάχη ενώ είστε πρηνής | Δύσβατη |
| Μάχη σε μερική νύχτα (αμυδρός φωτισμός) | Δύσβατη |
| Μάχη σε απόλυτο σκοτάδι (καθόλου φωτισμός) | Ηράκλεια |
| Τυφλωμένος ή απώλεια πρωτογενούς αισθητικής αίσθησης | Ηράκλεια |

### Εμπλοκή

Ένας χαρακτήρας θεωρείται Εμπλεκόμενος εάν βρίσκεται εντός εμβέλειας όπλου κοντινής μάχης από έναν αντίπαλο. Αυτό δεν απαιτεί απαραίτητα να μπορούν και οι δύο μαχητές να αγγίξουν ο ένας τον άλλο· μόνο ότι ένας από αυτούς μπορεί να χτυπηθεί δυνητικά από τον άλλο. Μόλις ένας χαρακτήρας Εμπλακεί με έναν αντίπαλο, δεν μπορεί πλέον να αποχωρήσει ελεύθερα από αυτή τη μάχη (η έφοδος είναι μια ειδική εξαίρεση). Απαιτεί κάποια μορφή σκόπιμης πράξης για να διακόψει την επαφή με τον εχθρό, που συνήθως πραγματοποιείται μέσω μιας Ενέργειας Μάχης όπως ο Υπερκερασμός, ή η χρήση του Ειδικού Εφέ Αποχώρηση.

### Έφοδος

Η ενέργεια Έφοδος είναι η πράξη της γρήγορης κίνησης για την αύξηση της δύναμης και της επίδρασης μιας επίθεσης. Μια έφοδος απαιτεί από τον επιτιθέμενο να επιλέξει μια ενέργεια Έφοδος, κινούμενος είτε σε ρυθμό Τρεξίματος είτε σε Σπριντ - δείτε την ενότητα [Ρυθμός Κίνησης](0002_Characters.md?id=movement-rate) παραπάνω. Οι κανόνες κίνησης καλύπτονται πλήρως στο βιβλίο κανόνων του _Mythras_.

Εάν αυτό τους φέρει σε επαφή με τον αντίπαλό τους, πρέπει να σταματήσουν εκεί και να εκτελέσουν την ενέργεια Επίθεση, όπως τροποποιήθηκε για την έφοδο. Δεν υπάρχει ελάχιστη απόσταση που πρέπει να καλυφθεί πριν από μια έφοδο, καθώς η απόσταση που διανύθηκε είναι λιγότερο σημαντικός παράγοντας από την ορμή που επιτεύχθηκε. Εξ ου και το τρέξιμο ή το σπριντ.

Κατά τη διάρκεια της εφόδου, η ρίψη επίθεσης υποφέρει μια Βαθμίδα Δυσκολίας Δύσκολη. Σε αντάλλαγμα, η έφοδος αυξάνει το Μέγεθος του επιτιθέμενου όπλου κατά ένα βήμα και τον Τροποποιητή Ζημιάς ενός δίποδου επιτιθέμενου κατά ένα βήμα, ή δύο βήματα εάν είναι τετράποδο (ή περισσότερα πόδια). Οι χαρακτήρες που ιππεύουν μπορούν να αντικαταστήσουν τον Τροποποιητή Ζημιάς τους με αυτόν του οχήματός τους αντ' αυτού. Η τελική θέση του επιτιθέμενου χαρακτήρα εξαρτάται από το αποτέλεσμα της ανταλλαγής και από το αν ο επιτιθέμενος επιθυμεί να σταματήσει ή να συνεχίσει να περνάει, διακόπτοντας αμέσως την εμπλοκή.

Αντί να αποκρούσει ή να αποφύγει, ο αποδέκτης μιας εφόδου μπορεί ταυτόχρονα να αντεπιτεθεί στον επιτιθέμενο που κάνει έφοδο. Σε αυτή την περίπτωση, ο κάτοχος του μακρύτερου όπλου χτυπά πρώτος. Εάν το όπλο μπορεί να ρυθμιστεί για να δεχθεί μια έφοδο (όπως ένα δόρυ), ο αποδέκτης μπορεί να χρησιμοποιήσει τον Τροποποιητή Ζημιάς του επιτιθέμενου αντί του δικού του.

### Κάλυψη

Οι αντίπαλοι χρησιμοποιούν την κάλυψη για να εμποδίσουν τις επιθέσεις εναντίον τους, είτε μπλοκάροντας φυσικά ένα χτύπημα λόγω της σκληρότητας του ενδιάμεσου υλικού, είτε αποκρύπτοντας πού ακριβώς βρίσκεται ο εχθρός. Η ακριβής αξία της κάλυψης εξαρτάται από το πάχος της προστασίας της και την έκταση της κάλυψής της.

Μια επίθεση εναντίον στόχου που προσγειώνεται σε ένα ασαφές Σημείο Πλήγματος θα αποκλειστεί στο βαθμό της εγγενούς προστασίας της κάλυψης. Η κάλυψη μπορεί να εξουδετερωθεί εν μέρει είτε με τη χρήση του Ειδικού Εφέ Επιλογή Τοποθεσίας για να στοχεύσετε ορατά μέρη του στόχου είτε χτυπώντας μέσω της κάλυψης εάν το όπλο μπορεί να τη διαπεράσει.

Σε καταστάσεις όπου η κάλυψη είναι πλήρης, αλλά ο επιτιθέμενος πιστεύει ότι η επίθεσή του θα διεισδύσει, μπορεί να χτυπήσει τυφλά τον κρυμμένο στόχο. Σε τέτοιες περιπτώσεις, η ρίψη επίθεσης είναι μία βαθμίδα δυσκολότερη, και η Επιλογή Τοποθεσίας δεν μπορεί να γίνει εάν κερδηθεί ένα Ειδικό Εφέ. Αυτό υποθέτει ότι ο επιτιθέμενος γνωρίζει αρκετά ακριβώς τη θέση του στόχου πίσω από την κάλυψη.

### Αποφυγή

Η αποφυγή σε μάχη είναι το να ρίχνεις τον εαυτό σου μακριά από μια συντριπτική επίθεση. Αυτό απαιτεί μια Αντιτιθέμενη Ρίψη της δεξιότητας Αποφυγής του αμυνόμενου έναντι της σχετικής ρίψης του επιτιθέμενου. Αυτό θα μπορούσε να είναι οτιδήποτε, από τη δεξιότητα Οδήγησης ενός εμβολίζοντος άρματος μάχης μέχρι τη δεξιότητα εκτέλεσης ξορκιών ενός μάγου. Εάν ο επιτιθέμενος κερδίσει, τότε προκαλεί ζημιά όπως κανονικά. Εάν ο αμυνόμενος κερδίσει, η ζημιά αποφεύγεται πλήρως.

Εάν ο νικητής της Αντιτιθέμενης Ρίψης επιτύχει ένα ή περισσότερα Επίπεδα Επιτυχίας έναντι του αντιπάλου του, μπορεί να επιλέξει κατάλληλα Ειδικά Εφέ όπως σε κανονική μάχη. Όποιο και αν είναι το αποτέλεσμα, η αποφυγής κίνηση αφήνει τον αμυνόμενο πρηνή – συνήθως απαιτώντας από τον χαρακτήρα να ανακτήσει τη στήριξή του στην επόμενη σειρά του – εκτός εάν μπορεί να χρησιμοποιήσει το Ειδικό Εφέ Ανάσταση.

### Χτύπημα Πίσω

Μια επίθεση που προκαλεί περισσότερη ζημιά από το SIZ του αποδέκτη θα οδηγήσει, εξ ορισμού, στο να χτυπηθεί ο χαρακτήρας πίσω. Η ζημιά σε αυτή την περίπτωση είναι πριν από οποιαδήποτε μείωση λόγω απόκρουσης ή πανοπλίας. Κατά τη λήψη ενός τέτοιου χτυπήματος, ο αποδέκτης πρέπει να περάσει μια Εύκολη ρίψη Ακροβατικών ή μια Τυπική ρίψη Αθλητισμού για να αποφύγει την πτώση πρηνής. Επίσης σπρώχνεται ένα μέτρο για κάθε πέντε σημεία ζημιάς (ή κλάσμα αυτής) πάνω από το SIZ του. Η προληπτική χρήση της Ενέργειας Μάχης Στήριξη μπορεί να μειώσει ή ακόμα και να εξουδετερώσει τις επιπτώσεις του χτυπήματος πίσω.

### Επιθέσεις Άλματος

Οι επιθέσεις άλματος μπορούν να εκδηλωθούν από διάφορες καταστάσεις, οι περισσότερες ενεργοποιούνται ως μέρος μιας ενέδρας ή στο τέλος μιας εφόδου, αλλά ορισμένα πλάσματα μπορούν να πηδήξουν πάνω σε έναν αντίπαλο χωρίς την ανάγκη φόρας ή υπερυψωμένου σημείου.

Μια επίθεση άλματος επιλύεται με μια Αντιτιθέμενη Ρίψη της δεξιότητας Αθλητισμού του άλτη έναντι της δεξιότητας Δύναμης ή Αποφυγής του αμυνόμενου. Οι τετράποδοι στόχοι κάνουν τη ρίψη επίθεσης έναν Βαθμό Δυσκολίας δυσκολότερη. Εάν ο άλτης κερδίσει τότε ο αμυνόμενος πέφτει αυτόματα πρηνής, με τον επιτιθέμενο πάνω του. Η αποτυχία σημαίνει ότι ο αμυνόμενος έχει αντέξει ή έχει αποφύγει την πρόσκρουση. Εάν ο νικητής της Αντιτιθέμενης Ρίψης επιτύχει ένα ή περισσότερα Επίπεδα Επιτυχίας έναντι του αντιπάλου του, μπορεί να επιλέξει κατάλληλα Ειδικά Εφέ όπως σε κανονική μάχη.

Δεν προκαλείται ζημιά ως μέρος του άλματος· ωστόσο, ένα θύμα που βρίσκεται στη συνέχεια πρηνής δεν μπορεί να ανακτήσει τη στήριξή του έως ότου είτε ο επιτιθέμενός του αποχωρήσει, είτε κερδίσει τελικά ένα Ειδικό Εφέ που του επιτρέπει να Ανάσταση. Επιπλέον, οι επιθέσεις άλματος λειτουργούν μόνο εναντίον αντιπάλων έως και διπλάσιο του SIZ του επιτιθέμενου. Ένας στόχος μπορεί να καταστήσει τον εαυτό του πιο δύσκολο να χτυπηθεί κάτω με προληπτική χρήση της Ενέργειας Μάχης Στήριξη, υπό την προϋπόθεση ότι γνωρίζει την επικείμενη επίθεση άλματος.

### Υπερκερασμός

Ένας χαρακτήρας που αντιμετωπίζει πολλούς αντιπάλους μπορεί να χρησιμοποιήσει την κίνηση για να περιορίσει τον αριθμό που μπορεί να του επιτεθεί ανά πάσα στιγμή, μετατοπίζοντας συνεχώς τη θέση του, αναγκάζοντας τους εχθρούς να παρεμβαίνουν ο ένας στον άλλο. Ο Υπερκερασμός απαιτεί ο χαρακτήρας να έχει χώρο για να κινηθεί και να μην είναι καθηλωμένος σε περιορισμένο χώρο.

Ο υπερκερασμός απαιτεί ο χαρακτήρας να εμπλέξει τους αντιπάλους του σε μια ομαδική Αντιτιθέμενη Ρίψη δεξιοτήτων Αποφυγής. Κάθε συμμετέχων, τόσο ο χαρακτήρας που ελιχθεί όσο και οι εχθροί που επιθυμούν να τον παγιδεύσουν, πρέπει να ξοδέψουν έναν Πόντο Ενέργειας. Στη συνέχεια, ρίχνουν όλοι μία φορά και όσοι αποτυγχάνουν να ξεπεράσουν τη ρίψη του χαρακτήρα που ελιχθεί δεν μπορούν να του επιτεθούν για το υπόλοιπο αυτού του Γύρου Μάχης, όντας μπλοκαρισμένοι από τους συμμάχους τους ή τα χαρακτηριστικά του εδάφους.

Εάν ο χαρακτήρας που ελιχθεί νικήσει όλους τους αντιπάλους του, έχει την επιλογή είτε να εμπλακεί με ασφάλεια με έναν μόνο εχθρό για το υπόλοιπο του Γύρου είτε να αποχωρήσει εντελώς από τη μάχη.

### Παθητικό Μπλοκάρισμα

Το Παθητικό Μπλοκάρισμα επιτρέπει σε έναν χαρακτήρα οπλισμένο με όπλο να το κρατάει με τέτοιο τρόπο ώστε να καλύπτει μια επιλεγμένη τοποθεσία (ή τοποθεσίες όταν χρησιμοποιεί ασπίδα), αλλά με το κόστος να μην μπορεί να αποκρούσει ενεργά με αυτό (δείτε Φύλαξη Τοποθεσίας). Κάθε χτύπημα που προσγειώνεται στις τοποθεσίες λαμβάνει αυτόματα το όφελος του όπλου ή της ασπίδας, μειώνοντας τη ζημιά κανονικά. Η τεχνική χρησιμοποιείται συνήθως όταν ένας πολεμιστής επιθυμεί να φυλάξει ένα αδύναμο σημείο ή τραυματισμένη τοποθεσία. Τίποτα δεν εμποδίζει έναν μαχητή με διπλό όπλο, ή όπλο και ασπίδα, να χρησιμοποιήσει το άλλο του όπλο για να αποκρούσει ενεργά.

Επειδή το Παθητικό Μπλοκάρισμα λειτουργεί με τον ίδιο τρόπο όπως η κάλυψη, το Ειδικό Εφέ Επιλογή Τοποθεσίας δεν μπορεί να χρησιμοποιηθεί για να παρακάμψει το μπλοκάρισμα σε στενή μάχη, αν και οι αντίπαλοι μπορούν ακόμα να χρησιμοποιήσουν το εφέ για να χτυπήσουν άλλες μη μπλοκαρισμένες περιοχές. Το σκύψιμο πίσω από μια ασπίδα επιτρέπει σε έναν χαρακτήρα να διπλασιάσει τον αριθμό των καλυμμένων περιοχών κατά το Παθητικό Μπλοκάρισμα.

### Αιφνιδιασμός

Ο αιφνιδιασμός συμβαίνει όταν μια απροσδόκητη επίθεση εκδηλώνεται εναντίον αντιπάλων που αγνοούν την παρουσία ή την πρόθεση του επιτιθέμενου. Μια ενέδρα θα ήταν ένα παράδειγμα του πρώτου, ενώ η δόλια επίθεση σε έναν ανυποψίαστο σύμμαχο κατά τη διάρκεια φιλικής συνομιλίας απεικονίζει το δεύτερο.

Οι επιπτώσεις του αιφνιδιασμού σε έναν στόχο είναι ισχυρές:

- Ο στόχος υφίσταται ποινή –10 στην Πρωτοβουλία.
- Μέχρι να φτάσει η Πρωτοβουλία του, θεωρείται απροετοίμαστος και δεν μπορεί να αμυνθεί.
- Η πρώτη επίθεση στον στόχο, αν είναι επιτυχής, κερδίζει ένα μπόνους Ειδικό Εφέ.
- Για το υπόλοιπο του Γύρου, δεν μπορεί να εκτελέσει καμία επιθετική Ενέργεια.

### Επιθέσεις Σάρωσης

Οι επιθέσεις σάρωσης συμβαίνουν κατά την κρίση του Games Master, όταν όπλα ή πλάσματα ασυνήθιστου μεγέθους επιτίθενται σε μια στενά συγκεντρωμένη ομάδα αντιπάλων – η κοφτερή ουρά ενός τεράστιου δράκου ή η ασταμάτητη έφοδος ενός γιγάντιου τρικέρατου, για παράδειγμα – χτυπώντας πολλούς εχθρούς ταυτόχρονα.

Μια επίθεση σάρωσης γίνεται εφαρμόζοντας μια ενιαία ρίψη επίθεσης του όπλου ή του πλάσματος σε όλους τους στόχους στην πορεία του. Κάθε αμυνόμενος πρέπει να επιλύσει τα αποτελέσματα της επίθεσης ξεχωριστά και τυχόν Ειδικά Εφέ που επιβάλλονται στον επιτιθέμενο θεωρούνται ότι συνέβησαν ταυτόχρονα.

## Μάχη Εμβέλειας

Η μάχη εμβέλειας περιλαμβάνει όλες τις μορφές όπλων που απαιτούν βολή, ρίψη ή εκτόξευση για να χτυπήσουν τον στόχο τους. Οι επιθέσεις εμβέλειας επιλύονται με τον ίδιο τρόπο όπως η στενή μάχη. Ωστόσο, τα Όπλα Εμβέλειας μπορούν κανονικά να αποκρουστούν μόνο με ασπίδες· όσοι δεν έχουν πρέπει να βασίζονται σε φυσική κάλυψη ή να χρησιμοποιούν την Αποφυγή για να βουτήξουν έξω από τη γραμμή πυρός. Έτσι, εναντίον ελαφρά θωρακισμένων εχθρών, τα Όπλα Εμβέλειας μπορούν να είναι τρομερά αποτρεπτικά.

Κάθε Όπλο Εμβέλειας διαθέτει διάφορα συγκεκριμένα χαρακτηριστικά που καθορίζουν την αποτελεσματικότητά του. Αυτά που έχουν καθοριστική σημασία για τη μάχη είναι τα εξής:

- **Δύναμη:** Το ισοδύναμο του Μεγέθους για τα Όπλα Εμβέλειας. Είναι ένα αφηρημένο μέτρο της διατρητικής δύναμης ενός όπλου ή των πυρομαχικών του για να καθοριστεί αν το χτύπημα υπερνικά μια απόκρουση (ασπίδα).
- **Τροποποιητής Ζημιάς:** Αυτό το χαρακτηριστικό δείχνει αν ο Τροποποιητής Ζημιάς του επιτιθέμενου μπορεί να χρησιμοποιηθεί για να ενισχύσει τη ρίψη ζημιάς του Όπλου Εμβέλειας. Γενικά, μόνο τα τόξα που τραβιούνται με το χέρι και τα όπλα ρίψης επιτρέπουν στον χρήστη να εφαρμόσει τον Τροποποιητή Ζημιάς του.
- **Εμβέλεια:** Τρεις αριθμοί διαχωρισμένοι με κάθετες γραμμές, που αντιπροσωπεύουν τις μέγιστες Κοντινές, Αποτελεσματικές και Μακρινές εμβέλειες του όπλου ή των πυρομαχικών του. Η Κοντινή Εμβέλεια είναι η απόσταση κατά την οποία μπορεί να χρησιμοποιηθεί το Ειδικό Εφέ Επιλογή Τοποθεσίας, υπό την προϋπόθεση ότι ο στόχος είναι ακίνητος ή ανυποψίαστος για την επικείμενη επίθεση. Η Αποτελεσματική Εμβέλεια δεν έχει σημαντικούς τροποποιητές. Στη Μακρινή Εμβέλεια, το όπλο μπορεί ακόμα να προκαλέσει βλάβη, αλλά η ποσότητα ζημιάς μειώνεται στο μισό, και η Δύναμη μειώνεται κατά ένα βήμα.
- **Φορτίο:** Ο χρόνος σε Γύρους που απαιτείται για τη φόρτωση ή επαναφόρτωση ενός όπλου που εκτοξεύει πυρομαχικά. Ένας χαρακτήρας μπορεί να μειώσει τον χρόνο που αφιερώνεται στη φόρτωση ή την προετοιμασία με τη χρήση του Ειδικού Εφέ Γρήγορη Επαναφόρτωση.
- **Μέγεθος Διαπέρασης:** Το Μέγεθος ενός διαπερασμένου όπλου διαφέρει από τη Δύναμη με την οποία χτυπά. Τα όπλα διαπέρασης εμβέλειας έχουν μια ειδική στήλη που υποδηλώνει το πραγματικό τους μέγεθος όταν εξετάζονται τα εμπόδια που προκαλούνται από το Ειδικό Εφέ Διαπέραση.

### Καταστασιακοί Τροποποιητές

Όπως και στη στενή μάχη, μπορούν να εφαρμοστούν καταστασιακοί τροποποιητές όταν χρησιμοποιούνται Όπλα Εμβέλειας σε συγκεκριμένες συνθήκες. Η Δεξιότητα Μάχης ενός χαρακτήρα μπορεί να περιοριστεί όταν χρησιμοποιεί το όπλο σε δύσκολες συνθήκες, όπως η ρίψη ενός δόρατος ενώ ιππεύει ένα άλογο.

### Στόχευση

Ξοδεύοντας επιπλέον χρόνο για τη στόχευση ενός Όπλου Εμβέλειας, ένας χαρακτήρας μπορεί δυνητικά να αυξήσει τις πιθανότητες να χτυπήσει. Η στόχευση απαιτεί έναν ολόκληρο Γύρο Μάχης για τη σταθεροποίηση του όπλου και την αναμονή της καλύτερης ευκαιρίας για βολή, για παράδειγμα, την αναβολή μιας βολής τόξου για μια στιγμιαία παύση στον άνεμο ή μέχρι ένας στόχος να κινηθεί μεταξύ δύο εμποδίων. Με τη στόχευση, ο χαρακτήρας μπορεί να μειώσει τη δυσκολία ενός τροποποιητή Εμβέλειας ή Καταστασιακού τροποποιητή κατά ένα βαθμό. Πρόσθετοι Γύροι που δαπανώνται για στόχευση δεν παρέχουν περαιτέρω πλεονέκτημα.

### Αυτόματα Πυρά

Ορισμένα πυροβόλα όπλα γεμίζουν αυτόματα την επόμενη βολή, χρησιμοποιώντας τα απόβλητα αέρια, την ανάκρουση ή την ενέργεια από την προηγούμενη βολή. Χωρίς να χρειάζεται επαναοπλισμός, αυτό δίνει στο όπλο μια σειρά διαφορετικών ρυθμών βολής, όλα εξαρτώμενα από το πόσα πυρομαχικά επιθυμεί να εκτοξεύσει ο χρήστης.

Αυτοί οι τρόποι πυρός είναι γνωστοί ως Ημι-αυτόματος, Ριπή και Πλήρης-αυτόματος. Ενώ αυτή η ικανότητα μπορεί να κάνει ορισμένα όπλα ικανά να προκαλέσουν τρομερή ζημιά, η προσπάθεια διατήρησης της ακρίβειας με όπλα με ανάκρουση είναι προβληματική. Υποθέστε ότι οι ακόλουθοι κανόνες εφαρμόζονται κυρίως σε αυτόματα όπλα που εκτοξεύουν βλήματα.

#### Ημι-αυτόματος

Αυτός είναι στην πραγματικότητα ο προεπιλεγμένος ρυθμός πυρός για τα περισσότερα όπλα, όπου κάθε πάτημα της σκανδάλης εκτοξεύει μια μονή βολή και φορτώνει τον επόμενο γύρο.

#### Ριπή

Για πυρά ριπής, κάθε πάτημα της σκανδάλης εκτοξεύει ένα μικρό αριθμό βολών, συνήθως όχι περισσότερες από τρεις έως πέντε, ανάλογα με το όπλο. Ο αρχικός του σκοπός ήταν να επιτρέψει περιορισμένα κατασταλτικά πυρά, αλλά να περιορίσει την υπερβολική χρήση πυρομαχικών. Δεν προορίζεται για τη βελτίωση των πιθανοτήτων του σκοπευτή να χτυπήσει, καθώς η γρήγορη, διαδοχική ανάκρουση συνεχώς τραβάει το όπλο εκτός ευθείας. Τα πυρά σε λειτουργία ριπής επιβάλλουν μια ποινή δυσκολίας Δύσκολη στην ρίψη επίθεσης. Εάν όμως η επίθεση επιτύχει, ο σκοπευτής ρίχνει ένα ζάρι για να δει πόσες από τις βολές αυτής της ριπής χτύπησαν τον στόχο. Έτσι, μια ριπή τριών βολών θα χρησιμοποιούσε 1d3. Ρίξτε ξεχωριστή τοποθεσία και ζημιά για κάθε βολή που χτυπά.

#### Πλήρης-αυτόματος

Συνήθως χρησιμοποιείται για πλήρη κατασταλτικά πυρά για την καθήλωση πολλαπλών στόχων, η πλήρης-αυτόματη βολή μπορεί να είναι καταστροφική εάν εκτοξευθεί σε μια στενά ομαδοποιημένη ομάδα εχθρών. Ο σκοπευτής αποφασίζει πόσες βολές πυρομαχικών θα εκτοξεύσει (μέχρι το όριο του κυκλικού ρυθμού του όπλου) οι οποίες στη συνέχεια κατανέμονται ομοιόμορφα σε όλους τους στόχους που περιλαμβάνονται στο τόξο πυρός. Τυχόν εφεδρικές βολές θεωρούνται χαμένες κατά τη διάσχιση της στόχευσης (έτσι η εκτόξευση 15 βολών σε 6 στόχους θα ήταν δύο σφαίρες η καθεμία). Γίνεται ξεχωριστή ρίψη επίθεσης για κάθε στόχο, αλλά σε βαθμίδα δυσκολίας Δύσβατη λόγω των αυξημένων επιπτώσεων ανάκρουσης της συνεχούς βολής. Οι στόχοι που χτυπήθηκαν υφίστανται τυχαίο αριθμό βολών όπως στα πυρά ριπής.

Σημειώστε ότι μόνο η πρώτη βολή που χτυπά κατά τη διάρκεια μιας ριπής ή πλήρως αυτόματου ψεκασμού μπορεί να ωφεληθεί από ειδικά εφέ. Έτσι, μια κρίσιμη επίθεση κατά τη διάρκεια πυρός σε αυτόματη λειτουργία θα επέτρεπε την επιλογή Μεγιστοποίησης Ζημιάς, αλλά μόνο με την πρώτη βολή, ακόμα κι αν όλες οι υπόλοιπες βολές χτυπήσουν τον στόχο.

#### Γιατί η Αυτόματη Πυρά Τιμωρείται τόσο Σκληρά;

Μπορεί να φαίνεται ότι οι κανόνες για τα αυτόματα πυρά είναι άδικα σκληροί σε σύγκριση με την φαινομενική ακρίβεια των σύγχρονων πυροβόλων όπλων. Υπάρχουν πολλοί λόγοι για αυτό, με τους κυριότερους να είναι η απλοποίηση και η ισορροπία του παιχνιδιού. Οι παίκτες που επικρίνουν την ανακρίβεια του χαρακτήρα τους θα πρέπει να υπενθυμίζεται ότι τυχόν τροποποιήσεις στους κανόνες θα εφαρμοστούν και εναντίον τους, κάτι που δεδομένης της θανατηφόρας φύσης των ταχέων πυρών θα πρέπει να είναι μια νηφάλια σκέψη.

Φυσικά, ορισμένες από αυτές τις ποινές ανάκρουσης μπορούν να μετριαστούν, για παράδειγμα χρησιμοποιώντας Σταθεροποίηση Όπλου για να στηρίξετε αυτόματα πυροβόλα όπλα. Η αλήθεια είναι, όμως, ότι οι ανταλλαγές πυροβολισμών είναι πολύ διαφορετικές από τη σκοποβολή σε σκοπευτήριο. Το ένστικτο να καλυφθείς όταν πέφτουν σφαίρες, η ανάγκη να παραμείνεις κινητός και κάθε βολή που εντοπίζει τον σκοπευτή από τη λάμψη του ρύγχους ή τον ήχο, συχνά εμποδίζει κάθε ευκαιρία για στόχευση ή στήριξη. Η φράση «Ψέκασε και Προσευχήσου» επινοήθηκε για πολύ καλό λόγο!

### Διπλά Πυροβόλα Όπλα

Οι χαρακτήρες μπορούν να χρησιμοποιήσουν διπλά πυροβόλα όπλα, πυροβολώντας και τα δύο όπλα ταυτόχρονα κατά τη σειρά τους για να επιτύχουν ταχύτερο ρυθμό πυρός. Ωστόσο, αυτό συνεπάγεται ποινή στην ακρίβεια. Ο πυροβολισμός με ένα πιστόλι σε κάθε χέρι υποφέρει ποινή Δύσκολη στο χτύπημα του στόχου. Όσοι προσπαθούν να πυροβολήσουν ταυτόχρονα με ένα καραμπίνα ή ένα SMG σε κάθε χέρι υποφέρουν ποινή Δύσβατη.

### Πυροβολισμοί σε Πλήθος

Ο πυροβολισμός στην αναταραχή μιας μάχης σώμα με σώμα μπορεί να είναι μια επικίνδυνη υπόθεση, επομένως υπάρχει πάντα ο κίνδυνος να χτυπήσει κανείς κατά λάθος κάποιον άλλο εκτός από τον αρχικό στόχο. Όταν πυροβολεί κανείς έναν συγκεκριμένο στόχο στην άκρη ενός πλήθους ή μάχης, η ρίψη επίθεσης υποφέρει μια Βαθμίδα Δυσκολίας Δύσκολη. Εάν προσπαθεί κανείς να πυροβολήσει μέσα από το πλήθος, η δυσκολία θα πρέπει να αυξηθεί σε Δύσβατη. Η στόχευση επιτρέπεται ακόμα για να μετριαστεί ο κίνδυνος.

Ένας σκοπευτής που περνά τη ρίψη επίθεσής του παρά την ποινή δυσκολίας έχει στοχεύσει σωστά και θα χτυπήσει τον επιδιωκόμενο στόχο του. Εάν, ωστόσο, ο σκοπευτής αποτύχει στη ρίψη, αλλά θα είχε επιτύχει αν δεν υπήρχε η ποινή «πυροβολισμοί σε πλήθος», τότε χτυπιέται ένα παρακείμενο θύμα αντ' αυτού (το οποίο είναι ελεύθερο να Αποφύγει ή να Αποκρούσει κανονικά).

Εάν περισσότεροι από ένας στόχοι βρίσκονται στη γραμμή πυρός, ο Games Master θα πρέπει να καθορίσει τυχαία το θύμα. Οποιαδήποτε Ειδικά Εφέ που κερδίζονται ως μέρος της επίθεσης εφαρμόζονται μόνο στον αρχικό στόχο, όχι σε κανέναν τυχαία χτυπημένο περαστικό.

### Αδέξια Χειρισμός Πυροβόλων Όπλων

Δεδομένου ότι τα πυρά όπλων δεν αντιστέκονται συνήθως από μια αντίπαλη δεξιότητα, θα πρέπει να γίνει ειδική πρόβλεψη όταν ένας σκοπευτής αδέξια στην ρίψη επίθεσής του. Σε αυτή την περίπτωση, όποιος ήταν ο επιδιωκόμενος στόχος επιτρέπεται να επιλέξει ένα ειδικό εφέ συγκεκριμένο για αδέξια κίνηση, το οποίο στη συνέχεια υφίσταται ο σκοπευτής. Δείτε επίσης το ειδικό εφέ μόνο για αδέξια κίνηση - Δυσλειτουργία Όπλου.

### Απόκρουση Πυρών Όπλων με Ασπίδες

Όπως εκφράστηκε νωρίτερα, είναι σχεδόν αδύνατο να αποκρούσει κανείς σκόπιμα ένα υπερηχητικό βλήμα ή μια ενεργειακή δέσμη. Οι ασπίδες, ωστόσο, παρέχουν έναν βαθμό προστασίας, καθώς στην πραγματικότητα είναι φορητές «καλύψεις». Για να χρησιμοποιήσετε μια ασπίδα σε ανταλλαγή πυρών, απλώς χρησιμοποιήστε τους κανόνες Παθητικού Μπλοκαρίσματος από την ενότητα [Στενή Μάχη](0005_Combat.md?id=passive-blocking), παραπάνω.

Όσον αφορά τα πιο σύγχρονα πυροβόλα όπλα ή αυτά της επιστημονικής φαντασίας, ένας Game Master μπορεί να επιθυμεί να δώσει στην ασπίδα μια τιμή Πόντων Θωράκισης αντί να της αναθέσει ένα Μέγεθος, με βάση το εγγενές υλικό από το οποίο είναι κατασκευασμένη. Για παράδειγμα, μια ξύλινη ασπίδα μπορεί να αντιμετωπιστεί ως 4 Πόντους Θωράκισης, μια τακτική ασπίδα Kevlar 8 AP, και μια ασπίδα πεδίου δύναμης βραχιολιού Οδυσσέα 31 15 AP.

### Τηλεσκοπικά Σκόπευτρα

Ένα τηλεσκοπικό σκόπευτρο βελτιώνει την ακρίβεια ενός πυροβόλου όπλου σε μεγαλύτερες αποστάσεις όταν ο χρήστης αφιερώνει χρόνο για να στοχεύσει σωστά (βλ. Στόχευση). Τα πιο βασικά τηλεσκοπικά σκόπευτρα είναι απλώς οπτικοί μεγεθυντές, αλλά καθώς η τεχνολογία προχωρά, αρχίζουν να ενσωματώνουν ενσωματωμένους αποστασιομετρητές, υπολογιστές τροχιών βλημάτων, αισθητήρες κίνησης αέρα, αντισταθμιστές ατμοσφαιρικής φακού, ενισχυτές υπερύθρων ή χαμηλού φωτισμού και μια πληθώρα άλλων gadgets που βελτιώνουν την αποτελεσματική τους απόσταση «χτυπήματος».

Τα τηλεσκοπικά σκόπευτρα διευκολύνουν τις ρίψεις δεξιότητας κατά μία ή περισσότερες βαθμίδες (δείτε [Τροποποίηση Δεξιοτήτων](0003_Skills.md?id=modifying-skills)), ανάλογα με την πολυπλοκότητα του σκοπεύτρου. Κανένα σκόπευτρο δεν μπορεί να κάνει μια ρίψη δεξιότητας Αυτόματη επιτυχία, οπότε η καλύτερη βαθμίδα που μπορεί να επιτευχθεί είναι Πολύ Εύκολη.
