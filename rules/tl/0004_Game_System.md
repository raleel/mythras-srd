# Sistema ng Laro

Maraming iba't ibang bagay ang maaaring mangyari sa isang laro ng _Mythras Imperative_. Ang ilan ay nalulutas sa pamamagitan ng paggamit ng kasanayan o isa sa mga pangunahing mekanika ng laro tulad ng labanan o mahika; ang iba ay nangangailangan ng sarili nilang mga patakaran para sa paghusga. Ang kabanatang ito ay nagbibigay ng mga patakaran at gabay para sa iba't ibang sitwasyon na makakaharap ng mga manlalaro habang nilalaro ang laro.

Ang listahan ay hindi kumpleto ngunit dapat kumatawan sa mga pinakakaraniwang sitwasyon. Kung makita ng mga Game Master na may iba pang sitwasyon na nangangailangan ng ilang anyo ng paghusga ng sistema ng laro, at kung ang mga sitwasyong ito ay tila madalas mangyari, hinihikayat silang bumuo ng sarili nilang mga patakaran upang masakop ang mga espesyal na pangyayaring ito, gamit ang mga ibinigay sa kabanatang ito bilang batayan para sa replikasyon o pagpapaunlad.

## Asphyxiation (Pagkaubusan ng Hininga)

Ang mga karakter ay maaaring magpigil ng hininga sa loob ng ilang segundo na katumbas ng kanilang kasanayan sa Endurance. Gayunpaman, ang karakter ay dapat handa (pinupuno ang baga ng pinakamaraming hangin hangga't maaari); kung hindi, ang panahon ay hinahati kung ang karakter ay nasa isang pasibong sitwasyon, o nabawasan sa isang-kalima kung ang karakter ay nakikibahagi sa mabigat na aktibidad.

Kapag tapos na ang panahon ng pagpigil ng hininga, ang mga karakter ay dapat gumulong ng Endurance:

- Kung ang gulong ay isang Critical Success, walang karagdagang pagkasira ang mangyayari.
- Kung ang gulong ay Successful, ang karakter ay magkakaroon ng karagdagang antas ng Fatigue.
- Kung ang gulong ay Fails, ang karakter ay magkakaroon ng 1d2 karagdagang antas ng Fatigue sa Round na iyon.
- Kung ang gulong ay Fumbled, ang karakter ay magkakaroon ng 1d3 karagdagang antas ng Fatigue sa Round na iyon.

Kung walang tulong, ang kamatayan mula sa Asphyxiation ay karaniwang mabilis. Kung ang Asphyxiation ay matatapos bago mamatay ang karakter, mabilis silang nakakabawi ng mga antas ng Fatigue na nawala dahil sa pagkaubusan ng hininga, nakakabawi ng isang antas bawat minuto.

Depende sa paraan ng Asphyxiation, maaaring naisin ng Game Master na pigilan ang ganap na paggaling, nagpapataw ng isang pangmatagalang antas ng Fatigue (sumangguni sa seksyon tungkol sa [Fatigue](0004_Game_System.md?id=fatigue)) upang kumatawan sa pinsala sa baga na dulot ng usok o paglanghap ng tubig.

## Pagpapahusay ng Karakter

Bawat karakter sa _Mythras Imperative_ ay maaaring bumuti sa paglipas ng panahon at karanasan. Ang pagpapabuti ay maaaring gawin sa ilang mga lugar:

- Pagpapataas ng umiiral na mga kasanayan
- Pagpapataas ng mga Passion
- Pag-aaral ng mga bagong kasanayan
- Pag-aaral ng mga bagong Abilidad

#### Mga Paggulong ng Karanasan

Ang mekanismo para sa karamihan ng pagpapabuti ng karakter ay ang Paggulong ng Karanasan. Ang mga Game Master ay nagbibigay ng mga Paggulong ng Karanasan sa isang angkop na punto sa kampanya: sa dulo ng bawat matagumpay na senaryo o storyline; o pagkatapos ng marahil dalawa o tatlong sesyon ng laro kung ang kuwento ay mahaba at magtatagal upang makumpleto. Ang dalas ay nasa diskresyon ng Game Master. Ang mataas na dalas ng mga Paggulong ng Karanasan ay magdudulot ng mas mabilis na pag-unlad ng mga karakter.

Walang tama o maling oras upang magbigay ng mga Paggulong ng Karanasan, ngunit ang mga natural na pahinga sa kuwento ay maaaring magmungkahi ng angkop na mga oras. Ang Game Master, gayunpaman, ay palaging ang gumagawa ng desisyon dito. Bagaman hindi maaaring humingi ng mga Paggulong ng Karanasan ang mga manlalaro, may karapatan silang umasa sa mga ito sa ilang mga punto – dahil hindi makakabuo ang kanilang mga karakter kung wala ang mga ito.

Ang bilang ng mga Paggulong ng Karanasan na iginawad ay tinutukoy din ng Game Master. Gayunpaman, isang magandang panuntunan ay 1-3 Paggulong ng Karanasan bawat sesyon mula noong huling beses na sila ay iginawad.

#### Pagpapataas ng Umiiral na mga Kasanayan

Anumang kasanayan sa Sheet ng Karakter, Standard o Professional, ay maaaring pataasin sa pamamagitan ng paggastos ng isang Paggulong ng Karanasan. Maaaring piliin ng mga manlalaro na huwag gumastos ng mga Paggulong ng Karanasan sa pagpapataas ng mga kasanayan, sa halip ay mag-ipon para gamitin sa pagpapataas ng iba pang bagay – tulad ng mga abilidad ng Klase (tingnan sa ibaba).

Ang manlalaro ay gumugulong ng 1d100 at inihahambing ito sa kasanayang pinapataas. Ang INT ng karakter ay idinagdag sa gulong.

- Kung ang numerong ginulong ay katumbas o mas malaki kaysa sa kasanayang pinapabuti, ito ay tataas ng 1d4+1%.
- Kung ang numerong ginulong ay mas mababa kaysa sa napiling kasanayan, tataas pa rin ang kasanayan, ngunit ng 1% lamang.
- Kung ang isang karakter ay nag-Fumbled ng anumang kasanayan sa naunang sesyon(s) – i.e., sa pagitan ng huling hanay ng mga Paggulong ng Karanasan at ng kasalukuyan – ang Fumbled na kasanayan ay makakakuha ng libreng pagtaas ng 1%. Mas marami tayong natututunan mula sa ating mga pagkakamali kaysa sa ating mga tagumpay at ito ay kumakatawan sa pagmumuni-muni na dinadaanan ng isang karakter pagkatapos ng isang malaking kabiguan. Ang maraming Fumbles ng parehong kasanayan ay hindi nagsasama-sama. Sa isang kaso kung saan ang Fumbled na kasanayan ay pinapataas din sa isang Paggulong ng Karanasan, ang 1% mula sa Fumble ay inilalapat bago gawin ang gulong upang mapabuti ito.

Ang parehong kasanayan ay hindi maaaring makinabang mula sa higit sa isang Paggulong ng Karanasan bawat sesyon.

#### Pagpapataas ng mga Passion

Tulad ng inilarawan sa ilalim ng Passions sa seksyon ng [Passions](0002_Characters.md?id=step-8-passions), ang halaga ng isang Passion ay maaaring pataasin gamit ang mga Paggulong ng Karanasan sa parehong paraan ng isang kasanayan. Kung sinusuportahan ng paglalaro, maaaring pahintulutan pa ng Game Master na bawasan ang isang Passion gamit ang isang Paggulong ng Karanasan.

_Halimbawa, ang patuloy na pagtataboy mula sa isang hindi natupad na pag-ibig ay maaaring magpababa ng Passion ng kahit na ang pinakamasigasig na mangingibig, lalo na ang isang nagnanais na ituloy ang isang relasyon nang walang mga multo ng nakaraan na gumagambala sa kanila._

#### Pag-aaral ng mga Bagong Kasanayan

Ang ilang mga karakter ay maaaring magnanais na mag-aral ng mga bagong Professional Skills na hindi nila kailanman nagkaroon ng pagkakataong matutunan mula sa kanilang kultura. Bago sila magsimulang maglaan ng mga Paggulong ng Karanasan, kailangan muna nilang makahanap ng pinagmulan ng kaalaman upang matuto. Maaari itong maging kasing prosaic ng isang propesyonal na tutor tulad ng ibinibigay ng mga orden o guild. Bilang kahalili, maaari itong maging isang mas kakaibang pinagmulan ng edukasyon tulad ng isang sinauna at nagkakawasak na training scroll.

Kapag nahanap na ang pinagmulan ng edukasyon, ang karakter ay dapat gumugol ng isang buong buwan ng pag-aaral at pagsasanay upang makakuha ng batayang kaalaman sa kasanayang iyon (matuto sa Base Level). Nagkakahalaga ito ng 3 Paggulong ng Karanasan kasama ang anumang in-game na gastos na kinakailangan upang bayaran ang guro (kung mayroon) at bilhin (o rentahan) ang anumang kagamitan at tool na maaaring kailanganin.

## Sakit at Lason

Ang ilan sa mga pinakakinatatakutan na bagay na maaaring makaapekto sa mga karakter ay ang napakaraming sakit at lason na laganap sa madilim at malagim na mga lugar ng mundo. Walang gaanong nakakagulo kaysa sa isang salot na tumatama sa bayan ng karakter, lalo na kung sila ay nasa loob pa rin ng mga pintuan nito. Gayundin, ang paghahanap ng isang makamandag na ahas sa kama o pag-atake ng mga higanteng alakdan ay maaaring maging mga sandali ng matinding drama.

Ang mga sakit at lason ay halos walang hanggan sa kanilang mga uri ngunit, para sa layunin ng laro, ay pinangangasiwaan sa parehong paraan.

#### Mga Katangian ng Sakit at Lason

Ang lahat ng sakit at lason ay nagpapakita ng ilang mga katangian na mahalaga sa kanilang mga epekto.

**Aplikasyon:** Ang paraan kung paano ipinakilala ang sakit o lason sa biktima.

- **Pagkain/Pag-inom:** Ang mga epekto ay nangyayari sa pamamagitan ng pagkain o pag-inom.
- **Paglanghap:** Ang mga epekto ay nangyayari sa pamamagitan ng paghinga o pagsinghot.
- **Direktang Pagdikit:** Ang mga epekto ay nangyayari sa pamamagitan ng pag-absorb ng balat.
- **Iniksyon:** Ang mga epekto ay nangyayari kapag ang substansya ay ininjeksyon sa katawan sa pamamagitan ng pagbutas.

**Lakas (Potency):** Ang kalubhaan ng sakit o lason. Ang halagang ito ay itinakda laban sa isang angkop na resisting skill (karaniwang Endurance o Willpower) sa isang Opposed Roll. Kung mananalo ang karakter sa gulong, mawawala ang mga epekto nito. Kung matatalo sila sa gulong, mararanasan nila ang lahat ng Kondisyon ng sakit o lason, bawat isa sa angkop na oras.

**Paglaban (Resistance):** Kung paano nilalabanan ang sakit o lason – karaniwang alinman sa Endurance o Willpower, ngunit maaaring may mas kakaibang kinakailangan sa paglaban. Maaaring i-roll ang Resistance sa oras ng pagkakalantad o ipagpaliban hanggang sa Onset Time upang itago ang katotohanan na maaaring nahawahan o nalason ang karakter.

**Oras ng Paglabas (Onset Time):** Maraming sakit at lason ang hindi agad nagkakaroon ng epekto. Ang pagkaantala ay tinatawag na Onset Time, at ito ay maaaring ilang segundo, minuto, oras, o mas matagal pa. Ang mga lason o sakit na may maraming epekto ay maaaring magkaroon ng iba't ibang onset time para sa bawat isa tulad ng inilarawan sa ilalim ng mga Kondisyon.

**Tagal (Duration):** Gaano katagal tumatagal ang mga Kondisyon ng isang sakit o lason.

**Mga Kondisyon:** Bawat toxin ay may isa o higit pang Kondisyon. Ang mga ito ay may tiyak na epekto tulad ng inilarawan sa talahanayan ng Kondisyon sa ibaba, at kung ang biktima ay nabigo na lumaban, siya ay naghihirap sa lahat ng inilarawang Kondisyon.

**Panlunas/Gamot (Antidote/Cure):** Kung ang toxin ay maaaring gamutin, ito ay mapapansin dito. Ang matagumpay na paggamot ay pumipigil sa anumang karagdagang kondisyon na mangyari, ngunit sa ilang pagkakataon ay maaaring mag-iwan sa biktima na naghihirap sa pangmatagalang epekto na naidulot na.

##### Talahanayan ng mga Kondisyon

| Kondisyon | Epekto |
| :-- | :-- |
| Matinding Sakit (Agony) | Ang biktima ay nahihirapan sa matinding sakit. Kung sa isang lokasyon o sa buong katawan, anumang skill roll na kinasasangkutan ng paggamit ng apektadong bahagi ay dapat ding mas mababa o katumbas ng Willpower ng karakter, kung hindi man ay mabibigo ang pagtatangka, at siya ay dadaing o sisigaw sa sakit. |
| Pagkaubusan ng Hininga (Asphyxiation) | Ang biktima ay naghihirap sa pagkaubusan ng hininga – siya ay babagsak nang walang malay, hindi makahinga. Ang mga patakaran mula sa seksyon ng [Asphyxiation](0004_Game_System.md?id=asphyxiation) ay ginagamit. Ang Asphyxiation ay maaaring asthmatic sa kalikasan na nangangahulugang ang biktima ay naghihirap lamang sa paghinga, o kumpletong respiratory failure na nagreresulta sa kamatayan. Sa huling kaso, ang isang biktima ay maaaring panatilihing buhay sa pamamagitan ng pagwawagi sa isang opposed test ng kasanayan sa First Aid laban sa Potency ng sakit o lason. |

| Pagduduwal (Nausea) | Ang biktima ay hindi makakain, at dapat gumulong laban sa kanyang Endurance sa bawat pagkakataon na siya ay gumaganap ng isang nakakapagod na pisikal na aksyon upang maiwasan ang pisikal na pagsusuka. Ang pagsusuka ay tumatagal ng 1d3 rounds kung saan hindi siya makakilos. Ang mahabang tagal ng Nausea ay maaaring magdulot ng gutom. |

#### Halimbawang Sakit: Red Pox

Ang Red pox ay isang mapaminsalang sakit na kumakalat sa pamamagitan ng pagdikit sa balat ng isang taong nahawahan na, o sa isang bagay na kanilang nahawakan. Ang tsansa na mahawa ng sakit mula sa isang bagay ay nananatili sa loob ng 1d3 araw pagkatapos ng pagdikit, na nagpapahirap na pigilan ang sakit kapag nangyari ang isang pagkalat.

Sa simula, ang mga biktima ay nagdurusa ng isang maliit na pantal, ngunit sa loob ng isang araw ay magkakaroon sila ng matinding lagnat na karaniwang nagpapahiga sa kanila sa buong tagal. Sa panahong ito, ang pantal ay nagiging nakakatakot na makating mga paltos na pumutok sa pinakamaliit na paggalaw, at naglalabas ng maliliit na patak ng dugo – lalo na mula sa kilikili at singit. Ang yugtong ito ay maaaring tumagal ng hanggang isang linggo o higit pa, na posibleng maging sanhi ng kamatayan.

- **Aplikasyon:** Direktang Pagdikit (Contact)
- **Lakas (Potency):** 50
- **Paglaban (Resistance):** Endurance
- **Oras ng Paglabas (Onset time):** 1d6+6 oras
- **Tagal (Duration):** 1d6+3 araw
- **Mga Kondisyon:** Kapag nagpakita ang sakit, ang biktima ay nagiging Nakakahawa (Contagious). Pagkalipas ng 1 araw, magsisimula silang magkaroon ng Lagnat (Fever) at Pagdurugo (Bleeding). Bawat araw na nagdurugo ang mga paltos, ang biktima ay nawawalan ng isang antas ng Fatigue, na hindi gumagaling hanggang sa matapos ang tagal ng sakit.
- **Panlunas/Gamot (Antidote/Cure):** Maaaring gamutin ng kasanayan sa Healing.

## Pagkahulog

Ang dami ng pinsala na natamo sa pagkahulog ay depende sa distansya ng pagbagsak. Hindi binabawasan ng Armor Points ang pinsala sa pagkahulog, ngunit ang Damage Modifier ng nahuhulog na karakter o nilalang ay nalalapat sa pinsala na natamo mula sa pagkahulog.

##### Talahanayan ng Distansya ng Pagkahulog

| Distansya ng Pagkahulog | Pinsalang Natamo |
| :-- | :-- |
| 1 metro o mas mababa | Walang pinsala. |
| 2-5 metro | 1d6 puntos ng pinsala sa isang random na lokasyon. |
| 6-10 metro | 2d6 puntos ng pinsala sa dalawang random na lokasyon. |
| 11-15 metro | 3d6 puntos ng pinsala sa tatlong random na lokasyon. |
| 16-20 metro | 4d6 puntos ng pinsala sa apat na random na lokasyon. |
| Bawat +5 metro | +1d6 pinsala. |

#### Mga Nahuhulog na Bagay

Ang isang nahuhulog na bagay ay nagdudulot ng pinsala batay sa SIZ nito at sa distansya ng pagkahulog. Ang isang bagay ay nagdudulot ng 1d6 pinsala para sa bawat 6 puntos ng SIZ (o bahagi nito), kasama ang dami ng pinsala na katumbas ng kolum ng Damage Taken ng Talahanayan ng Distansya ng Pagkahulog – kasama ang anumang pagbabawas para sa mga bagay na mas maliit ang sukat.

## Pagod (Fatigue)

Sinusukat ng Fatigue ang pagod at ang mga incremental na epekto nito. Ginagamit ito upang subaybayan ang maraming iba't ibang bagay, mula sa matinding aktibidad hanggang sa nakakapanghinang epekto ng sakit.

Ang pangunahing paraan ng pagkakaroon ng Fatigue ay sa pamamagitan ng pakikipag-ugnayan sa ilang anyo ng pisikal na aktibidad. Kung mas matindi ang ehersisyo o trabaho, mas mabilis nitong pinapagod ang karakter. Sa diskresyon ng Game Master, ang mga karakter ay dapat gumawa ng angkop na skill roll – alinman sa Athletics, Brawn, o Endurance – upang labanan ang pagkakaroon ng antas ng Fatigue.

Ang bawat Failed roll ay nagdudulot ng isang antas ng Fatigue. Ang bawat antas ng Fatigue ay may mga parusa para sa paggamit ng kasanayan, paggalaw, Initiative, at Action Points. Ang Asphyxiation, Blood Loss, at iba pang epekto na nakabatay sa setting ay nag-aambag din sa pagkakaroon ng Fatigue.

Para sa karamihan ng mga karakter, ang anumang uri ng aktibidad ay halos imposible kapag naabot ang antas ng Incapacitated. Sa yugtong ito, ang karakter ay may malay pa rin ngunit walang kakayahan sa anumang bagay maliban sa pinakamahirap na aktibidad.

Lampas sa Incapacitated, ang mga karakter ay hindi na makakakilos. Ang mga antas ng Fatigue – Semi-Conscious, Comatose, at Dead – ay karaniwang inilaan para sa pagsukat ng pinakamatinding epekto ng pagkaubusan ng hininga, sakit, pagkawala ng dugo, gutom, pagkakalantad, at iba pa.

#### Pagbangon mula sa Pagod (Fatigue)

Ang mga karakter ay bumabangon mula sa Fatigue depende sa kanilang Healing Rate. Ang dami ng kumpletong pahinga na kailangan upang makabawi mula sa bawat antas ng Fatigue na naipon ay katumbas ng Recovery Period na hinati sa Healing Rate ng karakter.

Tandaan na ang talahanayan sa ibaba ay kumakatawan sa paggaling ng Fatigue para sa pisikal na pagsisikap. Ang paggaling ng Fatigue ay maaaring mas mabilis kapag bumabangon mula sa Asphyxiation o mas mabagal kung bumabangon mula sa Pagkawala ng Dugo.

## Apoy

Ang apoy ay palaging pinagmumulan ng panganib kapag ginagamit bilang sandata o kapag nagngangalit. Ang talahanayan ng Fire Intensity sa ibaba ay nagbibigay ng limang magkakaibang intensity para sa pinsala sa init, na may ilang mga halimbawa. Ang pinsala na idinudulot bawat Round ay ibinibigay sa kolum ng Damage. Dahil medyo maliit, ang mga pinagmumulan ng Intensity 1 at 2 ay naglalapat ng kanilang pinsala sa isang Hit Location lamang, karaniwan ay ang nakadikit sa pinagmulan. Ang mga pinagmumulan ng Intensity 3 at 4 ay mas malaki, naglalapat ng pinsala sa pinakamalapit na 1d4+1 Hit Locations, na nagpapahiwatig ng antas ng init na kumakalat. Ang mga pinagmumulan ng Intensity 5 ay nakakaapekto sa lahat ng Hit Locations nang sabay-sabay.

Ang apoy ay maaaring magpasiklab ng mga nasusunog na materyales. Kung hindi agad apulain, ang gayong mga materyales ay masusunog sa loob ng ilang Rounds ayon sa ipinahiwatig ng Intensity ng apoy. Kapag nasunog, ang mga nasusunog na materyales ay patuloy na nasusunog hanggang sa pisikal na apulain. Ang pinsala ay direktang inilalapat sa Hit Points ng materyal, hindi isinasaalang-alang ang Armor Points, at sa anumang laman na nasa ilalim. Kung hindi kontrolado, ito ay kumakalat sa ilang Hit Locations, bawat Round, na katumbas ng Intensity nito.

##### Talahanayan ng Intensity ng Apoy

| Intensity | Mga Halimbawa | Oras bago Masunog | Pinsala |
| :-: | :-- | :-: | :-: |
| 1 | Kandila | 1d4 | 1d2 |
| 2 | Sulo | 1d3 | 1d4 |
| 3 | Campfire | 1d2 | 1d6 |
| 4 | Conflagration na pumupuno sa silid | 1d2 | 2d6 |
| 5 | Lava ng bulkan | Agad-agad | 3d6 |

Ang mga pag-atake na nakabatay sa apoy ay karaniwang may porsyentong pagkakataon na magpasiklab ng mga nasusunog na materyales dahil karaniwan ay tapos na ang mga ito sa isang Round lamang. Maliban kung iba ang nakasaad, ang pagkakataong ito ay damage rolled x5% para sa mga pag-atake na nakabatay sa apoy, at damage rolled x1% para sa elektrisidad. Kapag nasunog, ang mga nasusunog na materyales ay nasusunog sa loob ng 1d3-1 pinsala bawat Round hanggang sa pisikal na apulain. Ilapat ang pinsalang ito nang direkta sa Hit Points ng materyal, hindi isinasaalang-alang ang Armor Points, at sa anumang laman na nasa ilalim. Ang natural na Armor ng isang nilalang ay nagbibigay ng normal nitong proteksyon, gayunpaman. Maaaring subukan ng biktima na apulain ang apoy sa pamamagitan ng pagbagsak nang nakasubsob at paggawa ng isang matagumpay na Athletics roll, na nangangailangan ng isang Action Point at maaaring subukan bawat Turn. Kung ang iba ay tumulong sa pag-apula ng apoy sa anumang paraan, ang roll ay isang Difficulty Grade na mas madali. Kung ang damage roll ay nagreresulta sa zero damage, ang apoy ay kusa nang namatay sa lokasyong iyon. Para sa bawat lokasyon na kusang namatay ang apoy, ang roll upang apulain ang apoy ay isang Difficulty Grade na mas madali.

## Oras sa Laro

Sa _Mythras Imperative_, mahalaga ang oras, higit sa lahat upang matukoy kung ano ang nangyayari sa anong pagkakasunod-sunod, upang mailapat ang mga mekanika ng laro kapag nararapat. Tandaan na ang in-game time ay karaniwang hindi katumbas ng oras na ginugol sa paglalaro. Minsan, maaaring kailanganin ng Game Master na buod ang mga pangyayari sa loob ng maraming araw sa isang pangungusap lamang, tulad ng 'Aabutin kayo ng isang linggo upang makarating sa Castle Anthrax,' habang sa ibang pagkakataon, partikular sa labanan, ang mga aksyon na ilang segundo lamang ay maaaring tumagal ng ilang minuto o mas matagal pa upang malutas.

Narito ang mahahalagang pagkakaiba ng oras sa laro:

#### Combat Rounds (Mga Round ng Labanan)

Ang isang Combat Round ay kumakatawan sa limang segundo ng totoong oras. Ginagamit ang mga ito upang sukatin ang maiikling, matitinding pagsabog ng aktibidad na tumatagal lamang ng ilang sandali upang makumpleto o pagsukat ng napakadetalyadong aktibidad na nangangailangan ng sunud-sunod na resolusyon. Halimbawa, ang isang paghabol sa pagitan ng mangangaso at biktima kung saan ang bawat Round ay tumutukoy sa mga pagliko at pagbaluktot ng paghabol.

#### Skill Time (Oras ng Kasanayan)

Ang Skill Time ay kumakatawan sa ilang minuto hanggang ilang oras at ginagamit upang sukatin ang mga aktibidad na hindi nangangailangan ng detalyadong atensyon ng isang Combat Round ngunit nangangailangan ng tiyak na paglalaan ng pagsisikap na may tiyak na resulta. Ang pagpili ng kandado ay maaaring tumagal ng limang minuto, samantalang ang pagmamasid sa mga patrol na ginawa ng mga koponan ng bantay sa paligid ng mga pader ng kastilyo ay maaaring mangailangan ng ilang oras at parehong mga halimbawa ng Skill Time.

#### Narrative Time (Oras ng Naratibo)

Ito ang dami ng oras na ginugugol kapag ang Game Master ay nagkukwento sa mga manlalaro, o kapag ang mga manlalaro ay nag-uusap. Maliban kung may tiyak na dahilan para dito, karamihan sa aktwal na roleplaying ay nangyayari sa narrative time. Kapag nagro-roleplay, ang narrative time ay kahawig ng totoong oras, kung saan ang isang pag-uusap ay tumatagal nang kasing tagal upang maganap tulad ng paglalaro nito. Kung ang isang sesyon ng laro ay naglalaman ng mahabang paglalakbay, o mga panahon ng aktibidad kung saan ang eksaktong oras ay hindi mahalaga, kung gayon ang oras ay lubos na pinagsisikip sa ilang sandali ng oras ng laro.

## Mga Lokasyon ng Tama

Karamihan sa matagumpay (sandata) na pag-atake ay tumatama sa isang partikular na Lokasyon ng Tama (Hit Location). Ito ay maaaring matukoy nang random o sa ilang pagkakataon ay mapipili sa pamamagitan ng Special Effects tulad ng Choose Location. Upang random na kalkulahin kung aling lokasyon ang tinamaan, gumulong ng 1d20 at ihambing ang numerong ginulong sa nauugnay na talahanayan ng Hit Location ng humanoid o nilalang.

Ang mga hayop at halimaw ay karaniwang may bahagyang magkakaibang mga talahanayan ng Hit Location kaysa sa mga humanoid upang ipakita ang kanilang sariling natatanging pisyolohiya. Ang mga ito ay nakalista bilang bahagi ng paglalarawan ng bawat nilalang at karaniwan ay magagamit lamang ng Game Master.

Ang talahanayan ng Hit Location ng humanoid ay inuulit dito para sa kadalian ng sanggunian.

##### Mga Lokasyon ng Tama ng Humanoid

| 1d20 | Lokasyon ng Tama |
| :-: | :-- |
| 1-3 | Kanang Binti |
| 4-6 | Kaliwang Binti |
| 7-9 | Tiyan |
| 10-12 | Dibdib |
| 13-15 | Kanang Braso |
| 16-18 | Kaliwang Braso |
| 19-20 | Ulo |

Tandaan na ang ilang pag-atake, lalo na ang mga mahikal, teknolohikal o ang mga nilikha ng partikular na mga halimaw (halimbawa, ang apoy na hininga ng dragon o isang sumasabog na granada), ay maaaring tumama sa ilang Lokasyon ng Tama nang sabay-sabay. Ang mga indibidwal na Lokasyon ng Tama ay maaaring may baluti, alinman mula sa suot na proteksyon o sa natural na tibay ng panlabas na ibabaw ng nilalang. Mahalaga ito kapag isinasaalang-alang ang Pinsala at Antas ng Sugat sa ibaba.

## Pinsala at Antas ng Sugat

Kung matagumpay na tumama ang isang atake, nagdudulot ito ng pinsala ayon sa damage rating ng sandata. Halimbawa, ang isang shortsword ay nagdudulot ng 1d6 pinsala.

Ang ginulong pinsala para sa sandata o atake ay binabago ng ilang mga kondisyon sa partikular na pagkakasunod-sunod:

1. Ilapat ang Damage Modifier para sa karakter na umaatake.
2. Baguhin ang pinsala mula sa anumang mahika na nagpapahusay (o nagbabawas) ng sandata.
3. Kung na-parry, bawasan ang pinsala nang naaangkop ayon sa comparative weapon sizes.
4. Kung ang Hit Location ay may baluti, bawasan ang pinsala ng halaga ng Armor Points.

Kung ang pinsala na idinulot ng atake ay higit sa zero, kung gayon ang Hit Location ay nakakakuha ng dami ng pinsalang iyon, na ang kabuuang pinsala ay agad na aalisin mula sa Hit Points ng lokasyon. Tandaan kung gaano karaming Hit Points ang natitira sa lokasyon, at ang kategorya ng sugat na nagreresulta.

- **Maliit na Sugat (Minor Wound):** Mayroon pa ring positibong Hit Points ang Hit Location.
- **Seryosong Sugat (Serious Wound):** Ang Hit Location ay nabawasan sa zero Hit Points o mas mababa.
- **Malaking Sugat (Major Wound):** Ang Hit Location ay nabawasan sa isang negatibong score na katumbas o mas malaki kaysa sa panimulang Hit Points nito.

Gamitin ang sumusunod na mga patakaran kung ang isang lokasyon ay makatanggap ng isang Serious o Major Wound. Tandaan na ang mga sugat na nangangailangan ng pagsusuri ng Endurance ay sinusuri ang mga ito sa isang Opposed Roll laban sa halaga ng orihinal na attack roll. Ipinapakita nito ang kakayahan ng mga napakahusay na mandirigma na gumawa ng mas nakamamatay na atake. Ang mga roll ng Endurance ay hindi inuulit maliban kung ang lokasyon ay masugatan muli.

### Minor Wound (Maliit na Sugat)

Ang mga maliit na sugat ay mga hiwa, gasgas, pasa, at sprain. Masakit ang mga ito, maaaring dumugo, ngunit hindi sapat ang kalubhaan upang pabagalin o hadlangan ang biktima.

### Serious Wound (Seryosong Sugat)

Kung ang isang lokasyon ay nabawasan sa zero Hit Points o mas mababa, ang biktima ay makakatanggap ng isang Seryosong Sugat. Ang lokasyon ay permanenteng may peklat, at ang biktima ay hindi makakaatake o makakapagsimulang mag-cast ng spell (ngunit makakaparried pa rin o makakaiwas) sa susunod na 1d3 Turns dahil sa pagkabigla o pagkaabala sa sakit ng sugat.

Ang isang karakter na nagdurusa ng Seryosong Sugat sa isang bahagi ng katawan ay dapat agad na gumawa ng isang Opposed Roll ng kanyang Endurance laban sa matagumpay na attack roll ng kanyang kalaban. Ang kabiguan ay magreresulta sa pagiging walang silbi ng bahagi ng katawan, hanggang sa maibalik ang lokasyon sa positibong Hit Points. Kung isang binti, ang biktima ay babagsak nang nakahiga. Kung isang braso, ang anumang hawak ay babagsak, maliban kung ang bagay ay nakatali (gamitin ang karaniwang sentido komon dito).

Ang isang karakter na nagdurusa ng Seryosong Sugat sa tiyan, dibdib, o ulo ay dapat agad na gumawa ng isang Opposed Roll ng kanyang Endurance laban sa matagumpay na attack roll ng kanyang kalaban. Ang kabiguan ay magreresulta sa pagkawala ng malay sa loob ng ilang minuto na katumbas ng dami ng pinsala na natamo sa atake na nagdulot ng Seryosong Sugat. Ang mga kasanayan sa First Aid o Healing ay maaaring gamitin upang matulungan ang isang walang malay na biktima na makabalik sa kamalayan.

Sa diskresyon ng Game Master, kahit na manatiling gumagana ang karakter, ang lahat ng gawain na nangangailangan ng paggamit ng Hit Location na iyon ay magdudulot ng patuloy na parusa ng isang Difficulty Grade, hanggang sa mabawasan ang pinsala sa isang Minor Wound.

### Major Wound (Malaking Sugat)

Kung ang isang lokasyon ay nabawasan sa isang negatibong marka na katumbas o mas malaki kaysa sa panimulang Hit Points nito, ang karakter ay makakatanggap ng isang Malaking Sugat. Ang karakter ay agad na Incapacitated, hindi na makapagpapatuloy sa pakikipaglaban. Ang isang bahagi ng katawan ay naputol, nabutasan, nabasag, o napunit dahil sa isang Malaking Sugat. Ang karakter ay babagsak nang nakahiga, pisikal na Incapacitated, at dapat agad na gumawa ng isang Opposed Roll ng Endurance laban sa matagumpay na attack roll ng kanyang kalaban. Ang kabiguan ay magreresulta sa pagkawala ng malay mula sa matinding sakit. Kung ang isang naputol, nabutasan, o napunit na lokasyon ay hindi ginagamot sa loob ng ilang minuto na katumbas ng limang beses ng kanyang Healing Rate, mamamatay ang karakter mula sa pagkawala ng dugo at shock.

Ang biktima ng isang Malaking Sugat sa tiyan, dibdib, o ulo ay babagsak nang walang malay, ganap na Incapacitated, at dapat agad na gumawa ng isang Opposed Roll ng Endurance laban sa matagumpay na attack roll ng kanyang kalaban. Ang kabiguan ay magreresulta sa isang agarang at malubhang kamatayan (pinugutan ng ulo, hiniwa sa kalahati, nabutasan sa puso, pinunit, at iba pa). Kung sila ay makaligtas, at ang lokasyon ay hindi ginagamot sa loob ng ilang Combat Rounds na katumbas ng dalawang beses ng Healing Rate ng karakter, mamamatay pa rin sila mula sa pagkawala ng dugo at shock.

Dahil karamihan sa mga Malaking Sugat ay nangangailangan ng ilang uri ng operasyon o makabuluhang mahika upang gumaling, ang naghihirap ay malamang na hindi makakabawi mula sa isang Malaking Sugat sa oras upang sumama muli sa labanan. Depende sa magagamit na paggamot, ang nasugatan na lokasyon ay posibleng mapilay (tingnan ang [Paggaling mula sa Pinsala](0004_Game_System.md?id=healing-from-injury) sa ibaba).

### Pagkawala ng Dugo (Blood Loss)

Ang Pagkawala ng Dugo dahil sa panlabas at panloob na sugat, karaniwang natatamo sa labanan, ay maaaring mabilis na magpahina sa isang karakter. Ang mga epekto nito ay sinusukat sa pamamagitan ng pag-ipon ng mga antas ng Fatigue, tulad ng Asphyxiation. Ang hindi napigilang Pagkawala ng Dugo ay karaniwang nagreresulta sa kamatayan kung walang tulong medikal. Hindi tulad ng Asphyxiation, ang mga epekto ng seryosong Pagkawala ng Dugo sa Fatigue ay pangmatagalan. Ang isang karakter ay nakakabawi ng Fatigue na nawala dahil sa pagdurugo sa rate ng isang antas bawat araw, simula sa araw pagkatapos tumigil ang kanyang pagdurugo.

### Mga Bayaning Huling Aksyon

Sa mga espesyal na kaso kung saan ang mga pangunahing karakter ay nakakatanggap ng Major Wound, maaaring payagan ng Game Master ang isang bayaning huling aksyon – tulad ng paglabas ng isang huling spell, pagtatangka na patayin ang kanilang pumatay, paggapang palayo upang magtago, o pagtatangka na itali ang kanilang sariling sugat. Dapat silang may gumaganang mga bahagi ng katawan na naaangkop sa Aksyon at magsunog ng isang Luck Point. Makakakuha sila ng isang pagkakataon upang subukan ang kanilang Aksyon, at pagkatapos ay agad na babagsak sa pagkawala ng malay, at posibleng kamatayan.

## Paggaling mula sa Pinsala

Ang natural na paggaling mula sa mga sugat at pinsala ay batay sa Healing Rate ng karakter. Dinidikta ng Healing Rate kung gaano karaming Hit Points ang nakakabawi sa isang lokasyon depende sa uri ng pinsala:

- **Mga Minor na Sugat:** Mga araw
- **Mga Seryosong Sugat:** Mga linggo
- **Mga Malalaking Sugat:** Mga buwan

Kaya, ang isang karakter na may Healing Rate na 3 na nagdusa ng pinsala na nagdala sa kanya sa –3 sa isang Hit Location, isang Seryosong Sugat, ay natural na gagaling sa rate na 3 Hit Points bawat linggo hanggang sa lumagpas sa zero ang kanyang sugat, at pagkatapos ay gagaling ng 3 Hit Points bawat araw hanggang sa ganap na makabawi.

Mayroong ilang mga paghihigpit sa natural na paggaling:

- Ang karakter na gumagaling ay hindi maaaring makisali sa matinding aktibidad: kung hindi, ang Healing Rate ay nababawasan ng 1d3. Kaya, ang isang karakter na bumabangon mula sa kahit na isang Minor Wound ay maaaring makita na huminto ang kanyang pag-unlad kung magpasya silang makisali sa anumang pisikal na gawain na maaaring magpalala ng mga pinsala.
- Ang natural na paggaling ay hindi magsisimulang magpagaling ng isang Major Wound hanggang sa magamot ang biktima sa isang matagumpay na Healing roll, o sa pag-cast ng ilang anyo ng healing magic. Ang mga Major Wound na hindi pinutol ang bahagi ng katawan na hindi ginamot sa loob ng ilang araw na katumbas ng ikadalawampu ng kasanayan sa Healing ay magiging pilay, permanenteng binabawasan ang Hit Points ng lokasyon.

## Mga Walang Buhay na Bagay

Ang lahat ng walang buhay na bagay ay nagtataglay ng Armor Points at Hit Points na ginagamit upang matukoy ang paglaban sa pinsala at pagkasira. Binabawasan ng Armor Points ang pinsala bago maapektuhan ang Hit Points. Kapag nabawasan na sa zero ang Hit Points ng isang bagay, ito ay walang silbi.

Ang mga pagtatangka sa pagbasag ay nangangailangan ng isang karakter na gumawa ng Brawn, Unarmed o Combat Style roll, kung nararapat, upang magdulot ng pinsala. Ang isang tagumpay ay nagdudulot ng pinsala sa item.

#### Mga Armas laban sa Walang Buhay na Bagay

Ang paggamit ng sandata laban sa isang walang buhay na bagay na may Armor Points na katumbas o mas malaki kaysa sa sandata ay nagdudulot ng pinsala sa parehong bagay at sandata. Halimbawa, ang paggamit ng palakol sa isang bakal na pinto ay maaaring makapinsala sa pinto, ngunit ang palakol mismo ay masisira bago pa masira ang bakal na pinto. Dapat gamitin ng Game Master ang kanyang diskresyon sa kung aling mga sandata o kasangkapan ang maaaring makaapekto sa aling mga bagay. Ang isang pait, halimbawa, ay hayagang idinisenyo upang mag-ukit ng kahoy o bato habang ang isang espada, bagaman matalim, ay hindi. Samakatuatuwid, ang pait ay hindi masisira mula sa isang bagay na tinatangka nitong ukitin habang ang isang espada ay malamang na masisira.

## Mga Luck Point

Ang mga Luck Point ay nakakatulong na ibahin ang mga bayani mula sa ordinaryong tao. Kinakatawan nila ang kakayahan ng isang karakter na posibleng gawing tagumpay ang kabiguan at lokohin pa ang kamatayan.

Maaaring gamitin ang mga Luck Point habang naglalaro at, sa simula ng susunod na sesyon, ay magre-replenish sa kanilang karaniwang halaga.

#### Paggamit ng Mga Luck Point

Maaaring gamitin ang mga Luck Point sa iba't ibang paraan. Isang Luck Point lamang ang maaaring gamitin upang suportahan ang isang partikular na Aksyon. Hindi maaaring gamitin ang mga Luck Point sa mga roll ng Karanasan o Pagsasanay. Ang bawat isa sa mga sumusunod na opsyon ay nagkakahalaga ng isang Luck Point.

#### Dayain ang Tadhana (Cheat Fate)

Maaaring gumamit ang mga karakter ng isang Luck Point upang muling gumulong o magpalit (halimbawa, ang 75 ay magiging 57) ng anumang roll ng dice na kanilang ginawa. Maaari itong isang skill roll, damage roll, o anupaman na mayroong epekto. Maaari pa ngang pilitin ng mga karakter ang isang kalaban na muling gumulong ng isang atake o damage roll na ginawa laban sa kanila.

#### Desperadong Pagsisikap (Desperate Effort)

Kung ang isang karakter ay naubusan na ng Action Points sa panahon ng isang laban at kailangan niyang makahanap ng huling pagsabog ng desperadong enerhiya upang maiwasan ang isang masamang kamatayan, maaari siyang gumastos ng isang Luck Point upang makakuha ng karagdagang Action Point.

#### Bawasan ang Pinsala (Mitigate Damage)

Ang isang karakter na nakakaranas ng isang Malaking Sugat (Major Wound) ay maaaring gumastos ng isang Luck Point upang ibaba ang antas ng pinsala sa isang Seryosong Sugat (Serious Wound). Binabawasan nito ang pinsalang natamo sa isang Hit Point na mas mababa kaysa sa kinakailangan upang magdulot ng Malaking Sugat.
