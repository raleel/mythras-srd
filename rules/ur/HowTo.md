# پیشگی شرائط

1. [مارک ڈاؤن زبان](https://www.markdownguide.org/basic-syntax/) کی بنیادی سمجھ۔

2. اگر آپ کا پہلے سے کوئی اکاؤنٹ نہیں ہے تو آپ کو ایک مفت [گٹ ہب](https://github.com/) اکاؤنٹ بنانے کی ضرورت ہوگی۔ تجویز ہے کہ آپ اپنی RPG کا نام استعمال کریں کیونکہ یہ URL کا حصہ بن جائے گا جسے لوگ آپ کے SRD تک رسائی کے لیے استعمال کریں گے۔

3. آپ گٹ کو جانے بغیر براؤزر کے ذریعے یہ سارا عمل کر سکتے ہیں۔ اگرچہ گٹ ہب ڈیسک ٹاپ سیکھنے سے آپ کو کسی بھی تبدیلی کو مقامی طور پر جانچنے کی اجازت ملے گی قبل اس کے کہ انہیں دنیا کو دکھانے کے لیے لائیو تعینات کیا جائے۔

> اگر آپ کا ڈومین نام رجسٹرڈ ہے، تو آپ اس عمل کے ساتھ اسے SRD بنانے کے لیے استعمال کر سکتے ہیں جیسا کہ میں نے https://srd.7thextinctionrpg.com پر کیا ہے۔

# عمل

## گٹ ہب پر ریپوزیٹری بنانا

1. گٹ ہب میں لاگ ان کریں۔

2. پبلک ٹیمپلیٹ https://github.com/7thExtinctionrpg/RPG_SRD_Template پر جائیں۔

3. "Use this template" پر کلک کریں۔

![Step3](/_media/Step3.png)

4. "Create a new repository" کو منتخب کریں۔

5. ریپوزیٹری کا نام درج کریں۔

![Step5](/_media/Step5.png)

6. 'Private' کو منتخب کریں۔

7. 'Create repository' کو منتخب کریں۔

ٹیمپلیٹ کو کاپی کرنے میں 5-30 سیکنڈ لگیں گے، پھر آپ کے پاس اپنی کاپی ہوگی۔

8. یقینی بنائیں کہ آپ کے پاس ".nojekyll" فائل موجود ہے، یہ ایک خالی فائل ہے جس کی ضرورت اس وقت پڑتی ہے جب کسی ویب سائٹ کو گٹ ہب پر ہوسٹ کیا جا رہا ہو ورنہ SRD صحیح طریقے سے ظاہر نہیں ہوگی۔ *(گٹ ہب پیجز کو بتاتی ہے کہ شائع شدہ فائلوں کو جیکل کے ذریعے نہ چلائیں۔)*

## اپنا مواد شامل کرنا

### *.md فائلیں

.md فائلیں مارک ڈاؤن فائلیں ہیں، مارک ڈاؤن ایک ہلکی پھلکی مارک اپ زبان ہے جسے آپ سادہ ٹیکسٹ دستاویزات میں فارمیٹنگ عناصر شامل کرنے کے لیے استعمال کر سکتے ہیں۔ آپ کے مواد کے سائز کے لحاظ سے آپ اپنے گیم کی تمام معلومات کو ایک *.md فائل میں رکھ سکتے ہیں، یا جیسا کہ میں نے [7thextinctionrpg.com](https://srd.7thextinctionrpg.com) پر کیا ہے ہر باب کے لیے الگ فائلیں بنا سکتے ہیں۔ "sample.md" کچھ فارمیٹنگ کی مثالیں فراہم کرتا ہے۔

مارک ڈاؤن زبان سیکھنا بہت آسان ہے، مکمل گرامر کی فہرست کے لیے اوپر Prerequisites کے تحت پوائنٹ 2 میں فراہم کردہ لنک کا استعمال کریں۔

### _Sidebar.md

یہ آپ کی ویب سائٹ کے بائیں جانب کا مینو، نیویگیشن پین ہے۔ ایک بار جب آپ اپنی ریپوزیٹری میں ایک مارک ڈاؤن فائل شامل کر لیتے ہیں تو آپ کو فائل کو یہاں شامل کرنے کی ضرورت ہوتی ہے تاکہ اس کے مشمولات شامل ہوں۔ موجودہ فارمیٹنگ پر عمل کریں۔

## ریپوزیٹری کو ویب سائٹ میں تبدیل کرنے کے لیے گٹ ہب پیجز کو فعال کریں

1. اپنی ریپوزیٹری میں 'Settings' پر کلک کریں۔

![Website-Step1](/_media/Website-Step1.png)

2. بائیں ہاتھ کے مینو میں 'Pages' کو منتخب کریں۔

![Website-Step2](/_media/Website-Step2.png)

3. اگر آپ نے ریپوزیٹری کو 'public' نہیں کیا ہے، تو آپ کے پاس دو انتخاب ہیں۔
    a. ریپوزیٹری کو public بنائیں جو پھر آپ کو اپنے SRD ویب سائٹ کو مفت میں ہوسٹ کرنے کے لیے گٹ ہب پیجز استعمال کرنے کی اجازت دے گا۔
    b. اپنے گٹ ہب اکاؤنٹ کو اپ گریڈ کریں (یہ تحریر کرتے وقت اس کی ماہانہ $4 یا سالانہ $48 فیس ہے)۔

4. 'Branch' کے تحت 'main' کو منتخب کریں اور save پر کلک کریں۔

5. اسے 10-20 سیکنڈ دیں پھر صفحہ کو ریفریش کریں۔

6. اب آپ کو اپنا URL نظر آئے گا جیسے https://coweater.github.io/mysrd/

![Website-Step6](/_media/Website-Step6.png)

## فونٹس تبدیل کرنا

1. /_assets/style.css فائل میں ترمیم کریں۔

2. 'body section' کے تحت آپ کو siteFont اور headingFont ملیں گے۔

3. انہیں اپنے مطلوبہ فونٹس میں تبدیل کریں۔

## اپنے SRD میں مفت آئیکنز دکھانے کے لیے فونٹس اویسم اور/یا مٹیریل آئیکنز پلگ ان کا استعمال

### فونٹس اویسم

مفت آئیکنز اور اینیمیٹڈ آئیکنز حاصل کرنے کے لیے فونٹس اویسم استعمال کریں۔ سبسکرائب شدہ صارفین پرو آئیکنز استعمال کر سکتے ہیں۔

1. [Fontawesome](https://fontawesome.com/icons) پر آئیکن تلاش کریں۔
2. آئیکن کو منتخب کریں۔
3. پاپ اپ ونڈو میں، HTML ٹیب کے تحت " " کے درمیان سب کچھ کاپی کریں۔

```html
<i class="fa-solid fa-camera"></i>
```

4. کاپی شدہ متن کو `:` کے درمیان بغیر کسی اسپیس کے رکھیں۔

```
<i class="fa-solid fa-camera"></i>
```

### مٹیریل آئیکنز

مزید مفت آئیکنز کے لیے مٹیریل آئیکنز استعمال کریں۔

1. [Material Icons](https://fonts.google.com/icons) پر آئیکنز تلاش کریں۔
2. ویب اسپین لنک کو براہ راست اپنی مارک ڈاؤن فائلوں میں کاپی اور شامل کریں۔

```html
<span class="material-symbols-outlined">
face
</span>
```
